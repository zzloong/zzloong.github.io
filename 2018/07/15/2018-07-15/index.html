2018-07-15问答记录

发表于 2018-07-15 更新于 2021-04-09 分类于 Stock
本文字数： 2.6k 阅读时长 ≈ 2 分钟

问：飞哥，您好，对于投之家的暴雷，您对现在的网贷有什么看法？人心惶惶啊，投之家的运营团队在网货界也有名气的啊？也是网贷之家的兄弟，还有几轮名投，这样也在非法集资？真的看不懂。作为分散投资也投了一些，能收回的概率多少，谢谢！

【羿飞】 回答：集资诈骗立案了，还有什么可说的啊？

  • [零壹] : 分散也不能太多，不然也关注不过来。实际上股权变更多次后就有风险了，还是投飞哥的合作平台好了，至少不会被骗了[流泪][流泪][流泪]

问：地球村文化传播有限公司口啤不好 虹金所都不行了 单总是不是有病乱投医啊

【羿飞】 回答：我觉得是单总看到壹佰金融等情况，想尽快找其他股东进来，把银河的弄出去。

  • [WANG] : 但这个股东是不是也有问题？
  • [小管子] : 关键就是这个，不能找个也有问题的
  • [空中婴儿] : 而目前是出现了行业性危机，导致整个行业流动性在丧失，泥沙俱下。
  • [高园] : 经营不好就完了？没有以后了？
  • [维A] : 不能加分，反而减分的股东还是慎重吧。前门拒狼后门进虎。
  • [请多指教] : 公告一是地球村，公告二飞哥能不能透漏下

问：大飞好！饭饭金服可否也给关注一下！现在投资人非常焦虑，平台负责人也没什么具体还款方案，大家一团散沙，请费心协调一下！谢谢

【羿飞】 回答：对不起，我不是救火队长，也不是zf。能竭尽全力照顾好我们自己身边熟悉的人就不错了，算是尽良心。
平台那么多，怎么可能管得过来呢？如果我盲目膨胀，到处做救世主，结果一定是能力不足，全部都失败。力量要集中在一个点上才能取得胜利。

  • [果子] : 饭饭 饭团傻傻的分不清楚
  • [渡劫 回复 果子] : 不是一个名字，就不是一个台子。
  • [自然山水] : 佩服飞哥这种做法，做的对，ZF也不可能照顾到全国的贫苦人民
  • [水边上] : 不过话说回来，不光饭团，有这么多平台做活期，这次风波表明，这些平台对活期的理解确实不到位，挤兑一类的风险其实一直都是存在的，民营机构没有央行发钞背书，很难真正大规模做这类业务。过去总说银行风控僵化死板，银行的人出口就是风险，现在看，银行是保守的极端，民营机构则往往走到了激进的极端。无所谓指责，社会的发展往往就是这样渐进螺旋式的前进，潮水退去，留下的是真金。
  • [果子] : 看来做活期要有足够的背景
  • [龙之梦] : 看来规定活期=不合规，有道理。
  • [啊才] : 支持飞哥
  • [松东河沙] : 银行（中国的例外）活期遭遇挤兑也得跨，08年前后的美国，以及希腊、冰岛等欧洲小国死了多少银行大家都忘了？我自己在那阵子还积极地参与了这个话题的讨论，也一直坚持不投活期平台。但是。。。但是。。。。，恕我直言，在一个氛围里呆久了，或者说太平环境呆久了，人就懒惰了，失去了自我思考，近两三个月来也把买房子的钱放在了饭团（一线城市的首付），但是终究心里一直不踏实，限提前出来了绝大部分。所以，我一直点赞那些‘不和谐’的发言，希望打破自己的思维定势。

问：玖融网6月排名34，看飞哥评分，上月主要是成交量和投资人数比较少，拉分了，下降了6名，虽然也是和图腾贷天鸽互动入股的平台，但我认为天鸽互动控股36%，相对来说控股比例比图腾贷5%高，安全性从上市公司控股比例来讲，认为比图腾贷安全性高，我仓位不低，图腾对玖融有影响，降低到10%如何

【羿飞】 回答：现在上市公司增信背书，屁用没有，沃时代，图腾，哪个出问题了他们负责任了？还信这个就是智力问题了。
也不是说玖融不好，而是说要抛开这些增信看问题，就看他们做的什么业务，能不能赚钱，能不能生存。

  • [木木三] : 飞哥一针见血
  • [财僧] : 选平台的逻辑要变了
  • [琳Lin] : 国资 风投 上市 民营 里面选哪个？
  • [细水长流] : 主要是这个h股太垃圾了😁
  • [Kelvin Fis] : 前不久咨询过他们，他们近期有天鸽互动渡危机的。玖融在这节骨眼上搞实时显示投资排名，也是蛮胆大的。每天都可以实时看到这个月的累积投资额前10排名及金额*
  • [豆号] : 个人觉得背景还是是一个因素 比如饭团是飞系
    当然也不是全部。
    天鸽对车贷是真爱 他本身不是仙股 市值一百亿左右记得。
    有没有用看各自了 信就有用不信则无
  • [苹果树] : 玖融似乎在有接洽渠道资金

问：飞哥，广信贷张小鹏张总您接触应该不少吧？能客观公正聊聊他是一个什么人吗？另外，在网贷行业，掌门人对平台影响能占到50%以及更高吗？

【羿飞】 回答：张总是比较低调的大佬，平时工作重点放在资产端，所以大家熟悉。实际上网贷这个行业，最重要决定生死的还是资产端。
网贷行业掌门人的影响肯定是非常重要的，重要性在70%不算夸张。投平台往往就是投人了。

  • [Kelvin Fis] : 感觉还是要投有团队的，不能投老板独掌大旗的，容易踩坑！*

问：飞哥可否讲解一下转债的收益到底应该怎么算？圈友们各有各的说法不知道哪个正确呢？

【羿飞】 回答：每个平台都不同的啊，并没有固定的算法。

  • [三儿] : 用irr函数算呗
  • [琪琪 回复 三儿] : 这IRR函数真好用，我用它算了365的转债，一算才知道没有表面那么高[捂脸]
  • [三儿 回复 琪琪] : 别忘了年化要乘12
  • [罗晓莹 回复 三儿] : 能具体说说嘛？我是小白
  • [琪琪 回复 三儿] : 有滴
  • [三儿 回复 罗晓莹] : Excel+IRR函数的使用方法_百度经验
  • [罗晓莹 回复 三儿] : 谢谢！研究一下，不懂再请教[握手]

问：第一次买365，看到给咱们圈的标，安全等级都是最低的A级，别的标都是AA以上的。不会有问题吧？

【羿飞】 回答：同一个平台，都是平台兜底兑付，安全等级没有什么本质区别。

  • [龙之梦] : 现在没有单给圈的标吧。
  • [辉 回复 龙之梦] : 早没了，是飞哥回答的太滞后了。
  • [Kelvin Fis] : 这次图腾贷不是展示了同一个平台不同标的区别了吗？*

问：请问下，用什么方法可以比较好的确认一个平台标的真实性呢？谢谢

【羿飞】 回答：需要长期观察，看资料，和投友交流，不少人也会上门去看，会有口碑。并没有一个简单的方法得出结论。而且每隔几个月半年的，就得按同样高标准再过一遍，因为可能内容变了。比如国诚金融我当时决定撤掉，就是因为上了玉石之类不靠谱的东西。

  • [豆号] : 当时GC一度仓位75%
    感谢大飞
    排名看了很久了
  • [7676刘昆] : 与其自己埋头不敢，不如坐享其成

问：飞哥，我是学院刚报到的新生，请问加息卷如何领卵？

【羿飞】 回答：加一下水池学院助理的微信，会帮助你领取。水池圈应该会有自动回复告诉，微信号是Hopetangtang

问：飞哥你好，想问下如果网贷企业宣布清盘，一般可以拿回本金多少？

【羿飞】 回答：一般号称能拿回50%左右的比较多，但是最终执行和落实到30%就不错了。大体上是一地鸡毛。
