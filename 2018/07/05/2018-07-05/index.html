问：飞哥，请问一下，易融恒信最近几个月的标量偏少，去年这个时候标量还挺大的，今年为什么这么少？是受到车贷政策影响？此外，代收也从年初的8亿多降到目前的6.8亿，资金净流出超过一亿多，对平台的安全性是否有些影响？

【羿飞】 回答：就是因为催收方面有阻力，所以现在批款的风控标准高了，这样就会业务少。业务少了自然待收收缩一些，但是平台安全性会提高，资金流没什么压力。这也是毕竟理性的选择，生存第一， 等资产端监管宽松了再大发展也不迟。

问：飞哥，融资租赁模式的，比如一起好的车融宝就不会像抵押车模式被最近的严打波及到吧，因为车融宝的车辆产权是一起好的，借款人只是租借，还款后车才过户给借款人，一起好想把车拖走，警方也无权过问，就是融资租赁的利润没有抵押车利润高是吧

【羿飞】 回答：融资租赁是法律关系比较清晰简单的，收车无障碍。但是融资租赁比押车的利润要低，所以收益率会低一些，平台不怎么赚钱。这是在当前收车高压的情况下，很多平台不得已的选择。

问：股份制银行小柜员一个，已四年，从开始的信心满满到现在的痛苦绝望，现在月月有存款、信用卡、理财、基金等任务，永远也完不成任务，工资也不高，经常加班开会，天天重复一样的工作，没有技能的提高，看不到前途光明，想要辞职，趁现在还算年轻，边学习边找工作。但家里人坚决不同意，辞职了能干什么，家庭也是普通家庭，当初为进银行也是费了很大功夫，但这样的工作感觉让人窒息。该怎么办？

【羿飞】 回答：可以想象，作为银行职工，你要舍弃这个岗位来自于家里人的压力一定是会特别大的，毕竟这在社会上已经是比较不错的岗位了。
你贸然辞职开始新的工作和生活是不明智的，这点我支持你家人的观点。你还没在社会上踢打过，社会上的竞争压力，可能会更让你崩溃，或者是你能力不一定能应付。
当然我不是否定你的能力。我觉得应该是两手抓，一方面是例如现在业余时间理财，另外也可以尝试和朋友一起合作一些生意。在你没有丢失掉当前稳定岗位的前提下，开源。直到你具备的综合实力，你现在的岗位已经完全无所谓的时候，才作出辞职的举动，是明智的。
只要你有足够的实力和成绩，你的家人也会理解和松动的。目前他们的反对主要是因为未来不确定，会失去的很确定。

  • [书画江湖] : 人到中年都会遇到这样的问题，挺过来就好了
  • [大吴] : 马上有人问职业规划了😁
  • [北市仲垣] : 职业转换运动犹如玩高低杠，高杠没抓住，低杠别撒手。否则两手悬空………
  • [张伟] : 你四年，我七年了，银行业现在都这样。跟着飞哥走，早日财务自由，就可以干自己喜欢做的事了
  • [姚磊] : 你要是营销能力强，还是建议寻求机会转信贷岗位。
  • [自然山水] : 银行工作是一个让人羡慕的工作 珍惜吧
  • [ALICE 回复 自然山水] : 很苦逼，其实就是个售货员，不过卖的产品是理财、保险、基金，工资和完成的任务挂钩，底薪很低，不推销产品就没有收入。
  • [自然山水 回复 ALICE] : 现在没有工作的，有工作没有社会保障的，这些人怎么办
