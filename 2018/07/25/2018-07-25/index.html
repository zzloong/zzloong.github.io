发表于 2018-07-25 更新于 2021-04-09 分类于 Stock
本文字数： 4.6k 阅读时长 ≈ 4 分钟

问：飞哥好！看到消费日报社控股365易贷，也有说是全资控股，不管哪种情况，既然被控股了，单总还能控制得了365易贷吗？像您说的，投一个平台看人。也是看重和相信单总和飞哥，我才很相信365，如果单总失去了控制权，找个再牛的爹也是个问题，现在甩锅的爹太多了。

【羿飞】 回答：他是要保证控制权的，这样才能避免出现被银河绑架的情况。这次能及时制止跟有实际控制权有关。新的消费日报进来，是替换银河的，单总控制权仍在。消费日报毕竟是国务院举办的企业，是纯国资。这个自融可能性不大，自融也自融不到自己的口袋里。
短期内作为解决银河问题的手法，即使存在一些不够完美的地方，也比之前的好。非常时期就不要完美要求了。希望单总尽快搞定银河，恢复平台常态运营。

  • [羿飞] : 另外，有人乱传365股权结构，实际上是三方企业信息不准，应该看国家公示系统的版本。
  • [羿飞] : 这个是目前的版本，消费者日本还未变更，等变更以后才能显示。
  • [醋小姐 。 回复 羿飞] : 刚才我在企查查查看的365股权结构 三方没有更新 导致我误解 已删帖 对大家造成的困惑感到抱歉[可怜][可怜]。
  • [羿飞 回复 醋小姐 。] : 没事，你的提示很及时，最近大家都是紧张，该误会的一定会有。能提前意识到，披露正确的信息是防患于未然。
  • [龙之梦] : 第一重仓，经历淮安标波折，看好单总的人品与能力。受这次行业重大危机冲出与银河影响，期待365平安渡过。
  • [夕拾朝花 回复 羿飞] : 谢谢飞哥解答！！

问：弱弱问一句，是不是点击确定以后立刻会有两万余额可以进行提现，这提现是不是秒到？😳

【羿飞】 回答：每人每天最多买5w，会有5k到余额，一样是要t+10的。但是只要到了余额，就没有问题可以提现，这个没出问题过。

  • [羿飞] : 这个活动的本意，是让能支持平台投长标的人，优先领钱，互相支持。
  • [水边上 回复 羿飞] : 以50000为例，那投了后本金是48000吧
  • [羿飞 回复 水边上] : 买了5w活动标，是直接到余额5000啊。剩下45000买了120天标，继续排队向前。
  • [水边上 回复 羿飞] : 明白了，复投时提出10%，剩下90%作为本金买120

问：网贷最近这么动荡，国美黄金、黄金钱包这些会受影响不？要不要撤回？

【羿飞】 回答：这些，其实在网贷行业的动荡开始之前，就在自行收缩了，因为央行发布了不能私自开设黄金账户。所以最近可以看到各平台都在降息，实际上是在慢慢的清退用户。
还有就是虽然今年黄金没什么涨幅，但是可今年糟糕的金融市场比起来，属于比较稳健的，最近减持抄底其他东西，是不错的选择。有的时候，没有赔的就是赚。

  • [冰河] : 所以飞哥，现在黄金管家能要开始逐步分批出手了吗？我从2016年底拿到现在。
  • [御翼] : 感觉黄金在目前位置可以稍等一下吧。。。
  • [罗晓莹] : 飞哥，减持黄金抄底什么好呢？
  • [牛牛] : 飞哥的意思这两个黄金平台也逐步退出了吗？

问：飞哥，是不是应该 教教我们，怎么应对爱钱帮事件，例如大家应该去哪里报案，怎么准备材料，接下来有可能会出现什么情况，您也雷过热贷，希望说说相关维权经验

【羿飞】 回答：我第一时间就说过建议了，但是，没有人感兴趣啊……

    如果能按说的做到还是不错的

  • [波牛] : 投资人基本是一盘散沙，难以形成一致，这个方案实际很难实施，除非有个别团体组织宣布，可以折价接手，跟投资人谈判
  • [波牛] : 成立一个资产清算公司，问题平台第一时间介入，跟投资人谈判，五折（打个比方，总归也要赚钱）清算，时间六月一年等，每月兑付，总比报警强，这是不是也是一种思路

问：一周两个，君融贷，爱钱帮。请问飞哥入股平台清单，重新整合规划。

【羿飞】 回答：还是要下功夫研究资产端啊，学习去了解平台。单纯看我入股了哪个，如果我失误了呢？

  • [大斌] : 飞哥也从来没说他每笔股权投资都保证赚钱[偷笑]

问：飞哥，网贷之家受投之家影响，它自身的估值会不会下降呀?

【羿飞】 回答：肯定会大打折扣吧，甚至成了负资产。这种情况下，有谁会去敢投资给他们，谈不上估值了。

  • [李海波] : 羿飞投资学院可以冲一冲了，估值飙升[呲牙][呲牙]

问：飞哥，资产端的真实与质量是平台的根本。饭团整改转型日趋正轨，圈友十分信任支持，请飞哥再了解介绍一下饭团资产端情况，使圈友更放心。一，底层资产的构成与来源，期限分布;二，目前回款与有否逾期情况;三，下一步转向散标的基本思路与安排。谢谢。

【羿飞】 回答：近期已经在进行这方面的数据的统计了。因为还款到各种期限的标都有，这不是个小工作量的事情。而且监管现在要求也下来了，都要求去错配，前提也是得先清算标的情况。

  • [许晓伟] : 底层资产清晰投友们才放心呀
  • [wyf] : 饭团为啥现在还上30天60天90天的标，不如直接上一些散标，我们如果复投散标优先提现或什么的
  • [三儿 回复 wyf] : 在开发呢
  • [龙之梦] : 底层资产的错配是一回事，更重要的是真实，是逾期很少并受控，没有老赖。请饭团王总能够介绍一下。
  • [弥勒] : 底层资产的来源和真实性一直想问

【羿飞】 回答：爱钱帮的情况，真的是他们自己处理方式不当造成的，近期行业情况特殊，跟投资人交流，进行暂时的展期，都是能接受的，共渡难关。为啥就突然出了个清盘公告呢，莫名其妙。

  • [徐伟伟] : 要么羿飞资本去收购掉吧[偷笑]
  • [大班长] : 经侦一立案 就麻烦了

问：飞哥系平台标的真实，老板良心，但对一点有些疑虑，标的逾期压力大否？抵押物变现速率又怎样呢？

【羿飞】 回答：这是个问题，真标也怕行业长期萧条的，因为需要续贷。还好大多数真标平台，都是没有错配的，短期流动性他们自己都有一定的应对能力。

问：关于饭团，我有个建议，按照您之前说法，其实饭团就是差几千万的流动性。 我觉得圈里就能筹到，如果饭团能拿出部分股份，给圈里入股，有机会能筹集千万级的资金。而且这千万资金是有对应的债权的，说起来也是比较稳妥的。如果饭团能度过这关，我相信以后能有大发展，入股的圈友也能获益。 当然是我浅薄的想法，很不成熟。只是多给一个思路。不要见笑。

【羿飞】 回答：这是不可以的，等于拿无关投友的钱去冒险。饭团的问题，必须靠饭团自己去解决。等完全恢复正常以后，倒是可以大家一起去投，要个好的补贴，然后帮他更快的恢复生机。
至于入股就别提了，那点钱谁也不需要，千把万的有什么毛用。

问：羿飞老师，爱钱帮我还有待收6.3W，就在这几天到期，本来想到期就撤了，没想到一夜之间雷了，还是用亲戚身份证注册的账户投的，之前投的二笔计划类标已经被拆分成了6个散标，合同也不能不载了，看到昨天发的几个版本的处理和维权方式，看着就觉得筋疲力尽…如果不参与报案维权的，是不是就视为放弃了，我查了下我在此平台已赚利息4.8w，这样算下来就算亏了1.5万（这样想安慰一下自己）…想听听您的建议
