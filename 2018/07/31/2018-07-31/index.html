【羿飞】 回答：奇怪，水池学院的，应该都有我微信，你直接微信发我就行了，我帮你问问。

  • [涛心依旧] : 我也没你微信呢
  • [周瑜] : 我也没有
  • [00] : 没有的，加了都没通过[捂脸]
  • [羿飞] : 水池学院的成员，要加我微信的，请先联系助理，助理验明正身以后，会给你推荐链接，这样我才会通过。[晕]否则搞不清是不是自己人啊。
  • [雯竹] : 我有[呲牙]
  • [风雨声] : 我也没有，怕担误你时间，有问题这里问就行了。
  • [财富耕耘者] : 我有[呲牙]
  • [向日葵] : 飞哥，自己人[呲牙]

问：飞哥，能不能有空点评一下365 和 和信贷，这两家估计都是圈友重仓，很多圈友都比较关心的。最近365掌舵人变化、收购方变更、收购比例变化，这些好像都与预期不同，不知道到底怎么回事，猜测，误传多了，对平台也不利。麻烦飞哥了。谢谢了！

【羿飞】 回答：最近特别不适合瞎评论，很容易被人断章取义。对于你在投的平台，最好的方式就是多观察， 少评论，给平台减少舆情压力。
现在是行业动荡，都不是平台自身有什么严重问题。行业平稳了，投资人信心恢复了，就都能很快向好。

  • [66] : 问下飞哥：365连最起码的电子帐户余额与平台余额都不能同步，弄得充、提都不行，这种细节都弄不好，不是无端制造恐惧吗?飞哥您与单总沟通方便，能否邦忙反映下。非常时期平台自身的每一个细节更要过硬才能打好仗不是吗。
  • [醋小姐 。 回复 66] : 这是银行对接的问题 回复说是预计今天解决 两个余额差额不大 按照电子账户余额提现即可。
  • [弥勒] : 不同步就不能取现。
  • [Ms Liao 回复 66] : 现在已经好了
  • [66 回复 Ms Liao] : 嗯这回是好，不知道下回会否重复出错，我的充与提一直都不能同步，每次要人工处理，这回是忙于扑火三天才解决的。

问：羿飞老师：你好！可能你没时间看微信，在这里咨询一下，虽然现在网贷行业动荡，我还是想进入网贷工作，特别想在饭团工作，主要认可你和王总，不知可行否？

【羿飞】 回答：哈哈，你这个要求好。不知道你是什么专业，适合什么岗位。可以作为办职工，半投资人代表的角色。帮助我们监督团队，团队能正常运转，我们就有希望，一步步理顺。

  • [果子] : 打入饭团内部[偷笑]
  • [hhl] : 严重支持[拳头]。找对的人做对的事，你离成功越来越近[呲牙]
  • [龙猫] : 这也行。饭团还能解决就业，不错啊。
  • [wyf] : 现在饭团催收岗位应该缺人[微笑]
  • [高园] : 我又想起了那个故事，想买房，就去售楼处工作，想买车就去4s店工作，车房都有了，就该结婚了，就去了婚庆公司上班
  • [顺] : 排队求飞哥介绍工作，211，985
  • [道情] : 我也报名参加，去饭团上班，没什么本事，干个电话催收也可以啊！
