问：飞哥，问个尴尬的问题，您有想过百年之后您积攒的财富如何分配吗？祝飞哥身体健康!

【羿飞】 回答：这个问题一点都不尴尬，我也从不回避。我自己每年春节期间闲着没事，会重新写一份遗书，每年更新。因为每年资产都会变，必须重新调整一下。
我自己的财富，会给老人，妻子，孩子都各留一份，孩子的那份在他到达30岁以前会分别由老人和妈妈代管，不能让年轻人就奢侈享受，得自己有个没有钱奋斗的阶段。
还有5%是打算给我姐姐的，5%会设立家族教育和医疗基金，给看着我长大的长者们也享一些福，小孩子们多一些教育机会。
最后还有10%会要求捐给慈善机构。我就是小城市走出来的孩子，还在农村生活过。能有后来的成绩，90%是时代和社会给予的， 适当的回馈社会是应该的。

  • [顺] : 赞(≧▽≦)/
  • [羿飞] : 大家不要畏惧写遗书，这是一种很奇怪的感觉。会有一点小悲伤，然后你会安静下来，真切的感受到在这种时刻，你才能分清楚对你人生最重要的人是谁。
    写完以后，你的心会非常宁静，对未来更有安全感。
  • [Viavia] : 真好，解决好身后事才能轻装前进
  • [请多指教 回复 羿飞] : 什么时候出月报，等你月报呢
  • [书中自有 回复 羿飞] : 问一个问题，群主给姐姐的部分是不是要从现在就开始给比较好一点？
  • [书中自有 回复 羿飞] : 等到百年之后再给姐姐，这个是不是太迟了？
  • [蓦然回首] : 我是农村出来的，有能力为家乡父老做点事也是欣慰的。赞同飞总！
  • [渡劫] : 我还没到写遗嘱的水平。[奋斗]

问：飞哥 去杠杆对房市的影响是怎样的呢？

【羿飞】 回答：去杠杆，一定会影响房价的。这轮房价上涨，就是2015年开始周小川说中国人需要提高一些杠杆率开始的。目前各地去库存的任务也基本完成了，去杠杆稳房价成了新的任务。

  • [Carmanxin] : 很多房产抵押标逾期是不是限购不让过户交易有影响？

问：各位圈友：我投小饭团27.8万，投大饭团21.05万，显示昨天总收益155.22元，即按365天计利息为56655元，那么56655/(27.8万十21.05万)=11.6%的收益率，但小饭团利率为8.88+2.3=11.18%，怎么我连小饭团的收益都达不到？谁帮分析下，谢谢

    我觉得已经到底了，只要这几天量能跟上。

问：飞哥，在现场辛苦了，14年看过你和王吉涛在央视的几期节目，觉得你和王总都特别靠谱，王总的团队实力也很强，相信饭团一定会有好的发展，但有一个疑问想咨询，圈内入股饭团的工商撤掉了，资金还在饭团吧？ 如果在饭团，请飞哥回复一下，也提振一下大家的信心。

【羿飞】 回答：呵呵，看来真的是老投资人，我跟王总，就是在那次节目认识的，从此以后多有各方面的合作，成为非常了解和信任的朋友，想来也有快五年了。
我们羿飞资本投资饭团金服的事情是切实的，暂时在工商移掉是在做增资变更，饭团是要增资到7500万的，之后再把羿飞资本添加进去，占比跟以前一样。这样先出后进，是为了避免我们羿飞资本的股东们产生不必要的额外税费。
男人立于世，靠的是诚信和担当。王总在饭团发展强势之时以非常优待的价格让我们参股，是一种信赖。所以当前即使面临巨大风险，我也不会在这个时候撤掉投资保全自身。即使真的处理不好，其他人的股份损失以后我来承担，也不能落井下石。

  • [骑车远行] : 投资难免有波折，信心十分重要，希望大家在这个网贷的严冬里，全力支持饭团，也全力支持大飞

  • [羿飞 回复 马小丫] : 只要饭团成功转型完毕，大家都安全了。应该是能圈粉不少。不过这样做并不是以圈粉为目的。是事情做好了，自然大家会有认同。
    关键是做正事，行大道，好人活的不会差。公道自在人心。
  • [自然山水] : 飞哥始终如此有担当
  • [马小丫 回复 羿飞] : 危机时刻才能看出人的真面目，所谓时势造英雄[强]图腾罗总就缺少飞哥这样的铁杆高参[憨笑]
