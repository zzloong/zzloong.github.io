2018-11-10问答记录

问：最近股市有太多股票已经严重低估，有些个股优势明显超过股票池，飞哥不要光顾网贷，也要抽出点时间雨露均沾啊。

【羿飞】 回答：好的，我也觉得，大家应该站起来了，继续向前走，后面希望大家尽量多讨论投资吧。也希望能正视我或者他人所贪婪的观点。别人的观点是给你参考的，不是保证。
如果要求一个人必须准确才能说话，我觉得所有人都可以闭嘴了。

  • [YI] : 飞哥之前分享股票池，现在是不是没了？
  • [华子 回复 YI] : 把最后的一次存起来，看飞哥的意思近期会多关心股市了
  • [春江水] : 现在我们更需要个股点评，💪飞哥
  • [羿飞] : 我自己早都已经放下包袱，轻装上阵开始新的布局了。但是现在搞的有点不敢说话。投资观点，本无对错，仅只参考，但是有些人不能正确的对待，要求说了就得对，不对就别说。怎么可能有这种保证呢？这样的心理带来了很大的表达阻碍。
    大家先聊起来吧，圈子的气氛先调整到正常一些，我也会积极加入。
  • [付碧海 回复 羿飞] : 别管个别的别人，还有我们这些占绝对多数的人需要通过其他的方式方法把屁土屁的损失挣回来。
  • [谦谦 回复 羿飞] : 我感觉心态不一样的本质区别在于大家和飞哥的挣钱能力不一样造成的。飞哥的钱记坏账可以再挣，没啥大不了的，可以创造更大的增量。可很多人的钱是存量的钱，丢了就再也挣不回来了，比如说我这样情况，就是拿着我们三家的钱在理财，其实也做到了各个圈子平台的合理分布，无奈一个接一个的出问题。我们换位思考，明白今天的情况是大环境逐步造成的，始料未及的，不能怪飞哥，但也请飞哥换位思考一下不同人的处境，其实也是都有不同的难处和苦衷。当然，黑子们无端乱骂街不在讨论范围之内。
  • [羿飞 回复 谦谦] : 这是配置比例的教训啊。收入高低，从比例来说没区别，你们是用了不是自己的钱，这是关键问题。投资要敬畏，要考虑风险承受力。
    我一直告诉大家不要用杠杆，听进去的不多。

问：我把网贷退出的资金，转到了飞哥基金池中适合长期定投的基金中去，用之前一周一次的投资额，每天定投，每月投资额是之前的5倍。我这样的节奏会不会快了？

【羿飞】 回答：因为最近大概率是底部区域，所以提速一点是合理的。关键是还是记得那个原则，一定是自己能长期使用的钱，不要借钱做投资。

  • [hulk] : 坚持按飞哥说的，不上杠杆
  • [天使] : 买定投可不可以理解为交房贷？等若干年后看收益？
  • [阿狼] : 美联储又在搞加息预期，有无可能会导致美股下跌，拖累A股?
  • [剑气满天花满楼] : 时间拉长来看，日周月定投的业绩是趋向一致的，所以选对且坚持更重要
  • [阿狼 回复 剑气满天花满楼] : 长期看差不太多，只是曲线更为平滑一些
  • [剑气满天花满楼 回复 阿狼] : 那是人的错觉，蚂蚁财富里有历史数据为证的，里面的慧定投和普通定投越是长期越收益趋近，少判断少出错，买定离手以后就是判断什么时候要卖就行了
  • [阿狼 回复 剑气满天花满楼] : 因为有的人熬的住，有的人熬不住，全在心态
  • [Gloria 回复 阿狼] : 我觉得经历了过去这半年，应该可以明确没人能准确做的了判断，涨跌都是有可能的。还是坚持闲钱定投吧，做为普通投资人的我们不要指望一下子能赚很多。怀有快速赚钱的心里反而适得其反。就定投放着，也许放一两年涨上来，也许放三五年。

问：飞哥不要和垃圾人一般见识，榜单该出出，饭团该管管。他们不想看评级自己不看，有很多投友人想看；他们在饭团诋毁飞哥，他们自己处理自己的去，我们99%的投资者需要飞哥和投监会领导。他们算老几？无非是想方设法看重仓饭团投资人笑话的可怜虫，偏不让他们看笑话。

【羿飞】 回答：是的呀，该做的事情还是要做的。我只是退群不在群里浪费时间了。一直以来都是我在各群跟黑子们战斗，但是太浪费精力了，老生气也伤身。 所以我先退了各群了，精力要用来做更重要的事情。
现在需要大家都行动起来，主动和黑子战斗了，否则得来不易的稳定就会失去。大家不要做漠视抢夺司机方向盘事件的乘客，最后只能一起覆灭。后面要积极的跟黑嘴战斗，积极的投诉黑子群。
那些黑子，其实只是希望看到别人比他们惨而已，这样的变态现在充斥网络。

  • [凤来] : 要做飞毛腿，直踹其面门
  • [雯竹] : 司机安稳开车，乘客搞定破坏分子[微笑]
  • [苏苏] : 一群很和谐，大家现在已经很理智！
  • [骑车远行] : 坚定地支持，同心协力比什么都重要！
  • [鹅卵石] : 支持羿老师，支持继续出榜单，坚决与黑嘴作斗争！
  • [Leon Lu] : 国人心态不好
  • [山林] : 说得好！

问：飞哥你好，18年三季度业绩披露完毕。我发现股票池的一些股票业绩比17年年初还好，价格却比那个时候低不少。加仓否？却又不知道毛衣战后面继续如何推演？担心又一波承压。纠结！

【羿飞】 回答：这确实是事实，很多企业已经超跌出若干年没见过的低估值。这种时候，不要再愚蠢的预测后续事件，谁能做到准确预测？调整流动性，用多余资金大胆加仓即可。
不过老生常谈，大多数人，我还是更推荐用定投基金的策略，而不是直接炒股。
我总是劝大家不要投资太集中一个领域，这样出了风险也好承受。不要借钱用杠杆。股票市场要敬畏，基金和自己直接炒的比例73比较好，这样既能学习风险也可控。
我这些话，没出风险的时候当耳旁风，希望大家深刻的记住这次教训吧，人生投资的路还长。

  • [羿飞] : 对金融，一定要有敬畏之心。长期生存下来，靠的不是判断全对，是坚持操作纪律。
  • [请多指教] : 现在看来，王威事件飞哥已经救了很多圈友了
  • [一秒一个梦 回复 请多指教] : 王威怎么了
  • [阿狼 回复 一秒一个梦] : POS加杠杠
  • [一秒一个梦 回复 阿狼] : 现在他怎么样了？不在圈子了？

问：有个网贷问题急需飞哥回答，就是目前雷友把资金都撤到了一线平台，尤其是上市公司的平台（不包括那种只是上市公司入股的），这类平台因为已经港股和美股上市，安全性大大提高，
在网贷当前状态下，上市公司平台有可能暴雷吗，出现的极端情况会是什么样呢，是否能保证本金兑付

【羿飞】 回答：一线平台，有直接上市的，评价还是ok的，在这波雷潮中比较有抵抗力，因为有一定的避险现象，品牌是一定的防火线。近期多关注持续的交易额和待收数据。

问：飞哥你好，网贷这波倒闭潮本来以为要回暖了，最近看好像又恐慌了，投资人信心很差，你如何看待后续发展？

【羿飞】 回答：一线的都已经基本企稳，数据开始恢复了。不好的也基本就这样了，等行业慢慢复苏把。

  • [Kelvin Fis] : 公告过的在恶化，坚持着的在回暖！*
  • [王姐] : 我感觉跟现在经济差也有关系，整体大环境不好，借款人借款企业甚至上市公司都没啥钱，怎么还钱呢。如果经济回暖，股票上涨，平台才可能有更多回款，更稳定。

问：飞哥抽空说说咱们池子里的股票吧？很多都已跌透，期待能看到一些详细的分析，天天圈里都聊那-些雷，网贷出事了还是考虑考虑其他的机会比较不浪费时间

【羿飞】 回答：网贷这块，不可预期的损失，逐步的可预期以后，就能预算以后的资金流了。对于这样的平台，应该积极把资金做其他布局。在其他地方站起来才是解决之道。
大家知道资本市场的厉害了，首先应该积极的去杠杆，然后努力把回款用起来，以及努力去工作造血。经历都是财富，吃亏越早越好。

  • [阿狼] : 飞总说的对，现在与其抱怨还不如积极寻找新的投资方向逐步回血，强化自身。提高技能正当时。。。[拳头][拳头][拳头]
  • [Gloria] : 跟国家保持同一节奏，去杠杆

问：飞哥，最近证券疯长，这个势头能持续吗?而且前段时间出现了板块涨停，都没见过这种势头，需要减持吗?飞哥怎么看?

【羿飞】 回答：目前股市的态势，有和美股走相反的势头，在震荡中这个月a股比美股抗跌。这有国际热钱开始转移战场的迹象，当然还需要时间来验证。
我还是更建议大家分批建仓基金吧，不是炒股的老手不要自己直接入股票。股市是大家在网贷损失弥补的好机会，见仁见智了。

  • [霞] : 现在政策面暖风吹拂，应该会有短期行情，可以少量买点[愉快]
  • [阿狼] : 搓衣板行情继续，回款低位布局低估品种，以观后效。
