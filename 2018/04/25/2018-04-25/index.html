【羿飞】 回答：一般的经验来看，还是有好学区的房子潜力更大，中国人最重视教育了，不计代价。

  • [心善美] : ⭕友，你是巢湖哪里的呀？
  • [嘟嘟 回复 心善美] : 就是市区呀
  • [心善美] : 哦，本人合肥高新区。
  • [王威] : 但是学区给予房子的溢价，是有可能随着学区打破而消失，这是潜在的巨大风险

问：河钢股份3.34元，市净率0.77，正统雄安概念收益股，这是不是严重低估了？够格入股票池不？

【羿飞】 回答：已经调整充分了，周末又重提了雄安，估计雄安概念会再来一波，有机会。

  • [春江水] : 现在不是钢铝加征税吗？还买河钢啊
  • [王威 回复 春江水] : 提了才有这个价嘛。。
  • [松东河沙] : 河钢在手里，人在坑里[吐]
  • [天地泉] : 是低估了，不过一直跌看不到支撑了，或者周线级别的250周线附近能撑住。
  • [王睿晗] : 可以关注，买一波，等反弹，估计会有一波强力反弹吧

问：飞哥，您看看高鸿股份怎么样？个人分析股价比较便宜，公司质地也还不错，财务报表还算靓丽，技术图形上看也算是比较底部。想让飞哥帮忙分析一下公司未来走势，是否值得持有？

【羿飞】 回答：应该是不错，调整充分了，财报一季度会比较好。而且最近有芯片概念，有机会。

问：飞哥，请问地产股里面的华侨城、金地这两个怎么样？他们的市盈率、市净率都比绿地要低。

【羿飞】 回答：呵呵，在我看来，都不如绿地便宜。市盈率不是绝对标准，每个季度都动态变化的。

问：飞哥，帮忙分析下中天金融控股的招商贷平台如何?目前中天金融重大资产重组停牌中，控股股东的变数是否会影响到招商贷平台的正常运行和偿付能力?中天是想收购控股华夏人寿，交易定价310亿。最近资金面紧张，不知道有没有断链的风险

【羿飞】 回答：你说的这些是控股母公司的资本运作。这个平台从股权结构看，确实是上市公司控股的，上市公司的整体历史给这个平台背书没什么问题。实力不够最多是并购失败，项目不做。所以这个事情不管是不是运作成功，跟平台关系都不大。

  • [龙之梦] : 招商贷的安全性及体验都不错，但收益低了。

问：飞哥 中美贸易摩擦对匹凸匹有啥影响没？会表现在哪些方面？

【羿飞】 回答：短期内不会有什么影响。长期看，中国的会越来越倾向于发展内需市场，对于国内经济活跃度的提升有好处，资金需求也会增加，是好事。

问：飞哥 经常看某某平台获得融资多少多少。那么 据你了解 这些钱都用于哪些方面。另外 比如融资一亿 一般多久耗完？

【羿飞】 回答：这个并没有绝对的，因为平台规模不同，估值不同，融资使用方式都不会一样。一般领先平台喜欢拼命烧钱冲规模，一直亏损，然后靠上市去套现补上所有的窟窿。
而后面的平台，因为上市暂时看不到希望，所以就会比较谨慎，量入为出，更在意平时的盈利能力，营销方面预算控制的就严格一些。

问：中美双方你来我往的加码过程中，有一点特别有意思，就是特朗普一直声称正在跟中国谈判，并取得了进展，而中国一直对外宣布中美并未展开谈判。这是怎么样的一步走位呢？飞哥可否大概说说。

【羿飞】 回答：就是不断施压，逼着你去主动谈判呗，这样是老大，姿态高啊。中国这次一反常态，以往早就去谈了，所以摩擦不断升级。

  • [渡劫] : 就是谁先谈，谈被动吧。
  • [惟风无忌] : 中美之间迟早的事，中兴事件短期是利空，长期利好

问：是有一定这方面从业经验了 只是这个易鑫车贷飞哥是不是比较了解靠谱 各大风投是否真的 未来前景如何？

【羿飞】 回答：如果已经有一定的行业从业经验，是可以做的。优选还是知名平台，因为知名平台的融资成本低，后方支持充足一些。这个公司我确实不了解，不能给出具体建议。

  • [御翼。] : 车贷饱和了吧，竞争激烈现在。

问：飞哥好，针对股权投资想说说自己的想法。圈里每个人都缴纳会员费，但有一部分圈友包括我自己在内也在努力刷但依然抢不到，有一部分圈友通过注册多个账户因为手快或者经验足经常能抢到项目，我觉得能不能有的项目能让我们这些圈友也能享受到飞哥争取来的红利。说的不到位之处请见谅。

【羿飞】 回答：老话题了，僧多粥少，怎么分也不够，这是必然结果。所以我对股权的定义，就是加餐。我们所有人加入圈子的主要服务是知识+加息+其他福利。这些实际上都已经足足值回票价。
至于股权，随缘就好，切莫较真，我没有能力保证所有人都投的上股权，如果做这个保证，就只能降低门槛，引入垃圾项目，这样危害更大。

  • [王威] : 一是可能降低了门槛引入了垃圾项目，二是可能大幅提高圈费来降低圈子人数，从而保证投得上。。
  • [FENHS 回复 王威] : 重点是现在股票比较坑 大家的注意力都去股权投资 等爆发吧😂
  • [风华了谁] : 注册多个账号，就为了抢股权，不会吧，每一个加进来的账号都是缴纳过会费的，也有可能有钱人的世界我弄不懂
  • [龙猫] : 3000块钱两个机会，不知道划算不划算。
  • [御翼] : 现在抢的模式，谁也不敢保证自己一定比别人快，毕竟大家都是两只手，就是快也是要练的吧，总比什么抽签什么的更公开化，如果换成抽答啥的，那估计给人骂S。。。而LZ说的那种注册多个帐号抢股权……千人抢20名额的风险不言而喻。。。[坏笑]

  • [一蓑烟雨] : 我的意见就是先满足一次都没投上的人，这部分人后面还有额度再考虑第二次的人。飞哥说这样太苛刻我是有些不解的。新房摇号还向无房户倾斜呢。
  • [🐚 回复 龙猫] : 你这计算有问题啊，实际上是第一个账户抢到两个标，再注册一个，可以继续抢。

问：大飞你好，投之家下架了大批合作至台，是投之家有问题了？还是这些平台有问题？

【羿飞】 回答：都不是肯定的，必须具体问题具体分析。下架有潜在判断风险控制的可能。但是也不排除是合作条件不适合终止的，都会有。投之家不能定义一个平台是否安全。

问：未来二号，已经有分红，那么这1000万是否也参与获得分红呢？这1000万，是否和去年5-7月募集的资金平等资格来参与本次和日后的所有分红？

【羿飞】 回答：都是一样的，我们是拿到原有的全部资格，就是相当于去年5-7月参投的性质，因此这个才划算。虽然第一笔分红没多少钱，但这是个很不错的信号，陆续后面基金就都进入分红周期了。

  • [王威] : 那是否能获得已产生的第一笔分红的分红呢。。飞哥好像没回答。。要是性质相当于去年参投，那应该也能拿这个不大的第一笔分红？
  • [羿飞 回复 王威] : 是分红啊，待遇一样的。目前红利在母基金账上，之后发给我们。
