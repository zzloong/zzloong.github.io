问：飞哥 财报是不是经常有动手脚 甚至有作假的可能？那么常见伎俩有哪些 您怎么判断的呢？

【羿飞】 回答：圈友你好。我觉得很多人，可能90%以上的炒股的人，都对财报这个问题还有非常偏激的看法。首先财报肯定是会有假的，但是比例极低，因为这会触犯刑法，敢做的人不多。我们一般在看财报的时候，总是假设是真的财报，如果出现了假的，就会爆发严重的丑闻出来的，然后股价一口气跌成屎。
另外要说的就是，财报往往不是作假，而是很多是做账出来的，也就是设计出来的。这些财务做账好听的叫法是财务规划，是一门比较专业的技术，需要比较好的财务总监来落实。几乎所有的上市公司都有不同程度财务规划，但这与做假账完全不是一个概念。
这就像我采用了很多合法的政策来进行了避税和减免，实际所缴纳的个人所得税并不多，但是你不能说我逃税，我每一分钱都合法合规是一样的。合法避税和财务规划都是合法的，一种技术活，你要看懂并且会用才行。
很多人以偏激的心态来看，然后觉得既然都是假的，那么就放弃看财报了，一切凭感觉。

  • [沉默的羔羊] : 飞哥所言极是，财务规划与做假账是有本质区别的，兄弟一看就不是干企业的。

目前看，北京房价已经下跌，赤峰对北京是跟盘，所以应该会房价保持平稳1、2年，要阴跌也不太容易。

  • [大惟仔] : 有需求就早买，买了就不后悔，想投资就算了，五年内不会超过定期利率
  • [郝望春] : 恩，多谢多谢。

问：羿飞老师，美股大跌A股会不会跟跌？还是相互为翘翘板？再一个人民币要想成为强货币，中国黄金储备会不会在近一，二年内迅速增持达到世界第一？谢谢。

【羿飞】 回答：红花大姐你好。我认为长周期看是跷跷板，因为热钱可能会转向。但是短期还是会受影响的。美股对全球都有方向性的影响。但是目前两个市场的下挫意义不同。一个是高位暴跌，形成顶部，各种线都死叉了。一个的底部区间震荡，未来能止住跌势回暖。

  • [谦谦] : 既然飞哥早就预测到了美股的下跌，也知道短期A股会受美股下跌的影响，为啥不等到这个时机的出现再大仓位的入场呢？
  • [追梦人] : 其实飞哥是不清楚这一天何时到来而已，要是一两年都不来就不投资股了？
  • [羿飞 回复 追梦人] : 说的是，只能分析出估值高低，怎么可能知道什么时候发生呢？

问：飞哥好，咨询开通城际轻轨问题，江苏无锡至宜兴（宜兴属无锡市管辖），在2018年开始投资建设S2轻轨，城际轻轨开通后沿途站点的镇区以及江苏宜兴市中心城区的房价是否会大幅上扬，在目前房地产市场情况下，作为投资（非刚需）的话，请飞哥分析下投资风险。

【羿飞】 回答：轻轨开通，对区域中心城市，商业是繁荣，对于中心城市覆盖的一日生活圈，房价是正面影响。如果超出了一日生活圈，可能是人口倾泄的出口，不一定是是利好。

  • [自然山水] : 借着圈友的话题咨询下，城市轻轨选房要离站点和枢纽换乘站点分别多远比较好？即不会太吵还有利于商圈的出租等
  • [有书则强] : 这么说随着开通轻轨，人口有可能会进一步中心城市倾泄集中，对中心城市利好，对沿途特别是远一些的地方是利空。

问：飞哥 你好 长牛基金000311 他有重仓茅台和格力 还是适合继续定投？

【羿飞】 回答：这个基金，大周期的表现一直是不错的，所以如果是5年十年的定投，完全可以继续。如果是近期才开始建仓的则没必要，可以等等。

  • [太阳] : 汇丰大盘A和兴全趋势现在继续定投，应该没有什么问题吧？
  • [小管子] : 我也有同样的疑虑，可惜已经建仓了，有子弹就等企稳了再加仓吧！各位大佬同意不？

问：飞哥帮忙看看现在的比亚迪和海能达？

问：飞哥你好，50万的股权投资机会占每个人全年2次投资机会吗？

【羿飞】 回答：我还在准备项目书，因为起步金额比较高，预计符合的人其实不多，所以这个项目不适用隔项目投资和2次机会。
但是应该有优先级，比如如果人慢了，则优先没有投过项目的。
还有，有人说门槛这么高，搞的很多人参与不了不公平。这里也说明一下，这跟公平无关，一个项目谈好的时候确定了投资规模，然后按照有限合伙的人数限制框一下，就是标准。这是自然产生的标准不是刻意提高的。我不可能先制定了投标的标准，然后再跟项目方按照这种标准去谈，对于对方来说莫名其妙。

  • [大吴] : 飞哥，项目书是年前发布还是年后
  • [啊才] : 有道理啊，飞哥说的支持
  • [石生花] : 这个项目干上今年就圆满了！
  • [云逸] : 想参与，但是资金凑齐要到月底了，赶的上吗？
  • [磊磊 回复 云逸] : 阿飞说过，大概3月10-15日缴款，时间来得及。
  • [磊磊 回复 大吴] : 年后发。
  • [不进则退] : 请问飞哥，何时会有这类项目？会提前通知大家吗，我刚来，不太知道活动
  • [云逸 回复 磊磊] : 3月底才有这么多钱，所以可能来不及了

  • [请多指教] : 还好投之前问问，前段时间看财经新闻也不知道是哪个砖家的文章，差点上车！看来我的3000赚回来了
