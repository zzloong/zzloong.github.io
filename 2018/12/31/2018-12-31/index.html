2018-12-31问答记录

发表于 2018-12-31 更新于 2021-04-09 分类于 Stock
本文字数： 2k 阅读时长 ≈ 2 分钟

问：飞哥，原油除了华宝油气，有其它适合长拿的方式不？

【羿飞】 回答：华宝油气，南方原油，国泰大宗商品都可以，都是比较受油价影响的。但是最近并不是好时机，中美谈判结果出来，如果不打了，大家对全球经济未来乐观，才是好时机。

问：请问羿飞老师，油价跌到40元左右，能否抄底？买一些华宝油气基金？

【羿飞】 回答：油价暴跌了一波，进入弱势，短期内指望v型反弹是很难的，建议观望。中美贸易战的谈判，决定未来经济走向，落地了才能做判断。

问：大飞哥：我看到金融办近期发布了推动商业银行多渠道补充资本金，推动尽快启用永续债的进程的通知，我看下相关资料，永续债一般是针对企业，尤其是对国有大型企业发行的，本次为什么针对商业银行？商业银行在人行都有存款准备金的。资本金不足了是不是可以用一些存款准备金而不用有资金使用成本的债券。里面的逻辑关系不太清楚。

【羿飞】 回答：说明银行是亲儿子啊，还是要保银行不会引发债务危机。永续债推出，说明在银行借款的大企业们，已经艰难到了先过桥结清债务在，再重新 办续贷的能力都没有了。整个社会流动性太紧缺了，找不到过桥资金了。所以直接宣布债务永续，能还息就行，债务就不用掀开盖子了。
讲真的，掀开盖子，别说网贷平台，银行也没法看。

  • [阿狼] : 看来真是到处缺钱
  • [三儿] : 永续债是银行向别人借钱用于补充资本金，不是指发放永续贷款。资管新规导致很多不占用资本金的表外业务转为了占用资本金的表内业务，以及为了以后信贷扩张做准备，得给亲儿子找到尽可能多解资本之渴的渠道。
  • [无敌草] : 核心资本充足率和存款准备金是两个概念
  • [三儿] : 永续债这个玩意儿比较特殊，可以直接计入权益。
  • [羿飞 回复 三儿] : [握手]多谢指点。原来如此，那还是之前资管新规的后遗症，扎了个窟窿，又来打个补丁。
  • [阿狼] : 相当于长期以及短期债的组合。
  • [🐚] : 银行确实没法看，刚和同学交流没几天

问：🙂飞哥您好 最近天天听减税降费，但也没看到动静，是等着应对毛衣战，是非常复杂的系统性工作所以需要时间，还是我太单纯了人家只是说说而已，或是别的什么。想听飞哥说说。

【羿飞】 回答：减税降费应该是大势所趋的。之前也在做了，给企业减税，但是做的不太好，往往是给大企业降低了税收，但是最脆弱的民营企业增加了成本。
最近这轮减税是在个税角度出手的，希望增加普通民众的收入吧，提高消费潜力。首先是起征额提高了，其次是增加了很多抵扣项。综合来讲，大家在个税方面的负担会降低，社会消费力会有所提高。
但是，问题紧跟着就来了。这对于收入中下等的人是好事，相当于变相加薪了。但是对于收入中等和略高的人来说，一定是不好的。因为随着抵扣制的执行，未来大家收入一定是越来越清晰的，个税方面也会被当成zf的重要税源。其实这个问题在很多年之前我就看到了，早晚会个税越来越严格的，大家积累财富剩余的时间窗口要珍惜。
其实这是平抑各阶层收益差异的政策，社会注意好嘛~~~~~

  • [羿飞] : 大家也不用太慌，目测留给大家的时间窗口还是有很久的。因为个税要成为主要税种的话，会产生很大的社会矛盾，这之前福利改革必须跟上的。这都是需要时间去逐步落地的。
  • [Leon Lu] : 对于中高收入以前没有全额缴税的肯定不好，但对于中高收入全额缴税的，就像外企打工的中高层呢？还是未知之数
  • [羿飞 回复 Leon Lu] : 逐步落实的群体次序：公务员——半事业单位——国企职工——上市公司员工——大型民企——当地户籍群体——普通民企——个体小业者——流动人口(可能永远无法绝对落实)。

问：美联储加息，央行定向降息，飞哥怎么看？

【羿飞】 回答：美联储加息以后，全球角度流动性会更紧涨。所以中国作出一些宽松的行动或者信号。比如最近货币政策定调宽松，以及定向降息，都是要避免美国形成资金抽离造成流动性紧缩。
明年美国加息节奏会放慢，压力会小一些。现在唯一的利空悬念就是贸易谈判了，3个月内见结果。

  • [木屐] : 今天和发改委的朋友吃饭，据说贸易谈判至今尚未实质性启动。人民日报分析川普从叙利亚撤军可能要挥师亚太。英，法，意都混乱着，欧盟经济堪忧。山穷水尽疑无路啊。
  • [追梦人 回复 木屐] : 来吧来吧
  • [水边上] : 看了向松祚的讲话，向问，今年年初说中美贸易战搬石头砸脚，大打大有利，小打小有利的主流媒体，你们在哪？

问：飞哥你好，ofo退押金吗？

【羿飞】 回答：肯定是要申请退的呀，申请了慢慢排队吧。ofo已经败下去了。

  • [不老] : 从支付宝申请退款快两个月了，还没到账，十之八九雷了200押金[捂脸]

问：大飞哥，美联储今天凌晨宣布加息了。这个事对白银的影响有多大？我有一些银行的纸白银，一直是亏的状态，拿着的话，2019年有什么操作策略吗？谢谢大飞哥~

【羿飞】 回答：中长期看好的。因为明年的加息预期低于今年，最多3次，最少2次是最大的概率。这样黄金能走强，白银会跟一下。其实我一直不太鼓励炒银子，因为缺少金融属性，大长期看，银子很难比金强势。
