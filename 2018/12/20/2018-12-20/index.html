2018-12-20问答记录

发表于 2018-12-20 更新于 2021-04-09 分类于 Stock
本文字数： 1.4k 阅读时长 ≈ 1 分钟

问：飞哥，我家住北京市通州区某村，有套50多平米的一居室小产权房闲置中准备出售，因我是村民，所以过户和产权啥的没问题，位置在广渠路东沿线上，离在建北京市政府比较近，地理位置优越。
如果出租的话，每年租金3万，出售价格在100万，直接卖的话，钱放银行吃利息都够租金了，但我总觉得背靠通州副中心，未来如果房价调控放松的话，长期看房子还会涨价，在犹豫是卖是租，给点意见

【羿飞】 回答：这样的好位置，还是先留着观望的好。可以用现金去先找租金回报更好的项目，拿下以后，再卖掉这种低回报房产回笼现金，这样更稳妥。
你这个租金回报是3%。很多商铺写字楼，租金回报是7-9%，是有套利空间的。

  • [天使] : 商铺，40年公寓很难出手啊
  • [天使] : 商品门面，商品公寓基本没法出售的

问：飞哥你好，胜利精密，中环股份，酒钢宏兴，这三只个股如何？打算平均配置一份。

【羿飞】 回答：都还可以，调整也比较充分，如果分散，分批买入可以的。

问：飞哥好，近期布局、定投这几支50基金如何？嘉实基本面50，160716；建信央视财经50，165312；华夏上证50AH，501050。谢谢！

【羿飞】 回答：50的上一波涨的好，后面应该是阴跌为主。近期关注180和300的指数相对好一些。

  • [无敌草] : 首推基本面，其次AH
  • [霞] : 50之前涨太多了，现在指数都被拖累了！

问：飞哥，饭团的其他几个资产端不甚明朗，企业抵押贷100%回款，记得您说大概有一千万吧。。。。那现在针对轮动的以租代购标呢？我到期的标只回了11%，剩下的无音信，我们要不要去找轮动负责人，起码把真实的借款人信息给到我们，或者透明些，难道逾期的都没催回来吗？轮动应该能联系上，我觉得在这时候就先找能找的资产端，起码把标的弄清楚

【羿飞】 回答：有的时候，心急真的是不解决问题。以租代购属于比较优质的资产了，虽然有逾期，但不是没希望。如果大家都去找，几百上千人上门，那估计是资产端是真的没法干了。如果 想去也可以的，没有人拦着。
我也没什么好办法，因为大家要的满意结果是一分钱不耽误一分钱不差，这不是个人能做到的。

  • [顺～～ 回复 CRL] : 等吧
  • [xiangly] : 到期回款是0[流泪]

问：飞哥你好，为什么一夜之间共享单车几乎都要完了，好像只有hello在三线城市混得还可以。故事进行到现在，哪里出问题了？谢谢。

【羿飞】 回答：资金链压力导致的，这个行业，本身就是资金链之战。关于共享单车行业，之前我曾经写过一篇分析文章。现在看基本判断准确，最终能活下来的就是摩拜了，因为有腾讯大腿不断输血。以后也不会有太多新竞争对手加入了，因为行业已经发展到普遍不缴纳押金的阶段了，已经不可能自己造血发展。
新入场者资本投入太重，这个行业已经快速形成垄断。
之前单车行业的分析文章。
羿飞：共享单车行业的金融玩法逻辑

  • [一秒一个梦] : ofo咋就突然不行了
  • [羿飞 回复 一秒一个梦] : 资金链断裂。后续投资烧钱跟不上了，也没有腾讯那样强势的流量入口来降低成本。
  • [风华了谁] : 共享模式出现的太早了，国民素质还没来得及提高，各种暴力骑行，故意破坏，随意停放造成的故障率太高。
  • [三儿 回复 风华了谁] : 公共自行车这种模式，除了禁止机动车的区域之外，在全世界都没有成功过。
  • [无欲则刚 回复 风华了谁] : 这个可以循环推进，借力打力，主要是国家很多信用建设完全停留在口头，然后出的结果又相反，让人无所适从。以至于现在老赖猖獗，沉重打击守信者的内心。
