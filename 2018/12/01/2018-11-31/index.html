2018-11-31问答记录

发表于 2018-12-01 更新于 2021-04-09 分类于 Stock
本文字数： 2.5k 阅读时长 ≈ 2 分钟

问：飞哥你好，我以前在东北沈阳工作，属于事业单位，不交养老保险。现在在北京工作，一直交养老保险。现在原单位改革，事业单位也要交养老保险，而且要补交以前的几年，我想咨询一下，我现在要补交吗？补交以后可以迁移到北京，划算吗？谢谢。

【羿飞】 回答：理论上最终应该只能在一个地方退休，享受退休待遇。现在多个地方缴纳是因为社保还没联网。实际上可以申请迁移，吧俩地方的时间合并起来，这样对以后退休待遇有利。
还是得看你职业规划吧，如果是打算一直在北京生活的，那么最终还是得在北京退休，退休金高一些，就医也方便。
如果老家那边和北京有重合年份的，估计北京这办退休时不会承认。建议先去咨询一下，尽快把以前工作年限迁移到北京的手续，确认能办理及合并，还是补一下好。因为退休金，基础运算有一条是工作年限，每多一年退休金高1%。

  • [Fanny] : 补交，然后办迁移，但和北京的时间上不要有重合，我是去年刚迁过来的，因为两地交的有重合，要把外地的那段时间的退保，才能办理迁移
  • [霞] : 希望对你有帮助
  • [心得] : 大飞这个也明白[强][强]
  • [🐚] : 你这叫什么事业单位，应该是能享受到近乎公务员的退休工资比例才对啊，老师医生是典型的事业单位。
    保险都没得，是当成临时工了呀

问：飞哥你好，请问哪个网站对股票板块分类最靠谱？我看的同花顺什么的对同一只个股经常即在这个板块，又在那个板块，很糊涂。怎么能分清一些。谢谢！

【羿飞】 回答：板块本来就是基于一些主营的分类或者是上市公司最近热炒的题材。这些确实就是经常变动的，每个软件都会有点区别。你看多了也就有概念了，不会被误导，并没有一个软件更准确更权威的说法。

问：飞哥。您觉得现在AI智能炒股和人工智能搞投资。这个会成为以后的趋势吗？这方面靠谱吗

【羿飞】 回答：只会阶段性的出现一些公司，抓住一些套利机会。长久看不可能算法取代人去投资。因为很多伟大的项目，都是因为一些情怀，要改变人类社会中一些不足的因素，或者是让一些事情变的更好，这些事情开始的时候，并不具有效益。都是做着做着，事业上了规模，开始变的赚钱了，这些是不能通过算法完成的。算法在早期都给枪毙了，因为概率太低。
股票也是，你能设计一个算法去炒股，就会有人根据这些算法模型，去做盘给你挖坑。

  • [阿狼] : 光靠模型是不行的，一种策略不会永远都有效，可能在正常平稳的状态下有效，如果爆发金融危机的时候，模型就失效了。
  • [毛毛] : 靠谱
  • [许晓伟] : 样本数据太少
  • [龙行天下] : 最近在看未来简史，让我有了新的看法。现在数据流派认为整个世界都是数据，人的智能只是生化算法，人工智能是电子算法，整个世界的有机类、无机类，都是数据，是算法。人工智能的发展潜力巨大，当初我们认为人工智能不能的现在都突破了。目前人工智能下围棋、绘画、谱曲、做诗等都超过人或接近人类。所以目前人工智能不可以的事，在以后说不定就可以。[呲牙]
  • [阿狼 回复 龙行天下] : 感性认知比较多的方面，人工智能还是不行吧
  • [龙行天下] : 目前阶段的人工智能，不久后干掉的是初级脑力劳动者，比如医院里干B超的医生，挂号、交费员等等诸如此类的。人工智能再发展下去，一切皆有可能。

问：飞哥你好，jd最近跌的有点猛，能不能买点jd股票？您怎么看他这一波。

【羿飞】 回答：美股最近最好都先别碰，因为是大盘在跌，个股很难有特别好的表现。

  • [庄懿鸣] : 我倒是看好京东的。京东是因为黑天鹅事件发生的股价腰斩。但是底层资产和增长率都是OK的。而且团队也很稳定。
  • [羿飞 回复 庄懿鸣] : 业绩增长，发展潜力都没问题，只是要等好的交易时机，大盘是基础。
  • [苹果树 回复 羿飞] : 我这么理解飞哥的意思，好的股票也是需要大盘趋势的，所以大盘不好时进入，可能等飞的时间会较长，或者成本不够低，比较安全的时候是大盘趋势明确时，时间成本和确定性都相对更好，这样理解对吗，飞哥?

问：飞哥对我国通胀率实际为12%，物价6年翻一番，要想保证自己财富排名不落后，必须要达到22%年化怎么看。这是骇人听闻，还是事实。

【羿飞】 回答：拿m2高增长率说事儿，说这是通胀率，是很多砖家扭曲带路的说法。其实m2这种流动性的货币，是商业交易的工具，相当于我们普通人兜里的现金。
比如以前大家穷，每月赚100元的时候，兜里一般就是1块钱的现金零花钱。现在大家每月可以工资5000元了，增长了50倍。但是兜里的现金可能有1000元，增长了1000倍。但是这不代表你只有这1000块。其实你还有房子，有车了，这些都是固定资产。
2017年12月末，广义货币(M2)余额167.68万亿元人民币。中国社会的总资产现在是国有的五六百万亿，民间资本三四百万亿。

  • [龙行天下] : 不管通货膨胀12%对不对，但要22的年化才能持平，是什么逻辑？不是保持12的年化就持平了吗？！
  • [幻想 回复 龙行天下] : 人家说的要赶上M2的增速
  • [幻想 回复 龙行天下] : 他的算法是通货膨胀=M2的增速22%-10%的GDP=12%

问：咨询一下飞哥和圈友，首先佩服投复利，老大不在竟然撑这么好。但有个疑问，投复利不是做融资租赁业务的吗，按道理说，可以收车的呀，为什么抓人？还是说因为原来的车贷业务被抓？

【羿飞】 回答：我打听了一下，说是以前的车贷一点点遗留业务，在平台也就一千多万的规模了。所以现在的政策尺度，真的是恶心啊，就一点点事情，完全不顾我们这些几个亿规模的普通投资人的利益和死活。

  • [庄懿鸣] : 这个政策来回来去搞。真的认真做业务的平台。老板被搞进去了。。。无语
  • [天使] : 这9%得利息，实际只有0.75%年华，，，，
    一起好按两年兑付，这投复利啥时能把本金还了？。。。。85%的资金全出问题。。。。
  • [xue] : 要组织签名捞人吗？
  • [三儿 回复 天使] : 好的一面是资产质量比易融强，坏的一面是老大不在……希望祁凯家人这周能与资产端沟通出成果。
  • [小心面烫] : 两个月还没捞出来，凶多吉少，估计当地警方作为扫黑除恶大功一件
  • [阿狼 回复 小心面烫] : 想出成绩用尽各种办法
  • [阿狼 回复 三儿] : 今年真是时运不济啊
