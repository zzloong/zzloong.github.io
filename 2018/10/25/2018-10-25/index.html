发表于 2018-10-25 更新于 2021-04-09 分类于 Stock

问：飞哥，作为资金总量小的普通投资人，目前除了调整心态，给平台时间慢慢等，总结经验教训之外，我们还能积极的做哪些事? 想做些什么，但能力有限又不知道能做什么。

【羿飞】 回答：体量很小，就简直是太幸运了。在这轮巨大的金融危机和动荡中，可以用比较少的损失获得宝贵的经验。挫折这种东西，越早来越好，损失成本更低。
体量小，损失也会小，现在最应该做的事情就是积极工作，考劳动性收入重新积累财富，可以很快回血的。

  • [可爱多] : 体量大的怎么搞
  • [一只懒猫 回复 可爱多] : 扛着
  • [豆号 回复 可爱多] : 做和体量小人危机后一样的事情
    [皱眉]
  • [Gloria 回复 可爱多] : 单独发个帖问一下

问：飞哥，一般上市公司将持有股票质押给证券公司属于利空还是利好啊？一般什么情况公司才会这样做?

【羿飞】 回答：应该说股权质押只是一个财务手段，不是利空也不是利好。是大股东把资产盘活，获得流动性的一种手法。
如果股价持续下跌，质押率过低则有爆仓风险，例如乐视事件，就是利空。但是一个企业持续稳定发展，业绩增长价格增长，这股权质押是很正常的。

问：飞哥，咨询一下王老吉什么时候出来啊？目前处于什么阶段了？总不能无限期的在里面蹲着吧?

【羿飞】 回答：请翻我之前的帖子，有过详细的说明。

  • [一秒一个梦] : 仍然处于侦查，耐心等待吧。反正现在都在老王不在得情况下想办法个

问：飞哥。最近没有饭团的最新消息了。是否最近有饭团的最新讨论结果。

【羿飞】 回答：周六的投监会，内容和信息量比较大，后续这几天，会逐步的发帖子披露给大家的。

问：飞哥。最近网贷行业有所稳定和好转吗。

【羿飞】 回答：好转是肯定有好转的。没有继续恶化，现在是逐步修复中。
有点像一个人大病吧，病来如山倒。目前病情已经控制住了，人是死不了啦，但是要康复，起床下地蹦蹦跳跳还差得远，肯定得修养一阵子了。
目前来说，就是等资产端堆积问题的消化。

问：飞哥，目前已知的平台，一起好，易融，和信贷，广信贷，都在申请备案，365和饭团申请了吗?最终zf要求的p2p平台不允许做担保，不许刚兑，基本所有正常的平台都做不到吧，那岂不是说只有现在已经逾期的平台才更加合规?一起好对逾期的标进行分期兑付是不是也会影响备案结果啊？

【羿飞】 回答：目前备案早叫停了，各地都是在进行自查。自查主要的内容是规范化和是否有自融是主要内容，其他的，去刚兑的还真不影响。因为本来之前zf就要求不允许承诺刚兑。
但是老百姓是很诚实的，用脚投票，一个平台只有刚兑或者变相刚兑大家才会投。所以这些去刚兑的，即使通过了自查和备案，未来如果打算发展，也是得等行业艰苦期熬过去了，未来要恢复刚兑或者变相刚兑的。
道理很简单，没保障，谁投啊？

问：飞哥好，请问在您的榜单里面那几个属于一线平台呢，或者说您定义的一线平台要符合哪些标准呢

【羿飞】 回答：一线平台并没有绝对的标准的。一般认为是交易额高，规模大，品牌响，背书强的。我榜单里之前因为比较注重风险收益比，不太容易甄别一线平台。
近期榜单风格在进行调整，会不断提高安全性的权重，未来的话，直接看前十名就行，基本上就属于一线平台。

  • [娄琼] : 辛苦您了[微笑]
  • [凤来] : 目前的一线平台流动性不好，大部分主推计划类的一次性到期回本标
