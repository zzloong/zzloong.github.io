发表于 2018-08-25 更新于 2021-04-09 分类于 Stock

问：希望飞哥分析分析这波雷潮什么时候能够过去，预计几个月市场能够稳定下来。

【羿飞】 回答：我认为行业已经稳住了，目前进入信心恢复期，预计未来2-3个月内回到常态。

  • [王睿晗] : 相信飞哥的判断，毕竟在业内这么多年了，忘早日恢复
  • [莫念] : 以后会不会去刚兑？
  • [稀饭 回复 王睿晗] : 忘？望？
  • [水边上] : 大飞这么乐观，我们也就有信心了
  • [郑在忙…] : 希望早日拨开云雾
  • [龙之梦] : 一，同意行业已经稳住、进入恢复期的判断。二，行业还有2000多家平台，远远没有出清，大震之后可能余震不断。三，加快实施平台合规整改、检查、备案，真实通过备案的不会很多。

问：飞哥，365官群反映信息不乐观，单总视频承诺8月底恢复正常将难以实现，平台又不善于与投资人沟通与引导正能量，公告与承诺多次落空。圈内好多人重仓，为保护大家资产安全，请飞哥再次出面与单总沟通，了解一下具体进展及下一步打算，帮助单总出一些建议；向圈友通报365真实情况和应有的应对之策；同时向单总转达大多数投资人相信并会支持平台采取的一切积极措施，希望单总与各股东有担当有实际行动，一起努力渡过难关。

【羿飞】 回答：这个情况，我一点都不意外。其实前面投友发过一个帖子，把365目前的情况说的很清楚，其中至少90%的准确性。
所以当时单总说月底就能解决，我个人感觉是过于乐观了，难度很大，应该没有那么快。但是情况在明显好转也是现实，逾期在持续的回收和还款，甚至出现了很大提前还款的标，这说明单总的乐观并不是空穴来风，只不过是跟预期的时间点有差距。比如多用半个月或者一个月，最终恢复到正常。

很多人，总是抨击单总承诺落空。目前的市场情况，变动太大，谁能保证肯定在什么时间做到什么呢？给予一些理解吧，单总没有甩包，是在向着恢复正常努力呢。如果有人在期间有雷台，你们对比一下就能明白区别。为啥要对那些对你残酷的平台无可奈何，对在为投资人努力的平台，却无尽的指责和抱怨。
若人心过于让人心凉，单总最终放弃了，我想大家会后悔自己所做所说的一切。365的各群，都隐藏着一批银河安排的黑子，不断的在带节奏。他们希望能把平台搞倒，这样就能不还款了。而单总实际上现在是有好牌的，但是不能说，因为只要说了，就会被黑子们破坏。
大家应该做的事情，是支持单总，抨击黑嘴，而不是不断对平台职责，让单总失去努力的决心。

  • [有书则强] : 是的，比起那些“说走我就走”的平台，365单总算是有担当的了。我们还是不要添乱的好，淡定淡定。
  • [木屐] : 赞同
  • [三儿] : 不要让银河得逞，唱衰365只会让银河窃喜
  • [大吴] : 期待单总的好牌😊
  • [西北狼] : 我6月开始就有逾期，到现在全部一直逾期，没那么乐观，谨慎对待之。
  • [龙之梦] : 这是我前天早上的提问，谢谢飞哥的回应。
    我经历过淮安标风波，对单总的人品与担当是十分信任的，但今年以来在引入大股权资本上显得急躁不慎重，是造成平台现状的内因，还好及时果断切割了银河。我和大多数365投资人坚定的相信并支持单总，也希望股东们在危机面前要有担当，支持平台渡过难关。
  • [007] : 是啊，当年淮安标的处理就充分体现了365易贷和单总的担当，相信这次也能妥善处理好。大家共同努力，挺过去就是光明前景！
  • [郑在忙…] : 至少做到不添乱吧

问：你好，有没有长期看好的科技或者文化产业的股票推荐下

【羿飞】 回答：这两类普遍在创业板，创业板是本轮行情跌的最凶的，我一直在提醒大家规避风险。目前仍然不建议碰，最近比较好的是工程机械板块，因为最近经济不景气，国家的财政计划调整，开始大量投资，大概率还是从基建入手比较直接。这样工程机械类最先复苏，而且上一轮大放水的2008年，当时基建购入的设备10年折旧期已到，最近新购需求会很大。

  • [木火通明] : 谢谢解答
  • [向日葵 回复 木火通明] : 柳工，了解一下
  • [爱奇先生] : 个人感觉～除非上半年或更早就布局了基建机械工程板块～现在没布局的还是不要上了。因为我身边几个股票套了五六年还没出套的都在嚷嚷着要上水泥基建路桥了[呲牙]～个人以为人尽皆知的消息没有投资价值
  • [木火通明 回复 向日葵] : 谢谢

问：飞哥好，杭州拱墅区的（创道国际的商铺）以前是集体自留用地，卖给开发商，现在开盘销售，但是没有预售许可证，开发商说产权先签到开发商的名下，然后直接过户给我去办个网签就可以了，交房日期是2019年12月，现在也交了10万，接下来要在8月25日交，出现这样的情况怎么处理好，要求退定金还是继续交钱，（当时是我老婆去趟杭州就交了10万，现在才知道这个项目没有预售许可证）

【羿飞】 回答：大多数时候没太大问题，这样操作的情况并不少见。这样做说明开发商资金比较紧，提前卖房回笼资金。
购房合同有效，风险是开发商搞一房多卖的骗局，如果后买的人是手续齐全的，那个人才能主张到所有权。还有就是有些小开发商资金链断了跑路，你也不能保障自己的权益。
如果是能谈一下原额退定金是比较好的，有钱在，房子哪里买不到啊？因为他们证不全，不退款你就闹，估计他们得同意。

  • [一秒一个梦] : 这不是和最近九堡那个楼盘差不多

问：知道飞哥是从小就开始玩游戏，我想请问，现在在教育子女方面，游戏与日常生活学习如何取舍或平衡，有什么心得，谢谢

【羿飞】 回答：很抱歉，在子女教育方面，我不是很成功。或者说，我不是很确定是不是成功。因为孩子比较贪玩，学习成绩一般般。但是，谁也不知道未来的社会是什么样的， 每个人会有什么发展，我也不太认同拿分数去衡量一个孩子，所以说是失败也谈不上。我觉得能适应社会就可以吧，别急着给小朋友定标准，做结论。

就自己过去的经历，说实话，上学念书学的东西基本都没用上过。一直以来我就是爱玩，觉得什么有趣就做什么，然后 慢慢的把这个好玩的东西做成生意。
所以在孩子的教育上，我比较注重诚信，就是说过的一定要算话，违约或者撒谎惩戒非常严重。其次是比较鼓励多和同学玩，培养好社交本性。其他的我都不太关注了，学习成绩，特长班什么的比较佛系。

其实大家都没必要着急的吧，社会是在发展的。我们虽然每个人都有高低不同的成绩，但是几乎所有人，都比父辈，祖辈要有成绩，爷爷们月工资才几十块，父母月工资几百块，现在的人随便都是几千块。生活质量，教育素质都比以前好太多了。
因为社会整体发展，我们稍微努力一下，就比上一代强。也就是说，你的孩子肯定还是会比你强，你整天要给一个比你强的人做规划，人家能服吗？你累不累？

  • [羿飞] : 对于孩子的教育，在每个人心中，都是不一样的标准。
    我有个同学，当老师的。每次谈到教育孩子话题，他就冲我发火说我太不负责任。
    他觉得，我得辞掉工作，陪着孩子回老家，守着他考完大学，这才叫负责任的家长。不舍得为孩子付出一切的，都是垃圾！[囧]
  • [风扬 回复 羿飞] : 教育上我也比较佛系，飞哥在财商教育上重视吧？
  • [三儿 回复 羿飞] : 父母是老师的孩子自杀率特别高。
  • [xlzhang] : 看到飞哥如此佛系，着实佩服。我家也是一直主张快乐教育，小学以前任何东西都没学过。上了小学后真的就看到差距了，全班孩子都会，我家慢半拍。有时候真是着急呀[尴尬]
  • [羿飞 回复 xlzhang] : 看开点就对了。唯一可以肯定的是，我们的对成功的定义和价值观，和孩子们长大以后那个时候的价值观一定是不同的。儿孙自有儿孙福。
    [呲牙]在我老爸眼里，我一直是个废柴，因为他觉得我不懂技术，没有一技之长傍身。
  • [羿飞 回复 风扬] : 我比较鼓励孩子攒钱，以及跟同学做点小生意。
  • [风扬 回复 羿飞] : 我经常和女儿聊复利，算复利，投资渠道千千万万，懂得复利才是根本[呲牙]
  • [yxs] : 飞哥佛系那是因为有钱

问：飞哥说说池子中的基金吧？需要调整不？打算最近分批买入持有几年。

【羿飞】 回答：有持有几年的心理预期和打算，而且又是分批，策略风险很小。就买吧，现在这种价格。

  • [一秒一个梦] : 目前可以做指数基金。当做强制储蓄也不错
  • [第五] : 飞哥近期还打算更新基金池吗？

问：飞哥你好，怎么看房租上涨这件事？下半年还有3个月，留给人民币加息的操作空间好像不多了。

【羿飞】 回答：房租上涨，应该主要发生在1、2线城市，因为房价开始回调了，房租就会有所上涨。还有就是最近行为热点曝光的，几个长租公寓在抢市场，拼命补贴拿房源，拉高了成本，导致了价格快速上涨，应该不代表普遍价格。
我自己有房子出租的，确实有上涨，但是没有一年上涨20%那么夸张。多年以来，平均年上涨8-10%有的。
