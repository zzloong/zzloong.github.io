2018-08-20问答记录

发表于 2018-08-20 更新于 2021-04-09 分类于 Stock
本文字数： 2.2k 阅读时长 ≈ 2 分钟

问：飞哥是律师出身吧？请教个竞业协议的问题，在公司干了十多年了，一直在技术岗。公司近期在整改规范，要补签竞业协议，可老板耍赖，搞了个离职后月补贴2000的竞业限制，5%左右，说只是形式而已，要我干到退休。不想签这种不平等条约，僵着，飞哥有好建议吗？谢谢！

【羿飞】 回答：你的这个问题，要按照逻辑顺序来捋。
第一，这个工作对你来说是不是可替代的。假如你不用太费力就能找到类似的岗位，就可以明确表示烦感，明说我没跳槽打算，但是这种协议我看着恶心，绝对不签，如果非让我签，我现在就走换换地方。
第二种可能，就是这个工作对你来说也确实很重要，换工作找到同等待遇是概率比较低的。那么可能需要委曲求全来接受这个协议。
你们公司的协议存在不少问题：
1.竞业禁止范围滥用，竞业禁止一般只是用来约束公司高级管理岗位人员，或者有明确保密义务的岗位。经理，总经理，财务总监等等，并不是所有岗位都有义务签这个。
2.给的补贴不合理，应该至少是离职前12个月的平均工资30%以上。注意不是你现在的工资，是离职前12个月。按现在的也至少应该6000以上。
3.没有提到违约的赔偿金额，那么以后即使诉讼也没有合理的赔偿主张。

鉴于这种乱立内容的协议，你可以有两个选项：
1.提出不合理点，说你不是关键岗位，拒签。这样其实和打算离职是一样的，明确的拒签了，以后就会给你穿小鞋或者不重用你，最终你需要跳槽。
2.心理对这些不合理点有数，但是不指出。先找理由拖着，等公司6、7成的同事都签过了，你再签。因为有一定概率在前面执行的过程中就有人提出了协议内容的不合理观点，事情推了一半进行不下去了。所以你既没有得罪人，也没签。
3.拖了一定时间以后，已经比例非常高了，签完了，也没关系。因为这种大范围签订的东西，公司根本没有能力全部执行，执行成本太高。
如果诉你这种小概率的事情真的发生了，因为他们合同中存在诸多的不合理，将来这个事情也不难解决。比如离职第一个月，你就自己先主动去劳动仲裁告公司补偿标准不足，必须按劳动法最低标准补偿。然后大肆在公司老同事之间传播你在诉公司。这样公司因为不可能做到给全体员工更换补偿30%的新协议，只能跟你协商谅解解除你之前所签订的不合理协议。

其实这种大范围签订的竞业合同就是诈唬人，没有什么可执行力。多半是公司新来了一个什么副总或者人力总监之类的人，跟老板拍马屁提改革方案，老板一高兴信了，就搞了一波这种运动性的改革。
这种不合理的规则，很难推，大概率是推了一半无疾而终。

  • [羿飞] : 我不是专业律师，其中大多数内容是在公司管理过程中的经验心得。如圈友有专业律师观点， 欢迎补充和指正。
  • [北市仲垣] : 首先，要明确这个竞业协议是作为劳动合同的补充条款，还是用工单位和劳动者签署的独立协议，在明确是劳动合同补充条款这个前提下，再提示几点：1.竞业限制是有期限的，最长是解除或终止劳动合同之日起2年；2.竞业限制是有主体要求的，这个大飞已经讲过，实务操作是央企、国企以划定为涉秘人员并签订保密协议的群体为限，其他人员即使签订，竞业限制条款无效；3.竞业限制内容是基于双方约定，限制的范围、地域、期限、补偿金额都需要双方协商和明确，约定不明视为没有约定，如果不是劳动合同的补充条款，可以拒签。4.竞业限制是有维持条件的，自解除或终止劳动合同之日起，必须按月足额支付补偿金，如果没有履行，协议即行失效。另外，竞业限制的实务作用极其有限，稍微不慎，就终止。举个例子，一家知名央企，一位专业副总师离职，就一个月没按期收到补偿金，立即发律师函通知终止协议，然后自己成立一家公司开展相同业务，直接把这家央企的该领域业务挤垮。

  • [Kelvin Fis] : 环境还是如此不堪啊，明摆着逼人伪操作！*
  • [雾风] : 月薪4万的壕！如果这只是工资收入，那说明你的岗位对老板而言已经非常重要了
  • [天地泉 回复 北市仲垣] : 请问如果不是劳动合同补充协议，而是独立协议，会有什么额外风险吗？

【羿飞】 回答：不靠谱，有的平台，老板忽然不见了，CEO一脸懵逼，客服知道什么啊？
说了很多次，目前可靠的，只有搞清楚资产。

  • [不老] : [微笑]如果是搞不清楚，特别担心的平台8.5可以走了。 如果搞得清楚的，9.5也舍不得。

问：飞哥好，想问一下，平台的债权如果是有担保方，在现在这个环境下能发挥什么作用吗，还是说大部分都是骗人的，平台自己的担保公司给自己担保。

【羿飞】 回答：基本上都是自己给自己担保。对担保方都有强制回购或者免责协议。那些担保啊，保险之类的，都是做给投资人看的噱头。所以还是得穿透资产端，看业务做的怎么样。

  • [向日葵] : 担保什么的，听听就好了，这么多平台出现逾期或者展期什么的，还不是没辙，不过小赢有众安加持还是放心的
  • [丹] : 最近宜人贷一直给我电话让我投 他有人保担保的是吗 应该很安全吧
  • [向日葵 回复 丹] : 可以的，部分标的有宜信担保
