2018-08-31问答记录

发表于 2018-08-31 更新于 2021-04-09 分类于 Stock
本文字数： 4.4k 阅读时长 ≈ 4 分钟

问：飞哥，能了解下目前王吉涛怎么样了？现在还在警察局里吗？饭团金服下一步怎么展开工作，还继续运营下去还是就良性退出了？

【羿飞】 回答：昨天已经发帖做了非常详细的说明，你可以翻一下看看。周末投监会开会，会讨论一些话题，到时候再给大家分享。

问：飞哥，这个最近传的比较多的AMC资产公司缓解p2p暴雷有作用吗?比如咱们p2p平台真的经营不下去了，他们会来回购债券吗？本金打个5折收走?

【羿飞】 回答：这个算是一个政策表态意义更多一些，没有什么可实践性。真的来收，给你1、2折，你要吗？

  • [林梦] : 人家收银行的不良资产都是1-2折，会出多高的价收网贷不良资产，精神上支持而己。
  • [三儿 回复 林梦] : 1到2折太夸张了，一般是3折。
  • [林梦 回复 三儿] : 要看银行和资产质量，前段时间他们收农发行百分之2.5-5
  • [三儿 回复 林梦] : 2.5%[捂脸] 这是信用债不良资产咩
  • [林梦 回复 三儿] : 林权抵押
  • [kevin] : 1-2折[捂脸]，趁火打劫的感觉

问：飞哥，怎么看最近平台所谓的良性清盘，企业会不会至少拖延而已，并不会真的进行兑付。如果出现了清盘，投资人应该做什么？

【羿飞】 回答：很难讲，没有什么执行保障是最难点。各地金融办都在筹划拟定标准的清盘流程，但是目前还没出台。主要还是看平台负责人的担当和责任心了，不排除大量的是以清盘的名义拖延，扯皮。

问：飞哥，今天收到一笔私募回款40多万，本计划投网贷的，遇到最近的雷潮，是不是稍微等等，待危机过去再出手？像丁丁这样的我认可很稳安全，且我持仓还相对较低，是不是可以趁现在好投就投了？另外，您估计这场危机大概还有多久才能平静？

【羿飞】 回答：丁丁的任总比较厉害，提前意识到车贷行业存在潜在风险，提前剥离了高风险的业务，所以现在属于车贷产品里，这次受伤最轻的，确实可以投一些。
但是，大局布局，还是可以再等等，行业回暖趋势更清晰一些。

  • [龙猫] : 私募收益率多少啊？
  • [云海奇观 回复 龙猫] : 是一个熬夜了2年的定增，还有一部分没有清算，据说目前是浮亏，这个我已经非常满意了，因为我还有一个亏18%的，所以坚持投网贷，入圈后私募已经不在再考虑。

问：飞哥，易融停标这个事怎么看待呀？谢谢。

【羿飞】 回答：就是做产品转型了，车贷这块暂停，直到车贷这个业务类型整体回暖。目前工作以催收，回款为主了。
过段时间，会做一些消费分期的产品上线。

  • [刀刀] : 转做消费贷的平台还不少

问：我问下，车E融这个模式，假如投的人少，平台借给车商的钱也少了，但是只要车商正常还款，就没问题是吧，
极端情况，平台没有钱借给车商了，就全靠车商到期还款，平台也没啥问题，也能熬过寒冬，是吧
另外，车商为什么不去银行借，而到平台来借？

【羿飞】 回答：这个我了解过。车e融，有机构资金和平台用户资金两个入口。投资人投的多的时候就优先卖给普通投资人，大家不想投的时候，车商也是要到期先还款的，投资人没续投，车e融会给介绍对接机构资金。
这个模式，到期无论如何车商要先还款，所以这次行业收缩，他们比较平稳的应对了，挺厉害。

  • [维A] : 车商为什么不找银行借?
  • [小敏 回复 维A] : 银行慢呗
  • [feiman] : 现在银行借钱不是一般难
  • [波牛 回复 feiman] : 现在银行不是开始放水吗
  • [ALICE] : 车e融重仓，希望没事。但今天给我打电话，问我有没有继续投资意向。
  • [少凯 回复 波牛] : 放水也不是谁都借啊，以前不借的照样不借
  • [少凯 回复 ALICE] : 打这个电话很正常吧，像你重仓的更会重点维护呀
  • [波牛 回复 少凯] : 我提现了，他们打过来问，是什么原因让我不继续投资，我说行业动荡，落袋为安

问：飞哥，朋友一公司，营业执照注册地址被收回，工商说地址异常，需改迁地址，他找了一家公司委托办理地址变更，提前告诉代办公司他们的营业执照经营范围包含生产，对方答应，签了合同交了全款。后来不断提交材料，然后到工商换照时说，经营范围需要删除生产这部分，朋友他不同意，代办公司又换了一个地址说这次能批下来，一个月后，又说批不下来，我朋友放弃了这家公司，要求退款。结果对方说要扣款3-4千元（合同款3万左右），说他们办理期间有耗费，朋友拒绝，后来妥协想尽快了结此事，再后来代办公司拒接电话，或接听人员说自己没有权限决定，负责人不正面回应。朋友想尽快要回合同款，不想走诉讼，不知飞哥和圈友有没有好的办法要回合同款？

【羿飞】 回答：我认为扣款几千块，拿回其他钱比较明智，这点钱，都不够诉讼成本和时间成本的。

  • [谦] : 这个的关键好像是其他的钱也不好拿回吧，这算不算合同诈骗？

问：请教大家个问题，一年多前中了山东青岛宝易得汽车众筹平台的雷 报案人数和金额也远超立案金额了，可当地经侦打死不立案 在目前这个情况下，找那个部门投诉最有效 麻烦大家指点

【羿飞】 回答：我一直特别反对大家投众筹项目，就是因为这种项目在法律结构上就不是债权，如果赔了就是赔了，很难维权和立案。我感觉除了花钱买教训之外，没有太好的方案。

问：飞哥，你是怎样辨别网贷平台底层资产的真实性的？会去考察，做尽调吗？

【羿飞】 回答：我重点关注的平台，是进行过考察和尽调的，但是不能长期依赖这个。因为总是在动态变化的。长期看要依赖持续的信息关注，和众多投资人经常交流，因为总是有人去平台看的，实际上是通过大家来实现对平台的持续关注，靠一个人是很难做到的。

  • [阿狼] : 确实，风险管理是一个动态过程。

问：飞哥你好，咨询一下，我目前30岁。有渠道可以买到上海和北京的户口。目前上海的大概是10万。如果不买可能需要排队6、7年才能通过正规渠道。北京的大概70多万，要是不买以后可能买不到了。不知道该怎么选择。

【羿飞】 回答：目前来说，户口主要是跟孩子念书有关，可以在户口所在地参加高考。跟其他方面没有太大的关系和区别了。所以看你自己未来规划，如果孩子要走高考，就要买这个户口，因为一线城市录取分数线会有一些优势。如果是打算孩子出国念书的，就没必要花这个钱了。

  • [云] : 高考的话肯定北京户口啊。
  • [修一] : 求渠道
  • [空中婴儿] : 为高考可以去天津
  • [松东河沙] : 上海户口10万[惊讶]李强的小舅子👍
  • [feiman] : 10万一个，很多人可以买到你破产
  • [羿飞 回复 松东河沙] : [憨笑]圈子里卧虎藏龙
  • [7676刘昆] : 看孩子是不是学习这块料了

问：飞哥好，同学们好，宜人贷的宜定盈符合这次108条备案细则吗，有没有合规风险？宜人贷的交易数据为什么不对贷罗盘开放交易数据？

【羿飞】 回答：宜人贷就不用多费心思了，如果按照分散原则，比如算五个平台中的一个，放心投。

问：飞哥，这几天网贷的回款不敢再投进去了，想布置一些基金，感觉现在股市也不会再有大的下跌了，能推荐一些合适的基金吗？

【羿飞】 回答：之前分享的长牛型，定投着就挺好。

问：作为没有经历过历来股市大底的人，请教飞哥，现在这种独立于全世界的低迷，跟以往比较是否是有先例？

【羿飞】 回答：肯定是有先例的，而且可以说是一直如此。比如不管其他市场如何涨，他就是一直阴跌。也比如即使全球经济不太好，也可能有行情，2007年超级泡沫，2009年大反弹，2015年的暴涨，都是独立行情。
中国因为不完全是开放的金融市场，其实一直以来都是挺独立的行情。短期受美国有一些影响，但是中长期，是自己的独立行情。
影响最大的因素是资金，其次是热点概念，其次是实际业绩支撑。
中国股市的特点是70%的时候慢跌，30%的时候猛涨。所以如果不是长期投资，在这个市场里浸泡的人，即使市场指数整体是上涨的，也很容易赔钱，抓不到偶尔的上涨行情。大多数人还是定投基金吧。

  • [Kelvin Fis] : 之前医药基没出 [撇嘴]*

问：飞哥你好，最近没人问股票问题了吗？还是您选择性的暂时不回答了。我觉得这块的交流还是应该继续啊。不能因为最近行情不好，就不研究了。行情好的时候，哪有时间研究。希望飞哥给出一些针对性的建议或者学习的方向。希望能够一生都在股市赛跑。谢谢！

【羿飞】 回答：最近因为市场不景气，确实问的人很少。而且最近因为网贷行业动荡，我个人的精力主要是放到稳平台，守护大家财富这个方向上去了。虽然做的不算完美，但是还是要尽全力的，能做多少做多少， 否则良心过不去。
后面，网贷行业渐渐稳定下来，我也会逐步的再把精力挪回来。所以并不是在市场不景气的时候放弃研究，是近期确实没有时间。
综合来说，大家持有的基金池，基本上表现相对大盘还是不错的，大家做定投是没有什么大问题的。股票池的话，最近几个月表现的也比大盘要好。过段时间我精力够了，会好好筛选，再更替一些。

再给大家重复一些，股市方面的投资比例基金7成股票3成比较好，基金定投策略，这样不会犯大错误。
还有就是绝对！绝对！绝对！不要上杠杆，不要借钱投资。这个事情以前我不断的强调和重复，很多人当耳旁风。最近大家一起经历了如此局略动荡的各种市场动荡，应该已经知道风险的厉害，以后投资风格都会稳健很多了。

  • [庄懿鸣] : 之前一直在被杠杆诱惑着。经历了这次行业动荡，庆幸自己没冲动上杠杆。
  • [书签] : 守护住现有财富，才可以设想其他投资。
  • [顺] : 以前当耳边风了，其他圈友不要再犯我的错误了[流泪]
  • [66 回复 顺] : 守住底线，可谓血的教训。

问：飞哥，黄金上周跌到1159，后期走势你怎么看？ 很久没说黄金了，麻烦飞哥抽空看看近期走势

【羿飞】 回答：最近看起来强势美元已经到了中期顶部，今天的反弹动力已经薄弱，黄金建仓一些我觉得可以的。
今年黄金最大的价值其实是保值，因为人民币跟美元是相反走势， 涨跌之间，黄金价格基本平稳。这看起来是没赚钱，但是在市场如此动荡的情况下，其实已经属于表现不错的产品。

  • [罗晓莹] : 飞哥，现在用什么工具建仓黄金呢？之前有人提问的时候您说互联网那几家黄金最好逐步退出来，现在都不知道用什么工具买黄金安全？
  • [xlzhang 回复 罗晓莹] : 我也想问这个问题
  • [羿飞 回复 罗晓莹] : 大概只能实物或者纸黄金了。
  • [牛牛 回复 羿飞] : 飞哥的黄金钱包 管家 国美三个都已经退出了吗，车上还有40呢
  • [大吴] : 互联网企业不让做黄金账户服务了，但黄金管家和黄金钱包都签了平安银行了，能否绕过监管呢，国美黄金还是到期就撤吧，已经明显收缩业务了

问：飞哥，对创业板您怎么看，目前已经跌倒1400点附近了，您觉得它还高估吗？可以买一下创业板基金了吗？

【羿飞】 回答：创业板不行的，之前泡沫实在是太大了，股东们都是专业减持的，还是别碰。

问：想问飞哥，中国长城和佛塑股份这两支股票如何，现在可以买一部份吗？谢谢

【羿飞】 回答：这两个都不是太好，基本业绩不行，目前市场上跌透了，绩优股，市盈率10倍左右的遍地都是，不要碰这种业绩不行的了，没必要。

  • [静心] : 谢谢飞哥[微笑]

问：羿飞老师，今天突然发现之前忘了赎回的基金161025亏了百分之六十三，(富国中证移动互联网指数分级)，以前不舍得割肉，后来就忘了，现在补仓吗还是死等？该怎么办啊？

【羿飞】 回答：这是典型2015年追热点创业板的接盘侠基金。这样的基金不管是能力和道德都是没有价值的。尽快换吧，补仓或者死等都不对。
