2018-08-15问答记录

问：飞哥你好，最近有没有趁机主动清盘的平台？这样其实他们什么都不干，把这几年的投资人收益收割了。我感觉真是投平台就是投老板了。主动清盘的其实法律上也没有什么过错，完全合规。不知道飞哥怎么看？我感觉很让投资人痛心。

【羿飞】 回答：行业大地震，各种情况都有，如果按照从好到差来排序(约等同于资金可以得到保障比例的排序)，各种情况应该是：
平稳渡过(证明实力和品牌够强)——部分逾期展期(业务真实受资产端催收影响小问题)——计划类切底层资产(资产端问题不大流动性扛不住)——全部逾期展期(真实资产不是骗子，但风控水平低)——清盘(变相甩包袱，不干了让投资人承担窟窿)——跑路(资产端有一定自融并且无能力堵窟窿)——诈骗(几乎全部造假自融，自始至终就是一场戏，遇上了只能认倒霉吧)。

  • [羿飞] : 我认为前3个类型，会有动荡和影响，基本资金不会受损。
  • [Leon Lu] : 银豆属于跑路还是诈骗啊？
  • [龙之梦] : 平台主动清盘一定要让监管部门、经侦加入监管，股东以破产法承担连带责任，提高出借人偿还率，不能让股东赚钱走路。
  • [姗姗来迟 回复 Leon Lu] : 我认为诈骗
  • [Leon Lu 回复 姗姗来迟] : JZ只是立案非法集资[流泪]
  • [龙之梦 回复 Leon Lu] : 清盘也须报经侦受控的

问：飞哥，央行开始放水了，请问什么时候会传导到四五线城市？等到四五线的时候具体会对我们的生活和房产有什么样的影响呢？近期有购房打算不知是该现在立即出手还是等水到四五线城市再买呢？

【羿飞】 回答：目前地产的状态，是一线已经调整过，基本企稳。二线刚开始跌，三线火热，四五线启动不久的状态。不过小城市如果不是有人口流入特征的，涨价幅度大概不会有之前1、2线那波的幅度大，后续可能后劲不足，以后房子没有人接盘。

问：飞哥，看了饭团VIP群“高手”的聊天记录，有几个疑问，有时间时回复一下，谢谢飞哥👍
1.赞成切底层资产，而且感觉越早切越好，但是会像“高手”说的，切底层资产就是打算清盘？王总心里也是这么想的？
2.感觉毛利率没“高手”说的那么低吧，6亿待收，600-1000万毛利，1%-1.5%的毛利率是不是少了点儿？
3.车贷的毛利率多少？易融的待收，毛利，综合成本，利润情况是怎样的？

【羿飞】 回答：网贷行业的整个链条的营业利润，是在5-10%之间。但是这个利润，会分布在资产端和资金端两端来分配。一般的公司多数是在资金端分较少的利润，在资产端留较多的利润。
具体的手法就是给调整平台给资金端的批发资金价格。因为资金端是受到监管的，留了太多利润意义不大，除非是要上市了。利润留到资产端非常方便使用和分配，这是常见的手法。而且因为回购坏账消化坏账也是资产端负责的，资产端需要有更强的实力。
两端综合，净利润是2-5%之间是比较常见的，主要是看坏账控制水平和营销费用控制水平，做的不好的亏损的也很多。

  • [Viavia] : 谢谢飞哥的回复。。。唉，自己投的一个车贷平台已经垫付不起了，不过老板在加大催收力度。。。
  • [王姐] : 切底层我觉得不是清盘，这个之前飞哥说过，很清楚的介绍过，提问的朋友可以搜搜，简单说就是把你的投资都变成一个个的散标了，这个标是多少时间你就拿多久，原来是错配时间的，比如你投小饭团类似活期，实际上真正的债权并不是活期，你转让提现，由其他人再来接手这个债权。但是一旦提款人多，接手的人少，你就无法提现了，因为没人接你的债权。现在切底了，回款不是来源于其他投资人，而是来源于借款人的还款，只要他不赖账，你就有钱拿。
  • [Viavia 回复 王姐] : 谢谢王姐，这个提问是切底之前的。。。
  • [高懿] : 资金端、资产端都指哪方。
  • [木屐 回复 高懿] : 资金端是饭团，资产端是小贷公司，不止一个，与饭团是合作关系

问：这波雷潮应该就是 国家去杠杆 和 扫黑除恶 的结果。飞哥可否从此角度去分析下逾期平台 企业贷 车贷 和 房贷 的催收前景？

【羿飞】 回答：目前阶段还是比较艰难的。企业贷可控性比较低，车贷靠谱一些但是因为警方影响暂时催收进度受影响，给很多平台带来了垫付压力，消耗了很多现金流。但是最近监管风向有所转变，开始打击逃废债了，未来应该会逐步好赚。

  • [aLive青丝几渐] : 最重要是动了奶酪。引发了过量的资金流出银行。
  • [波牛] : 所有还在熊市探底
  • [Kai] : 为啥可控性低[撇嘴]
  • [WANG 回复 Kai] : 看看银河天成。。
  • [🐚] : 并不认为去杠杆导致。
    我倒是认为，某些案件的判决，特别是终审的判决，以及由此案发起的偏重一个方面的特别行动，对某些恶势力传递了信息。让他们以为做老赖成本很低，对他们很宽松。
    现在雷厉风行的往回抓，自然会对老赖严打一番，通过多方联手，对整个过程中的合法规范是巨大的推进。有了标准，各方做事就明确了，灰色地带少了，按照这个办理，就是合法催收了。只是执行起来之后，老赖知道疼才能有显著的效果。这个只能等了

问：飞哥。这一波的p2p还有股票，黄金，几乎所有的资产品都在下跌。这次和你08年碰到的那次金融危机。你觉得哪次严重。

【羿飞】 回答：实不相瞒，最近这波经济整体形式的急剧下滑和劣化，其规模和速度超出想像，把我打的也是蒙头转向。
不过要说程度，比起2008年来说还是差多了。那次是房价腰斩，股市跌65%。全面的债务违约，银行都要濒临破产。所以才有了四万亿。
这次的话，我自身受伤不多，主要是得益于在08年那次教训中，总结到的经验，就是不上杠杆，投资一定要用自己的资金。以及不能断了现金流， 一定不能只靠投资活着，要有能提供现金流的主业。
有了这两点支撑，我觉得自己今年最多是少赚，赔钱应该不至于。但是今年确实看到太多人遭遇了沉重的打击，心很痛，钱都是一点点积累的，一下就灰飞烟灭了。希望大家也能在市场的动荡中总结到自己的经验和教训，为以后打下基础。经济都是有周期的， 每隔一段时间来一次突发的危机是不可避免的。而且时间点无从预测，谁能准确预测那就真的是上天了。就像今年的危机，春节后各方面都是比较繁荣的，但是突发的贸易战，几个月就把所有的数据打烂。
所以，能救我们的，只有良好的纪律性。
