问：飞哥好，看了您公众号的文章，图中阐述的逻辑没想明白，分税制→地方政府权限大→土地财政→影响房产税实施。之前听郎某说的逻辑（比较能理解）：分税制→地方政府收入减少→土地财政。麻烦飞哥再对小白解释一下图中的逻辑，尤其是土地财政影响房产税的部分，多谢多谢。

【羿飞】 回答：道理很简单啊，分税制下，地方政府分走了一部分税源。但是权力大了，所以可以靠炒作地皮玩土地财政。所以中央如果要想结束土地财政模式，就得给地方解决新的财政收入来源，即物业税。
但是这个税种是长期总额收的多，但是短期不如卖土地来得快，地方政府是不愿意接受的，会抗拒会博弈。这需要一个过程，可能需要国家拿出几个靠新的税源也能过的好的试验案例出来，才能逐步推广开。
其实真的是物业税收的多，每年1.5-2%之间，70年就把一套房子钱交出去了。说是永久产权，其实是永远要租自己的房子住了。地方政府如果放眼长远，应该接受。但是领导们都是轮替的，谁愿意牺牲自己在位期间的发展数据，给后面的领导做嫁衣呢？这不科学。

    电视剧天道里的丁元英原型就是禅师，可以看一看
  • [南木 回复 🐚] : [强][强][强][强]

问：武汉齐海商务顾问有限公司和一起好是主要合作伙伴，合作伙伴有套路贷的行为，是否表明一起好的风控还是在较大隐患呢？飞哥可以解读一下这类事件吗？

【羿飞】 回答：这个可能是你信息太陈旧了，齐海商务在2016年，就是近2年前，就由短融网并购了，脱离了和一起好的合作。当时对一起好的待收形成了一轮收缩，用了一段时间去消化。所以最近的事情跟一起好没什么关联，有时候真的是塞翁失马焉知非福啊……

  • [可爱多] : 那是不是短融网风险会增大？

问：飞哥你好，进群后我发现圈友挺厉害的，前段时间说的那个瑞茂通要涨，没几天来一波；最近说的这个绿地，好像也起来一些。短期这些行情都是怎么发现的？有没有什么指标反映哪个股票最近有要活跃的迹象。可否说一说。

【羿飞】 回答：本身是质地优秀，随时可能发生行情。当然主要是偶尔了，其实每天都有人说热门股的，刚好你自己持有哪个，就会特别关注这个的话题，让你产生错觉，这是很多人的思维习惯。

  • [王威] : 说曹操曹操就到。这只是因为一旦发生了，人们就会加深这个印象，但是更多的时候不发生，就自动屏蔽忽略不计了。
  • [白云] : 这就是典型的孕妇效应
  • [WANG] : 股票大部分人观点都是错的，短期行情不确定性大，瑞茂通和绿地低估，被套也不怕，平安格力前几天也有几天反弹，很多人跟了就悲剧了。低估的涨10%进去也没事。

问：飞哥，有兴趣帮了解关注下网贷平台小微时贷吗？福建车贷头部平台，全抵押质押车贷，2015年成立，累积交易19亿，待收近6亿，直营60多家门店，江西银行存管，关键是能自动投标，24个月利息还在15%之上。是否可以作为备选替换平台？谢谢！几天未见回复，再次提问的 😄

【羿飞】 回答：这个我没太接触过，以后可以了解一下，目前给不出什么建议。

  • [空中婴儿] : 我去年就布了一些。目前看来还可以。
  • [贾政] : 前一阵子从贷罗盘的示范3里剔除了，我现在在慢慢退出。
  • [Kelvin Fis 回复 贾政] : 广信贷也从贷罗盘的示范基金撤了*
  • [贾政] : 广信贷有大侠在不怕的

问：飞哥 你一般从哪里拿榜单里平台的数据 如何辨别他们有没有作假呢？

【羿飞】 回答：尽量去跟平台直接拿，这样数据最精准。然后通过各三方数据进行交叉对比，如果都比较相似，优先信平台提供的，因为准确。如果差距大，找平台交流原因，一般是三方抓取出错，爬虫程序出差很正常。
如果是发现恶意作假，那么就会删除，不再采纳。但是目前位置没有发生过平台提交数据作假的，因为他们也知道有三方数据可以参考，很快就会露馅。

问：飞哥，能不能帮忙从技术层面上分析一下交大昂立是不是处于上升通道？之前5.4买的盈利卖掉了，现在是否可以重新买回？

【羿飞】 回答：都没多少营收了，就剩个壳子了，这种股票还是不要碰了。

  • [神和权力] : 买股票记住：是好的行业+好的公司+好的时机。3个条件；反之，缺少任何一个则是麦出的时机
  • [嘟嘟 回复 神和权力] : 谢谢🙏不过怎么做到你说的三好呢
  • [神和权力] : 需要自身努力学习，试想如果大家都OK了，怎么会有大部分人是赔钱的，投资从来不是一项容易的事，凡事一项进入条件越简单的行业，对个人的要求越高

问：飞哥，你好，听说过趣养鱼吗？朋友推荐是现在的互联网农业，利息高。现在是我劝他别投，他劝我别投p2p!有点想不明白，希望飞哥指点一下利弊谢谢

【羿飞】 回答：没听说过，这种乱七八糟名目的，以返利，回购产品为名义的，都是骗子。二十年前就有养蚂蚁的闹剧了，韭菜还是割不完。

*问：今天收到【车e融】短信：
尊敬的用户您好，受二手车行业发展影响，为了能够为广大用户朋友提供更为公平，稳定的购买环境与服务，平台将于2018年5月1日暂时下线“宜优享180”，本次下线不会影响您前期应享有的正常权益，详情见平台公告（）。

请教飞哥并诸位：近期车市有何变化，对车e融平台及后期收益率会有何影响，谢谢！*

问：飞哥，002385大北农，做猪饲料的，从10年跌倒现在5块钱，现年线企稳，猪肉价格已经连超跌，国家现在想扩大内需，通胀周期又开始了。公司发公告增持3亿，现在可以入手么？

【羿飞】 回答：属于有实际业绩支撑的，但是传统产业，增长空间比较小，所以目前处于中性，如果打算配置农业板块，可以配置一份，但不要重仓。因为目前市场上特别便宜的股票实在太多了，更有优势。

  • [WANG] : 当初找这个股准备对冲贸易战，但是特大嘴满嘴放炮，估计狼来了喊多了，农业板块表现并不持续，换成了探底回升的华泰。

问：飞哥 福田汽车后势您怎么看？

【羿飞】 回答：新能源汽车还是大势所趋的。福田去年没有拿到补贴，延后了，业绩比较差，不过会体现在今年，所以今年业绩大概率超预期。

问：飞哥好，帮讲讲近期车贷的状况吧？之前傻傻地以为车贷有抵押，相对安全。现在看几个平台的动作，特别是图腾贷，突然感觉车贷也不是那么好做的了，是不是会成为政策掠杀的首要对象啊？谢谢！

【羿飞】 回答：车贷长期看仍是优良资产，毕竟是小微，有比较有流动性容易变现。这一波对催收进行了限制以后，可能主要是以后各平台再防弹，风控要严格一些了，以前是认为可能违约，但是催收能搞定的也会放，现在的话，因为催收手段受到限制，可能放的就会谨慎一些了，所以缺标会延续一段时间。
