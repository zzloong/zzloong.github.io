问：飞哥好，圈友好，有个疑惑提出来，如有错误可笑的逻辑还请大家担待：中国95年开始这么多年M2增幅年均15%左右，到了25万亿美元左右世界最高，约是GDP的两倍。然而人民币不但没贬值，对比95年的8.27左右还升值了近30%，一般来说超发货币，必然会贬值呀，比如印度，俄罗斯。中国这种情况是什么造成的呢？
有的说是房地产的蓄水池起到了作用？
有的说是外汇管制作用？

所以钱根本就没多印过，或者说是多的不严重。这两年的话，开始收紧一些了，m2增速低了一些，我们就看到明显的钱紧，以及人民币升值了。
未来长期看，人民币仍然是升值趋势。

  • [龙猫] : 那说明回购是好事啊。可以抄底

问：飞哥，和信贷、团贷网跟银湖网，三家如果要做个排序的话，你会怎么排？都会高过翼龙贷吗？谢谢！

【羿飞】 回答：翼龙、团贷、和信、银湖，这样排。其实每期看评级不就行了吗？

  • [七灵八羅] : 团贷那么高[疑问]

问：昨天，云天化出了《2017年财报》，飞哥能解读下吗?是否能结束近期的下跌趋势，开始走强?

【羿飞】 回答：是2017年的修订，不是新的财报，内容太多了，没法去看，应该不会有大的变动。最近主要是2季度报，因为去年上半年是巨亏的，所以这次二季度的上升幅度一定是很大的，主营产品价格都在飙升。
