问：飞哥，我小白井底之蛙，一直看不懂海外很多存款负利率的市场，可否帮科普下，谢谢！现在国际化程度这么高了，他们不会搞海外投资吗？或者说就没有机构巧妙地来吃这块蛋糕吗？

【羿飞】 回答：负利率是因为国家经济增长不好，消费不振。所以用负利率，把老百姓的钱逼出来消费的，一般负利率的情况下，本币容易升值，所以即使利率亏一些，存钱也划算。
至于你说投资，其实有这个偏好的人，在任何国家都是少数的，都不会超过全部人群的10%，大多数人的钱还是话在生活和消费了。

  • [天地泉] : 我也不太明白，‘‘负利率本币容易升值‘’，之前理解是加息才会让热钱涌入导致本币升值，就像分析美元一样。另外本币负利率，那本国机构应该是将本币兑换成其他加息币种，这样也导致本币流出贬值，有点疑惑。
  • [羿飞 回复 天地泉] : 我说的‘‘负利率本币容易升值‘’，是一种结论和表象。实际上真实的发生过程，是本币升值，然后老百姓都不爱消费，都把钱留下来存到银行了。反正存款就能升值了，然后才导致zf希望把大家的钱逼出来，才定义了负利率，这是惩罚性政策。
    在古代，最常见的就是这种情况，无钱可用的钱荒。老百姓不消费，把银子都埋到了低下，社会上出现严重的通货紧缩。所有东西的物价都不是现在这样年年上涨，而是年年跌价，老百姓就更不愿意花钱，而是埋起来。
    古代大多数经济崩溃，朝代覆灭都是因为这个引起的。即朝廷无钱可用，朝廷官造币掺假。
  • [三儿 回复 天地泉] : 现在美元的加息和贬值在同时发生。
  • [龙猫 回复 羿飞] : 古代有没有通货膨胀的呢
  • [羿飞 回复 龙猫] : 宋朝最早发明了“交子”这个纸币，这是世界上第一个真正意义上的法币。所以宋朝有了世界上最繁荣的经济成就，占据当时整个世界GDP的近50%。就经济成就而言，宋强于唐，结果很遗憾大宋因为军事薄弱，经济繁荣被终止。
    我个人一直有个妄想，就是如果大宋的时候就已经发展到了今天这种靠装备的热兵器时代，拼的是经济，估计扫遍全球的就不会是大英帝国了。冷兵器时代靠的还是拼命精神，光脚的不怕穿鞋的。所以往往发生低级文明打败高级文明的现象。
    后来到了明和清时期，中国又一次迎来了货币增长带来的繁荣，主要是世界各地的白银都在不断的走私到中国，中国这种银本位国家迎来了货币充盈。在世界经济GDP占比，达到了30%以上，还是很不容易的。
    所以你们不要信比特币这种挖掘总量有限制的鬼话。这个世界的发展，随着生产效率的提高，需要的是更多用来标记生产成果的“一般等价物”，而不是无法增长的货币。这是让历史的车轮倒转，所以根本不可能成功。
  • [维A 回复 龙猫] : 通货膨胀是纸币现象
  • [天地泉 回复 羿飞] : 谢谢羿飞老师解惑，有些明白了，是最开始本币升值导致老百姓不愿花钱而是存银行，这样造成通缩，所以有负利率的惩罚性措施刺激消费，但是老百姓观念也没那么容易扭转过来，既然银行负利率，那大不了钱放在家里，就像古代埋地下一样，通缩的趋势下，刺激消费也难以见效，所以负利率时代本币依旧还是很可能升值。另外机构资金流出换外币，可能导致外汇市场本币贬值，但外汇市场升贬值似乎和国内市场本币购买力升贬值没有直接关系？就好像前几年人民币外汇大幅升值导致出国旅游暴增，但国内物价依旧上涨，没有升值的迹象。
  • [龙猫 回复 羿飞] : 学习了！谢谢！

问：飞哥好，本人工科专业，目前在国企工作。职业生涯一眼看到头，不想这样荒废人生。本人学习能力强但对工科不感兴趣，想自学考个金融相关的硕士，目前没有背景和人脉资源。不知道飞哥对各类金融专业有何看法和推荐，哪些更适合今后时代的发展？

【羿飞】 回答：如果能考到金融类的硕士，还是有前途的。至于没有人脉这个不用担心，金融业的人脉主要来自于学院系，这与大多数行业是高度依赖家庭门阀关系是不同的。
就是说只要你去读了比较好的学校的金融系，出来就有人脉了，助你成功。

  • [三仟] : 跨考金融，很有难度，尤其是工作的，简单说说原因：
    1，先说初试，你本身工科背景，自学能力强并不能代表你能考的过金融名校培养出来的本科生（前提是你想考的是名校，考普通院校金融专业对你的未来再就业并不能起到太多作用），这里并不是说这些学校培养出来的人都厉害，最少对自己有想法，想考研深造的还是多少有点本事的，另外，初试的公共课部分就不用说了，你按目标院校的招生简章的要求科目复习就行了，大家都一样，但是专业课部分是学校自主命题，说白了，他们本校的本科生是有很大可能是可以拿到部分考题的，所以说这一关即使你自学能力强，也并不代表你有优势。
    2，再来说一说复试，常识问题，金融专业是很吃学历的，据我了解国内外金融名校极少会招跨专业的全日制研究生，最少不能垮的太大，你工科背景。。。你可以试想一下你在面对其他本专业竞争者的核心竞争力是什么，如果你是面试老师在极有限的时间内你会选择一个本专业的（很可能还是本校的）并且拿过来就能用的学生，还是会相信你有异于常人的能力，毕竟你面临的竞争者很可能拿着世界500强的实习经历，还有各种金融业相关证书，会计证那种比较low的就不说了，比如CPA，CFA，FRM，ACCA等等，这些证书无一不体现出他们的专业技能及自学能力，如果你能在初试中杀出重围，你就需要考虑怎样说服你的面试老师了。
  • [三仟] : 3，最后说说找工作，这是建立在你顺利考上并毕业的基础之上，这就分两种情况了，其一，就像飞哥说的，金融业人脉确实是你在学院就能积累的，这在你就业的时候有很大帮助，因为确实有些大的金融业公司（券商，基金，投行等等）会有某些学校的学生在有的公司会机会比较大的情况，因为你可能有很多学长学姐在那工作，形成了一个你们学校的某种优势，通过他们你可能优先拿到各种实习机会，从而毕业内推，避免和其他人竞争，要知道现在好多大的券商之类的公司，一年招的人很少，并且基本不校招，内推的都消化不完。
    其二，如果你没通过第一种方式找到工作，那恭喜你，你就业的竞争压力可就远比你考研难度大多了，因为你面临的竞争者的综合实力又有了长足的进步，他们来自海内外你知道的不知道的所有金融名校，远远不止于清北复交等院校，在这种情况下，你的第一学历是工科背景，如果你想去国内综合实力前20−30%公司的话，我不敢说百分之百，至少有百分之90的可能你的简历不会被递到面试官的手里，要知道这些公司如果想招5个人的话，HR可能会收到5000份以上的简历，这时候他们会把你的简历录入到系统里，通过设置某些条件进行初步筛选，本硕都是金融相关专业211院校是标配，好的公司会要求本硕985院校，本科非金融专业基本不存在通过筛选的可能性，所以说如果你面临这种情况，你就得试图去在马云上车之前拦住他了。
