2018-09-20问答记录

发表于 2018-09-20 更新于 2021-04-09 分类于 Stock
本文字数： 2.4k 阅读时长 ≈ 2 分钟

问：飞哥，有关购房的问题想咨询您。我在三四线城市，有一套学区房电梯房（周边小学，初中，高中都有），父母和岳父母各有一套老的住房，没电梯，都在6楼，现在我单位有团购房有优惠，也是学区房（小学，初中比我现在的要好），我想买给我父母养老住。赶巧目前岳父也想住房升级用我们的住房公积金买房，住到我第一套房附近，意味着我们买单位的福利房要全款购房。不知如何取舍?请飞哥支招。

【羿飞】 回答：这个单位的福利房是非常值得买的，位置好还有优惠。关键就是正好岳父母要用你们公积金。
那么是否可以跟父母谈，买了新房，住进去以后，老房子卖掉钱给你，这样就不永远占用自己很多钱了。

  • [陈婷婷] : 顶楼的老房子趁现在行情好赶紧都卖掉，淡季的房市里没电梯的顶楼很难卖的
  • [太阳] : 确实第三套房全款占用太多资金了

问：希望飞哥建议365单总，每天也想一起好易融一样每天发布追回的逾期款，每天标的正常还款率，以平息大家的担心。

【羿飞】 回答：每个平台面对的情况和挑战是不同的，并不能用同一个方式。

  • [警王] : 可是飞哥，我并不能理解有什么情况使得“每天公布数据”这种手段会损害平台的正常运营和恢复生机诶[微笑]
  • [中国狐 回复 警王] : 平台有综合考量，有利的事会去做，反之不如不做，为什么不做?你相信你选择的平台就好。

问：飞哥，基金池里把申万证券分级去掉了，说是分级基金风险大，咱们这个基金不是母基金么，风险也大吗？这个基金我现在还有很多，可以继续持有吗？

【羿飞】 回答：如果你是分散买了好多个，有一个带点点杠杆的也没事。但是怕有人冲动，全压分级的。

  • [凤来] : 飞哥，不是我问的[嘘]
  • [徐伟伟 回复 凤来] : 吴哥，不是你问的跳出来干啥[呲牙]就算你问的也没人会怪你啊！
  • [凤来] : 分级券商和etf对比

问：飞哥，能请你了解下目前一起好和易融的共同资产端-齐通具体是什么情况？负责人被打黑了吗?此资产端是未来彻底不能还款了吗?以及未来平台分别有什么真对齐通的计划。

【羿飞】 回答：请等待后续了解以后，会跟大家进行说明。

问：飞哥，我有个顾虑就是打黑除恶运动是2018年1月发起的，据说要持续3年，jc是不分青红皂白一起打，导致老赖逾期不还钱，飞哥月报说春节后平台会明显好转，需要撑住等黎明，但打黑运动还有2年呢，如果jc继续添乱呢，平台还是没好日子啊

【羿飞】 回答：这确实是一个很大的压力，资产端什么时候能回暖，主要就是取决于这个运动式的瞎搞持续多久。虽然喊三年，肯定无法持续那么久的，警力也跟不上。各地都是样板式的搞一波，对上面有个交代，就歇了。大家本着充分考虑风险的思维，熬吧。

  • [凤来] : 希望咱们的平台能够抗过这波运动性执法
  • [Kelvin Fis] : 关键是要有在打击范围外的新业务运转*
  • [hhl] : 反腐的持续时间可是不短的，希望这次短些再短些！
  • [三儿] : 抱最大希望，做最坏打算，大大和川普都是执行力很强的领导人。
  • [三儿] : 比起经济环境恶化、银根紧缩什么的，圈友更多是栽在这上面[晕]前者戳破了虚假的泡沫，后者冻结了真实的资产
  • [三儿 回复 薇薇] : 目前知道的只有深圳属于

问：飞哥，求问下一起好祁胖子车金融啥情况，逾期超过80%了，群里都是这情况。其他平台再差也不会这样啊。求告诉真相，不要一直打鸡血了。

【羿飞】 回答：根据官方公布数据，没有那么严重。可能是你持有的待收部分严重。其实我情况也相似，因为老投资人，一般都是投车的多，因为收益高，这部分是逾期重灾区。
我没有刻意打鸡血，是这种时候，除了相信平台，给祁总努力的时间，请问你还能做什么，你说一个方案我听听？如果可行，听你的。

  • [卡卡] : 飞哥，一起好智能投标逾期80%多，9月11日切底层，切完后60%左右都是9月11号之前的逾期标，20%多被切到逾期严重的车金融了，是不是平台故意的？
  • [阿力 回复 卡卡] : 这种可能性非常大。
  • [卡卡] : 切底前客服说智能投标都是优质的车融宝，现金贷等，都不存在逾期情况，但出来后4-8月份的全部逾期，现在问客服也不回答。
  • [卡卡] : 胖总原来说智能投标可以分10期兑付，还真不如不切底，按胖总的方案来。
  • [金 回复 卡卡] : app端还看不出来拆分吧？

问：飞哥 圈里活动标大概率应该是全部展期，这类标我咨询后是南京资产端的，但是没有资产端联系方式，能不能你联系一下单总或资产端负责人这些标的展期是进入起诉阶段还是处理资产阶段展期。就是是怎样的一个后续进展，谢谢！

【羿飞】 回答：好的， 我最近会跟单总联系，了解一下这批标的详细情况，回头跟大家讲。

  • [可爱多] : 谢谢，全部展期实在不正常
  • [汉水涛涛] : 是啊，按常理，也应类似易融恒信，一起好的逾期率，不能全展期！！？
  • [WANG] : 有一个不太好的现象是群里客服早早就不说话了，最近一次群里说话是9月1号。客服按理说不会有其他任务啊
  • [星河 回复 WANG] : 3群周四还有客服在回答问题
  • [WANG 回复 星河] : 8群只有一个客服1号说过话，其他两个客服都是8月就不说话了
  • [星河 回复 WANG] : 比起其他平台是不怎么积极，不过现在也没什么办法，只有等
  • [姚磊] : 这次活动标和群友利益挂钩最明显，不像别的平台，希望飞总能想办法帮群友核实更多信息，共渡难关。

问：飞哥你好，持有申万分级券商一年了，亏损严重，难道现在割肉吗，😭

【羿飞】 回答：最近应该是分批建仓基金啊，割肉就被收割了。

  • [凤来] : 分级的换成ETF
  • [xlzhang] : 我也是浮亏37%[尴尬]
  • [hhl] : 亏36%
  • [xlzhang 回复 hhl] : 看来建仓的时间差不多
  • [御翼 回复 凤来] : 分极换成ETF后，如果大盘依然跌，那当然换了好些。。。如果是开始上升，那ETF的涨幅一般也会弱于分极的。。。所以换不换是两难，取决于后市走向了。[抠鼻]
  • [凤来 回复 御翼] : 我觉得就是底部区域了，500亿落地和嚷嚷的2000亿都是现在点位附近，就是不知道要低迷多久

问：飞哥你好，能否说一下北京公积金新政？感觉刚需也被打死了。

【羿飞】 回答：这个政策出来，很多年轻的刚需购买力不能释放，对北京房价应该也是一波利空。观望一下也好。

  • [青山不染尘] : 这个相当于废掉了公积金
