2019-07-31问答记录

发表于 2019-07-31 更新于 2021-04-11 分类于 Stock

问：飞哥，美元最近连创新高，黄金也一直阴跌不断，后市您怎么看?特朗普一直说美元太高，也没见他往下降啊

【羿飞】 回答：特朗普是在跟美联储博弈。特朗普希望美元低一些，这样对制造业有好处。但是美联储不希望美元马上跌下去，因为会吧美股拖垮，以及之前08-12年之间量化宽松购入的美债很多很多，想在高位慢慢减持。
最终两者会利益一致，美联储减持完了就可以让美元安心下跌了，低位的时候再搞一轮新的量化宽松哈。

问：请教飞哥，华宝油气是受原油和美股影响的，目前原油跌幅不大，美股还创了新高，为什么华宝油气却超跌到了新低呢？

【羿飞】 回答：华宝油气配置的是石油上游资源的企业股票，而非原油。美股虽然不错，但是油价仍然处于低位，原油产业链利润都不好。油价80美元以上油企才比较好过。60-80之间只是盈亏平衡点。60以下基本上都是普遍亏损的了。
反之，如果以后油价又到100以上了，这种基金妥妥的拿着，不管其他市场怎么折腾和动荡。所以要在不同的时机选择不同的品种，不能看着一个东西跌多了就买，涨多了就卖。

  • [幻想] : 意思就是说，华宝油气这种要等石油在80以上了才有持有价值，如果判断是短期反弹则配置其他基金更好。
  • [羿飞 回复 幻想] : 跟踪油价的比如南方原油，是跟价格同步的。

问：飞哥，市场上能做波段的资产，除了黄金、石油、股票，还有别的资产吗？最近个人针对这三类资产都持有比较少，在逐渐建仓，但是手头现金还是多，没了p2p，不知道现金应该怎么用。飞哥觉得现在市场上还有低估的产品吗？

【羿飞】 回答：我在基金池里已经建议大家了，替代p2p配置可以暂时选择债券型基金，请详细参考一下，平均7-9%的年化，又是随时可以提取的高流动性，很不错的资产。
另外就是如果有长期资金和支付能力比较强，可以配置商业地产，一般年收益5-8%不等，各地情况不同。缺点是一次性投入大，优点是长期现金流非常好。

  • [大吴] : 泰康那个基金长期业绩好牛
  • [Kelvin Fis] : 竟然有这么好的债券型基金，早知道还啥P2P啊*

问：飞哥，如果有400w左右的资金，想考虑配置房子一类（含北京和外地）收租子用，1)哪个城市你觉得可以尝试? 我看到长沙房价较低，不过外地人需要24个月凭税条儿才行；2) 有个中介介绍了一个商住两用的大开间，我有点担心以后变现不如普通住宅，毕竟住着不会太舒服，你觉得可以考虑吗？

【羿飞】 回答：如果是打算做现金流，可以优先考虑商业地产，买商业的回报率好一些。缺点就是房价方面没有什么起伏，所以如果有很强变现需求的，不能做长期投资，就不能选这个方向了。

  • [三儿] : 推荐个房事方面的专家——微博@通货朋仗

问：飞哥，1) 过去你的风格是比较积极的，经历过这次2018黑天鹅后，会不会回头去配置一些国债或者万能险一类?
2) 万能险和国债，哪个更好一些? 还是吧民生那种作为最保守的一种，依然完全不考虑国债和万能险这一类产品?

【羿飞】 回答：我投资风格一直是比较积极的，现在也不会就消极了。我是通过更多门类的资产来缓解风险，而不是整个投资风格变的过于保守。
国债，保险这些只能跟通胀斗一斗的东西我肯定不买的。
怎么说呢，其实风格积极和消极是每个人标准不同的。比如在玩杠杠投资，炒币的人眼里，我应该是特别特别保守的老古董。[憨笑]还是践行自己的风格。

  • [大吴] : 来这里学习的人是不甘心买国债的
  • [苹果树 回复 大吴] : 我记得统计过黑天鹅后大家的自救行为，不少都开始回去配置银行理财了（这个和国债本质是一样的），因为这次大家产生损失，其中自己的一个因素，就是资产配置失当。看飞哥最近的回答，我理解他的安全资产部分是房产和商业产，从中获得较安全现金流

问：飞哥，下半年配置什么资产好一点？美元筑顶了现在，是不是石油等大宗商品有机会?基金的话可以配置大宗类的基金吗？

【羿飞】 回答：下半年黄金应该还是可以保持强势，值得配置。原油可以等等企稳。

问：飞哥，你好，请问那些借网贷两三千块不还的人会最终会怎样（不上征信的平台）？在这个总是讨论还的平台，我想咨询下不还会怎样谢谢

【羿飞】 回答：我觉得吧，早晚国家会对这些失信人再打击一波，他们会为这些失信损失一生的信用成本。但是这与短期内的回款率关系不大，大多数平台熬死了，都成了烂账。
总结就是出借人失去金钱，老赖失去信用，双输。

问：咨询一下，精工刚构和鸿路钢构怎么样？可以当打新仓底吗？

【羿飞】 回答：打新还是银行保险之类的做舱底更好更稳。你说的这种适合做中短线，不适合做仓底。

问：飞哥，一个有关技术的困惑，GX和FT似乎都遇到了先锋支付问题，导致到账和提现有困难。我查了一下，先锋似乎是被自己旗下的一个网贷牵累，所以现在工作不正常。这样的话，有点担心他的问题解决需要一个较长的时间。
我的问题是，这种情况下为什么平台不能换一家支付方呢？这在技术上到底有什么困难呢？

【羿飞】 回答：正常来说，肯定都要换支付公司的。运营正常的平台比较容易换，运营困难的平台不好找，三方也怕惹麻烦。另外接了新的可以恢复新的还款给大家到账。但是被先锋扣住的部分，每个平台金额不等，影响也不同。暂时只能放着了，目前跟他们有业务关系的各方，都在观望和想办法。
支付业务监管是比较规范的，也有拍照。估计后面会有其他方来接盘，然后逐步恢复业务，都是需要时间的吧，只能等等看了。

  • [苹果树] : 谢谢，飞哥

问：飞哥看好本次大盘的反弹吗？后市科创板上来是不是会抽大盘的水，对上证属于利空吧？

【羿飞】 回答：目前看抽水不至于，对于大盘来说，科创板目前还太小太小了，至少等有100多只以后，才会对大盘真正有关联性和资金影响。

问：飞哥好，科创的新股，由于上市的前几天不设板了，您觉得什么时机抛合适呢

【羿飞】 回答：其实是风险更大了，可以放到第一天的收盘前吧。因为第一天拉升价格的筹码，至少要第二天才能卖。所以看第一天收盘之前是不是一个上影线，如果收盘前已经开始走弱，必卖。如果第一天收盘的时候，还是大平顶，可以第二天再看看。

问：飞哥，可转债怎么进行投资，看那些指标筛选可以买入的可转债?

【羿飞】 回答：可转债的投资，其实就是如果你无法判断股票是否上涨，则买入可转债，如果下跌了你至少保本有利息，上涨了则行权成股票赚上涨的收益。
这个故事听起来是美好的，稳赚不赔。但是在实践上，上市公司的财务总监你以为是傻子吗，他们是预判股价大多数可能是下行的，所以才选择发可转债，这样其实就是一笔低息债权融资。
真正做可转债赚到的情况，就是大盘意外反转，来了个突发牛，所有股票都会涨，那确实是可以赚到。
还有就是要看个股，个股价格比较泡沫的时候，发行了可转债，你也是很难赚到的，因为后面可能阴跌好几年，其实就是买了个低息债。
综上，可转债并不是买了就能多赚的东西，还是很需要对个股后续走势以及价值判断的能力。

问：飞哥：是什么原因造成中国对上市公司财务造假处罚轻得跟开玩笑似的？这不是鼓励诈骗吗？

【羿飞】 回答：都是依据法律啊，很多法律都是之前很多年制定的，尤其对于金额方面现在看实在是没有惩罚性和威慑力。.
经济类的案件，例如非吸诈骗之类的，现在看罚金也是小的可笑，只有刑期方面还有点威慑力。只能等法条修改了。

  • [东方小石] : 很多j法律法规都没有什么可行性，尤其是经济方面的，说实话，要是依照法律，99.9%的企业都违法甚至犯罪了，依法治国，呵呵！

*问：飞哥， 因为不擅长股票，所以只有少资金参与，重在锻炼能力，和保持一部分底仓；看了你的文章，特别同意每天坚持打新（科创板）这一点，但有个问题，账户里需要保证50w的资金。那些暂时不想买成股票的资金，有什么比较安全又有一定收益的好办法?

    问：飞哥，一个公司注册的资金，在走完流程以后，可以提走吗？

    【羿飞】 回答：注册资金不能提走，那属于抽逃注册资金。 你的注册资本，应该是只能用于自己的企业经营，财务人员会进行相关的记账。
    在早年因为注册公司都是需要实缴的，经常发生先注册再把资金挪走的事情。还衍生了工商的垫资业务。但是最终你必须把企业的固定资产和流动资金跟实缴注册资本对上，否则是违法的。
    当然，私营企业，尤其是小规模的没太有人管，一般还好比较宽松，否则其实很多企业严格意义上都违法过。这是中国十年前几乎99%以上私营企业的原罪之一。
    而现在，工商注册已经非常宽松了，写一个你目标的认缴额和计划日期就行，实缴可以很少，所以不必再做这种先注册再抽走的事情了，毕竟是违法而且有成本的。

    • [牛牛 回复 大吴] : 广信增资这对平台是正面消息阿
