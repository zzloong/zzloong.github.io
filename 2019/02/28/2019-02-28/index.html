问：飞哥，俄罗斯开始全面鼓励生育了，而我国奖励二胎都没开始，更别提全面放开了。
人是第一生产力。那飞哥现在关注人口问题吗，对相关经济与发展有什么相关布局？

【羿飞】 回答：我觉得不应该单纯的说人口多或者人口少就一定好吧，人口结构也很重要。人口结构包括劳动力人口占比，以及人口教育素质(劳动力转化率)。
就中国来说，以前所谓人口红利主要是吃人口数量优势，适合劳动密集型。现在向高端产业转型，主要拼的是高等教育人口数量，目前这个数量是在逐年增加的，所以中国未来应该不会缺劳动力。媒体普遍担心的人口红利结束是劳动密集型产业的人口红利结束，所以中国必须技术升级，没有其他路线可选的。
如果多生，比如全面放开。比较大的问题可能是除了养老以外，很长时间又会有比较大的人口抚养教育压力。其实现在放开也没意义了，养孩子成本太高了，大家都不爱生了。
越发达的国家，人口越衰减这是普遍现象，也是困扰人类社会发展的难题，目前好像还没什么好办法。

  • [羿飞] : 如果说考虑人口结构方面的布局，就多投资布局医疗健康产业吧。人们越来越老，也越来越有钱，肯定是要往这个方向花钱的。
  • [向日葵] : 慢慢的步入老龄化社会

问：飞哥，基金池是否考虑加入偏工程机械类的基金？

【羿飞】 回答：基金池有偏向于高端制造业的呀，未来应该是技术升级的趋势，低端制造业，越少利用劳动密集型的产业，越有前途。
你如果发泄比较好的，也可以提出来，我们一起分析，真的出众则加入基金池分享给大家。

问：飞哥你好，如果主板也有50万门槛，是否市场会大不一样？

【羿飞】 回答：如果主板限制50w，中国股市大概会跌到1000点以下。中国股市六成以上是散户，散户平均投资额只有3-5万。

问：飞哥，你发的债券基金我看每年的收益率都是浮动的，最近一年招商双债是9%，二年13%，三年19%，三年平均6-9%之间。最近的收益率最高。
我的问题是哪些因素影响债券基金的收益率呢

【羿飞】 回答：多数是投资一些企业债，收益率4-6%之间。会因为社会流动性波动，有不同的折价率。去年特别缺钱，折价率就高一些，持有到期收益率就提高了。
偶尔也有上市公司违约，不过还是比较少出现的。还有国债逆回购等，每次月底，季度末，年底，银行回笼资金的时候，年化能达到20-40%。不过只有几天，普通人操作是很操心的，也赚不到多少，这些专业机构长期配置债券，有流动资金，可以做的比较好。
大多数普通投友，之间买基金池里的就行了，比较省心，比自己直接入场买卖省心收益中上。

公司整体氛围就是奉承巴结领导，目前干的活基本都是形式主义，假大空的东西，每天工作看不到一丁点进步，各种小事多如牛毛，活的比较压抑，想辞职了。暂时找不到出路。请飞哥和圈友给点建议🙏

尤其是最近金融动荡期，做创业之类的失败率是很高的，并不是你适合做变化的时候，最近应该保守稳妥为上。等经济形式缓过来了，到处都需要人才和机会的时候，再考虑比较明智，所以最近忍为上吧。

问：飞哥，您前两天发的文章关于中国货币的，最后结论180万亿债务，但是人民币印的速率小于目前利息的增长，意味着实际目前情况属于通缩，那么现在是不是应该买债券啊？商品类资源少碰?

【羿飞】 回答：这几个点的逻辑理解有点问题。不论如何，长期看都是要通胀的。债券的配置属于资产中不可或缺的部分，但是安全的债永远只是比通胀高一点，不能全买债券。
商品类的价格，主要是美元涨跌影响的，美联储政策影响大，中国的货币情况对其影响很小。倒是中国的经济增长率，因为直接影响资源消耗，对大宗商品的价格影响更大一些。
