2019-01-20问答记录

发表于 2019-01-20 更新于 2021-04-11 分类于 Stock
本文字数： 2.8k 阅读时长 ≈ 3 分钟

问：飞哥好，请问股票池里的物产中大，嘉事堂等股票，最近5年的净利润都为正，但其中4年的经营现金流净额都为负，这个情况正常吗?

【羿飞】 回答：这些商贸类的企业，收到的钱都投入到新的产品采购库存周转里中去了，能留在手里的现金很少，也导致利润虽然高，现金流却不高。这部分在帐目里主要表现为存货。这属于正常现象。

  • [龙猫] : 营收特别多，规模特别大，利润感觉也一般，搞不太清楚到底好不好。

问：羿飞老师，今年网贷形势不好，你还坚持写2018年度的理财报告吗？

【羿飞】 回答：我很想写，因为这是多年以来的习惯。讲真的，因为我资产布局种类比较丰富，虽然网贷部分受行业影响跟大家一样损失惨重，但是整体属于可承受范围。感觉在大家都很伤痛的时候聊这个，有点伤口撒盐，所以一直没有动笔。
要坚持做自己，真的是挺难的，最近尤其感触很深。

  • [顺] : 没事的，我想学习飞哥的总结
  • [大吴] : 两篇，一篇总结，一篇展望

  • [Kelvin Fis] : 期待*

  • [羿飞 回复 木屐] : 没有赚啊，也是亏的一塌糊涂，不过还算是有能力调节好资产结构，站稳脚步。活下来不错了，坚信未来会更好。
  • [木屐 回复 羿飞] : 开玩笑的，不过说实话，每年看飞哥的分享还是有些羡慕的，房产一项就收入可观

问：飞哥你好，最近几个月大盘不好，网贷不不好，分享东西少了很多。
我个人认为，新年新气象，不管亏损也好，赚也罢，还是多聊聊投资，多分享一些，也算对得起大家交的圈费了。要是没人问问题，您可以分析一下之前以前交易操作过比较好的经历，股票也好，股权也好，失败的也行。其实输赢正常，多学习了解一些，不管什么时候，知识都是财富。一所普通大学一年学费4000左右，您这里3000，但是目的都一样，学到一些普适的方式方法，希望能物超所值，2019，看到飞哥分享不一样，更精彩的东西！谢谢🙏

【羿飞】 回答：非常认同这样的建议。希望大家都能快速的振作起来，一起前行。其实网贷只是大家配置的资产之一，现状也基本稳定了，后面是时间去慢慢消化。在此期间，应该集中经历创造新的价值和财富，以后才能更好。

  • [青云瓶] : 支持这个好建议，以前好像在个股方面分享过TCL和潍柴动力，挺好的！

问：飞哥，今年由于打黑除恶而引起的网贷大地震让投资人损失不小，而飞哥一直推举的中央水池投资法里，网贷作为稳定的债权收益和流动性一直充当着中间的大水池。
这次地震让许多投资人开始质疑网贷将来是否继续可投，也让自己的理财规划出现混乱
飞哥能否说说网贷的未来，以及中央水池法还会以网贷作为大水池吗，是否会有些调整？

【羿飞】 回答：中央水池法，我还是继续很坚信的。目前来说我们所遭遇的，是作为水池用的网贷这个产品，行业动荡了，但是这个投资方法模型的有效性没有问题。
坚持他是流动性配置而不是资产的大比重，则不会出现致命威胁。未来是不是继续可以当作水池，是需要观望的。主要是看行业稳定了，恢复了以后，会不会有牌照之类可以更规范更安全，以及相对的流动性是否还具备性价比。
这需要时间来验证。

  • [罗晓莹] : 那现阶段飞哥用什么做水池了呢？可以分享一下吗？
  • [冷了] : 那飞哥现在选择什么当水池？
  • [无欲则刚] : 以后只要考虑是否需要换池子就行，模式没有问题，认可的。
    作为池子，流动性，安全性是第一的。
  • [凤来] : 估计以后做中央水池比较差了，收益10%以下，而且一线平台都是计划类的一次性到期还本标，流动性也差
  • [罗晓莹 回复 凤来] : 大吴哥现在用什么做水池的？分享下？
  • [凤来 回复 罗晓莹] : 还是等飞哥披露更有权威性
  • [colour] : 是不是该仔细想想，因为打黑除恶而引起的吗

【羿飞】 回答：长期看医药板块是有持续性的，有好的基金发现，会加入后面基金池。

问：宜贷网的清盘方案，飞哥怎么看？

【羿飞】 回答：深深觉得，按减去收益的算法不太和谐。因为不管哪个平台，只要没有jz的，基本上还是按照待收算的。尤其是老平台，如果扣收益，绝大多数人的基数相信都没多少了。
一般来讲，没立案的时候属于民事关系，所以大家就还按照账面名义待收去算的。只有立案了，界线上属于非吸，出借人也变成了非吸参与者，才会剥夺了收益权，变成充值减去提现，差额为有效待收。

  • [逍遥红尘] : 强软，估计后期破产清算了
