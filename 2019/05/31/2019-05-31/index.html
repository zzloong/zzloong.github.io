2019-05-31问答记录

发表于 2019-05-31 更新于 2021-04-11 分类于 Stock
本文字数： 1.9k 阅读时长 ≈ 2 分钟

问：飞哥上次的基金更新，过了那么久了，好多上市公司的业绩变脸了，基金需不需要重新调整下?

【羿飞】 回答：咦，4月份在基金更新了季报以后，刚刚给大家更新了一版基金池呀，如果没有收到，请联系助理的微信告诉补发一下。

  • [幻想] : 你是4月21日更新的，好多公司业绩是4月30发的业绩报告，所以问问是不是需要再更新下，后面想到基金公司也不会重新发持仓，估计也更新不了。
  • [羿飞 回复 幻想] : 是滴，基金公司是每个季度更新一次数据。没必要担心太多，选基金就是选的他们的投资能力，好的基金有能力应付各种市场变化。下个季度看答卷。

问：请教飞哥，您买的民生那个是通过京东金融买的，还是通过网上银行就可以? 现在似乎银行理财也去刚兑了，是不是还有些比例留给最安全的国债、定期，如果需要有些资金放在银行的话? 最近YRHX的网站没法上去了，由此对京东金融这些网络渠道的安全也感觉有些不安全，这样形势下飞哥怎么应对?

【羿飞】 回答：我是通过银行的网银买的。银行也是不允许承诺刚兑的，但是银行自己的理财产品在实践中还都是刚兑的。但是对于银行代销的基金业绩的亏赚、其他三方提供的债权类产品，是不负责任和刚兑的，一定要看仔细。

  • [🐚] : 银行理财本来也应该是没有刚对的，之前是宣传都说有，现在是不让说了，其实还是有的。银行理财如果没了，再被宣传一下，乌合之众会吓出病来的。
  • [阿狼] : 飞哥，券商出的理财产品，投资方向是债券，债券基金，大额存单，银行理财等等，这类的风险如何?

问：飞哥能再帮忙挑几个债基吗？目前的确需要一个大水池，基金池里的都投了，但感觉不够分散，希望再加几个。

【羿飞】 回答：债权型的，都相似的，我选了2、3个收益长期看比较好的，已经够了。债权基金是合法的产品，不会出现p2p这种情况的，你太敏感了。

问：飞哥，目前大盘已经跌破了支撑位，是不是意味着大盘已经风险增加了，大盘趋势是不是该朝下了，又到熊市了？

【羿飞】 回答：并没有跌破支撑位，如果收在3000以下了，才会有明显的偏空弱势风险。月线看，是收在了3000点以上的，均线之上的上影线。

  • [🐚] : 劣者以为，这波3288以来的下跌随时结束，现在是好机会。

问：飞哥，请教一下，科创板的基金能投吗？

【羿飞】 回答：科创板应该是有大发展的机会的。但是主动型基金我不太放心，怕以后有利益输送接盘垃圾股。还是等第二波，有指数型基金的时候再上了。

  • [御翼] : 现在各个银行、理财公司的专属理财经理，都在推荐科创板基金，意思就是必须科创板必须做出来，是Z治任务，[呲牙]
  • [🐚] : 坐等etf，费用还低

问：飞哥你好！想判断一县城的大概常住人口，需要从那几个指标下手，还请多指教？

【羿飞】 回答：好像没有特别权威的数据。人口流动性很大，官方数据也不准。大城市一般总是比官方公布的低一些，小城镇会比公布的还少。县城的，一般是人口流出，还是别作为置业目标吧。
多关注全国性的品牌房产中介，这些机构是最敏感的，哪里房地产有机会，他们闻着味儿就来了。而且中介要赚钱，就得推高房价，鼓动提高交易。一般一个地方开始有大量的品牌中介开业，其当地房价的报价就会由当地人直接交流，变成慢慢的话语权被中介占领，大家都开始搞不清房价是多少，只能问中介公司。
大量连锁中介在开业的城市，这里就有机会。很多在关店的，就要谨慎小心了。

  • [幻想] : 关键是你看中介多起来了，房价基本都涨上来了，你能确定你不是接盘侠?
  • [羿飞 回复 幻想] : 房子不是股票，趋势性和周期都很长，不要短线思维。
  • [🐚] : 县里每年都有统计年鉴的可以参考参考，，，不是很准确，但每年减少是大趋势
  • [幻想 回复 羿飞] : 意思说一个地方的房子炒起来了都会有很长的上升周期，短期回调都会涨起来，例如飞哥你前面说的1线城市回调了20%，和现在二线城市回调都是暂时的？
  • [幻想 回复 羿飞] : 就算要上涨也会是个漫长的过程，等5年再看也合适

问：飞哥可否发掘些低估值、有定价权的蓝筹股，我觉得从长期持有看，这类股票才有投资价值的。

【羿飞】 回答：这个很难讲，一般具备这种公认好特征的股票，那么就很少会出现低估值。比如茅台，格力，大家都知道好，所以他长期处于高估状态，你想低估值买入是很难的。要等一些市场稀里哗啦什么都乱跌的时候才有机会。

  • [羿飞] : 我最喜欢的方法，是找好企业被错判的机会。比如圈子里天空很喜欢的新华医疗，这是个好企业，因为几个对赌没做好，以及现金流管理等问题被误解，现在严重低估中。
    但是也不一定他低估就能马上起来，需要经营数据重新回到良性增长阶段。所以我一直在盯着，什么时候他状态恢复了，数据稳定了就可以介入。
  • [苹果树 回复 羿飞] : 飞哥，新华医疗给你好企业的理由是不是可以说说?
---------- 本文结束----------
