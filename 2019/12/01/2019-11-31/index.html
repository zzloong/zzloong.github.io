2019-11-31问答记录

发表于 2019-12-01 更新于 2021-04-11 分类于 Stock
本文字数： 2.1k 阅读时长 ≈ 2 分钟

问：飞哥，身边又碰到了短线高手，如果真有活的久的短线高手，他们操作的逻辑有哪些呢？

【羿飞】 回答：短线的高手，我认为只在故事中存在，而不在现实中存在。道理很简单，如果短线高手真的方法管用，哪怕本金只有几万块，都是几年的时间就能变成若干亿了。
现实中痴迷短线的人肯定是比耐心做长线的人要多的多，这是把股票当彩票玩的心态。我炒股接近20年，身边认识的熟人，痴迷短线，痴迷杠杆玩期货的人太多了。反正每次股市来行情的时候，就会有一些人叫大家吃饭，然后亢奋的讲他们最近的神操作，一个月翻倍了如何如何。当然我相信他们说的是真实的经历，不是瞎编。但是呢，长期看，十几年下来，这些人手里并没有多少资金，能有1、2百万的都算多了。当然，其实更多人是已经消失了的。
做投资不急不闹的我，相对算是真实实力比较雄厚的，虽然他们经常笑话我胆小，说如果我的资金给他们，他们早成大富豪了。我不禁要问，如果他们的方法真的一直灵，为啥活一样的岁数，我资金比他们多呢？反正我是不信的，投资之路漫长，还是小火慢炖来的踏实。

做短线能成大器的肯定也不会完全没有，但是概率很低，大概只有几万分之一，他们的方法和逻辑都不能复制。人们总是听说华丽丽的暴富故事，其实只是幸存者偏差的错觉。

  • [大吴] : 加精
  • [三儿] : 我完全赞成投资应该做长线，但短线高手也不一定是把把胜，或许十把胜个五把半？
  • [今天Q] : 强烈支持上述观点！

问：飞哥好，能简单分析下江中药业吗，自己研究对比了一下，业绩不错，财报这两年借款都是零，这是问题还是优点，您怎么看

【羿飞】 回答：看了一下财务数据，估值也不算高，问题不大。我们的gpc没有入选这个，不是说他不好而是其他的比他好。比如：
仁和药业市值75亿，营收35亿，利润4.4亿。
江中药业市值65亿，营收17亿，利润3.8亿。
从表面上看起来，就是仁和的数据更好了。更重要的是成长性潜力，仁和2013年的营收是18亿，现在是年50亿左右，企业是一直保持良性发展的。江中2013年营收是27亿，而现在是20亿左右，企业实际上是有所衰退的。从更长远的周期看，是仁和药业更有前景。

  • [木屐] : 仁和营2016财报解释，营收大幅减少是卖出了九州通，但是利润上升一点，这算是优化资产结构吧

问：飞哥，你卖出股票的时候可不可以也在圈子里面分享一下心得，分析一下卖出它的理由和不看好它后市的逻辑，让我们增长一些卖股票的经验和知识！

【羿飞】 回答：操作就不分享了，会有引导性，会干扰大家的独立判断，而且法律风险很大。
隔很久的时间，拿一些以前的操作出来研究分享一下当时的过程是可以的，我感觉有适合的会发帖子。

  • [大吴] : 有案例挺好的，比如智度股份，我就3个板出了
  • [等风来] : 可以卖出之后来回顾一下股票的买卖心得，我们也从中吸取经验！

问：飞哥，能分析一下中国石油吗，近期咋频繁暴跌，是不是买入时机？

【羿飞】 回答：中石油不要碰，油价80美元以下中石化更好，油价80美元以上才是中石油更好。

  • [王睿晗] : 中石化和中石油还有这个区别？为什么
  • [春江水 回复 王睿晗] : 字面理解，可能一个采油一个炼油

问：飞哥，今年黄金暴涨，而美元却变化不大，主要还是因为冒亿站的原因吧。在战争结束时黄金应该有波回调吧，那时在配置黄金如何，目前看美元两次降息黄金也只是窄幅震荡。

【羿飞】 回答：美元在高位的时候，黄金提前发生了上涨，实际上是一种预期的表达。正因如此我预判美元一定会下来，因为黄金已经提前给出答案了。
黄金应该是高位震荡小幅阴跌横盘。然后每次美元偏弱的时候，忽然大涨一波。所以配置黄金资产应该是买入了安心放着等，不要带有短线思维。

问：飞哥，图片是我的股票账户截屏，应该算是很不适合炒股的了，看到您发的基金池里信达和万家都今年涨幅都很亮眼，不知道我现在适不适合清仓股票分散投到这两只基金里面。谢谢

【羿飞】 回答：不能光看持仓股目前的涨跌。因为这个基本上是3、4月份高点的时候大家交易，然后下跌被套的结果。之前上涨过程中，盈利的卖掉了，现在那些收益看不出来。应该是计算一下比如年初时候资金价值，看看现在大盘涨了多少，自己盈亏情况。如果还是输给了大盘，那么是肯定不适合直接自己炒股的了，还是踏踏实实的买基金池比较好。

  • [kevin] : 年初时候的资金价值怎么算？
  • [羿飞 回复 kevin] : 没法算，自己记账呀
  • [无敌草 回复 kevin] : 同花顺可以看今年以来收益率
  • [kevin] : ok，我看看去，谢啦

问：请问飞哥了解东南亚P2P市场吗？特别是最近很火的印尼？本人已经在新生活工作了一年了，想多了解东南亚是不是还值得投资点在P2P？如何投资……

【羿飞】 回答：东南亚的情况很像中国的前几年，目前如火如荼，最早是中国的现金贷企业先过去做的，掏到了第一桶金。我个人不建议去投资和碰，因为做这些互金企业的多是中国公司，到印尼本国人欠债到一定规模，引发社会矛盾的时候，一定会把这些企业都打死毫不留情的，东南亚有反华传统。

  • [幻想] : 适合去借钱不还[呲牙][呲牙][呲牙]
  • [不进则退 回复 幻想] : 有道理哈哈
  • [不进则退] : 确实是，出了事在中国还有个群，报个警，在东南亚就不容易了……
