2019-06-30问答记录

发表于 2019-06-30 更新于 2021-04-11 分类于 Stock
本文字数： 1.6k 阅读时长 ≈ 1 分钟

问：飞哥，这个圈子您还准备做下去吗？还是非常看重您对各类资产配置的判断的，尤其是股票池和基金池的分享方面。希望能一直做下去。

【羿飞】 回答：会的，因为不管是有没有圈子，写不写公号。我自己的投资都会继续，投资是我的爱好，也是我谋生之道。写点东西是已经自己做功课想明白的事情，顺手分享一下，做媒体本身不是我的生存基础和目的，一直如此。

  • [华子] : 真心希望圈子一直做下去[微笑]
  • [大吴] : 圈子的交流方式简单直接，希望飞哥继续[拳头]
  • [顺～～] : 没了p2p，圈子内容越来越少了
  • [进] : 就喜欢飞哥不瞎逼逼的风格[微笑]
  • [御翼 回复 顺～～] : 我觉得人家想表达的，会不会就是你个意思。[愉快]
  • [王睿晗] : 飞哥最近股票池要更新吗？快出半年报了吧最近
  • [天空] : 飞哥可能生活上多少受一些网贷影响，所以圈里分享少了。愿大家都能挺过难关，再迎辉煌。

问：飞哥，美元下跌，除了利好美元计价产品，利好A股吗？

【羿飞】 回答：美元最近有所下跌，如果形成趋势性下跌，则利好所有资产。因为美元是世界货币，他下跌，就意味着宽松。各国货币供应都会增加， 通胀也会发生，资产价格提高。

问：飞哥，我对期货没什么了解，想请问下，石油交易的话，基金和石油期货有区别吗？做哪个更好一点？

【羿飞】 回答：我不建议，应该说强烈反对你参与期货交易。就买买关联基金就行了，不要给资金上杠杠。

问：各位家人，请问丁丁app停止运行了，现在应该去哪里提现？

【羿飞】 回答：pc端一直可以提现的啊。应该是在不断收缩，不愿意承担app端的开发成本了。

问：飞哥您好，就想问问您科创板建议打新吗，如果可以打新，运气好中签以后该什么策略，前五个交易日涨跌无限制，什么时机卖出呢？

【羿飞】 回答：打新完全建议参与，估计新股开盘上浮比主板的还夸张。但是需要运气和毅力，每天申请。

  • [天空] : 科创板打新，好吧，你们都是有钱人[委屈]
  • [大吴] : 如果能中科创板新股那就嗨了

问：飞哥，这两天原油和黄金都暴涨，应该入手吗？可否分享一下你的思路？

【羿飞】 回答：基本符合我年初的判断，美元开始乏力，黄金走入强势，原油经过一波快速回调以后有所反弹，属于确认底部的过程。

  • [大吴] : 美伊打不起来，原油还会跌回去，等下月初开完会，原油再建仓
  • [罗晓莹] : 那黄金目前入手可以吗？好怕又买到山顶上

问：飞哥，油这波算跌稳触底了吗？可以出手建仓了吗？

【羿飞】 回答：对这波油价的暴跌，我能准确的预测到还是挺得意的。因为上半年大多数时候油价看起来很强势且稳健，整个市场普遍的看多观点和情绪。目前回调出现了，验证了判断。后面就是观察政治博弈以及技术底部的出现，就能建仓了，个人认为还是要等等，都还不清晰。

  • [罗晓莹] : [抱拳]

问：关于紫金矿业，想听听飞哥的的建议。

【羿飞】 回答：我认为是黄金板块，黄金概念股里目前基本面最好的，经营稳定，大量并购资源，储量丰富。他只是盘子大，看起来不那么活跃，是其缺点，所以比较适合做长线布局，不适合短线。

问：飞哥，你的行业数据是在哪里找的，我在之家和天眼找的数据都不如意

【羿飞】 回答：数据来源，三方的肯定会有一些错误，因为技术抓取不一定可靠和稳定。我是这样的，三方的数据也都看，平台自己公布的数据也看。
最优先相信的是平台官网自己的数据，要每月抄录一下上个月的数据，下个月更新了就能算出差额。自己数据不完整的用三方的，三方中数据持续稳定性好的优先采用。
最近这半年多，因为数据收缩，很多平台尤其是一线的封锁了三方数据的抓取，其实在以前他们都是欢迎三方抓取和公布的，因为数据越来越好能带来关注。现在则因为不想被外部了解难看的数据。只能依靠每月抄数据了。

  • [幻想] : 飞哥我是问你的行业数据，例如每月行业代收余额，交易额等。
  • [羿飞 回复 幻想] : 那个没准确的，一般就是看看之家和天眼估算的。平台太多了，还有那么多线下的，没有人能真正搞清楚规模。
---------- 本文结束----------
好人一生平安。
