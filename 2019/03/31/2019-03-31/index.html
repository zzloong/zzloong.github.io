问：飞哥你好，参考你卖出其他股票的分析，云天化是不是也应该在出现收绿的当天选择卖出？

问：飞哥，这个点位，在继续追加指数基金是不是意义不大，可以等等了。

问：飞哥好，你对指数估值定投基金的方法，怎么看呢？各大基金交易平台都会提高指数基金的估值表。今后投资希望更多地提前了解风险所在。谢谢！

【羿飞】 回答：我不认为这个有科学性。把一大堆不同领域，不同市场的指数放一起比较，pe低的就投，高的就不投是不行的。因为每个市场实际上会有长期不同标准的pe。
比如即使是a股，也是深圳长期pe高于沪指的。难道只有等深圳指数pe低于沪指的，才能买吗？那可能1000年都没有买入机会。

  • [阿狼] : 光看估值也不行，得全盘考虑综合判定，才能提高胜率。
  • [羿飞] : 同一个市场，不同时间段去对比指数估值贵还是便宜，有实际意义。
    不同市场，同一个时间段去比较各指数的差异，没有意义。
    有点像买鸡蛋，菜市场3块钱，超市5块钱的话。确实看起来菜市场3元的便宜。但是你打算在菜市场3元买了鸡蛋，要在同一个市场5块钱卖掉，可就难了。因为菜市场地处偏远，会长期价格便宜的，搞不好给你跌到2元去，或者都臭了还是3元。
  • [Kelvin Fis 回复 羿飞] : 是在同一市场、同一领域去评估不同时间段的估值的。不跨市场、也不跨领域的。*
  • [羿飞 回复 Kelvin Fis] : 这样的可以，估值在历史周期中较低的时候，就可以开始定投建仓了。*

问：飞哥你好，拿着你的股票池自己瞎操作的，目前亏损10%，算什么水平？后续还能不能继续下水玩了？您操作下来目前什么水平？谢谢，

【羿飞】 回答：不知道你是什么点位介入的不好评价。从去年开始算，我目前处于3600-3800点左右，所以应该是跑赢大盘的。还是定投基金吧，股票可以买一点点慢慢学习和体验，不要当成致富手段。

  • [羿飞] : 实际操作每个人差异太大了，大多数人抵抗不住追涨杀跌的行为本能。股票池不能保证赚钱，基金池更适合绝大多数人。
  • [羿飞] : 举例子，很早以前就买过一些海航，平均成本6元左右。后来海航长期停盘，海航复盘以后，连跌了4个跌停板。这里我就非常惊讶，于是做复盘分析。当时这4个跌停算是补跌停盘期间大盘大跌的幅度，也算合理。但是其后因为恐慌情绪蔓延，还在惯性下跌。
    跌到3元以下的时候，我认为已经完全偏离了基本面，再怎么着作为集团成员牵一发动全局，海航集团是不能放着海航科技债务违约的，所以最终债务危机不会发生，企业会回答合理的估值区间。(这里假设如果是乐视那种老板跑了，或者长生要退市的股票，跌的再多我也不会参与)。
    之后我就开始卖出其他跌幅比较小的股票，换到海航里。最近的一波反弹，盈利30%+出来。然后进入其他启动比较慢的股票，又有了15%+的收益。所以这一波整体表现要比大盘好。
    但是我这个操作，是不会建议别人这样做的，因为不具备同样的判断力和心理承受力。
    我觉得大多数人可能会在复盘跌停结束就开始不断的补仓，而持续下跌到3元以下以后，不光不会再补了，反倒大概率精神崩溃，开始割肉出逃。
    还有一部分，期间低成本也超低了，但是大盘反弹以后，回本就跑了，吃不到太多的涨幅。
    股票投资中最难的就是判断和定力了，要求非常高。大多数人，还是建议大家就基金定投就ok了，中上等收益加省心。

【羿飞】 回答：如果一个行业，或者概念的板块，有多只股票发生了涨停，并且几天内有一定的持续性，那么就可以认为是这个板块成为了热点板块。交易活跃，资金都会比较追捧。一个热点板块内，一般会有1、2个龙头股，总之就是一直涨，涨到人非常怕，普通人是不敢追的。这样的龙头会带动整个板块的估值持续上行，直到板块整体估值明显过高，跟风资金减少，板块行情也就结束了。
