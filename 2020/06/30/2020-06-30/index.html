2020-06-30问答记录

发表于 2020-06-30 更新于 2021-04-11 分类于 Stock
本文字数： 3.5k 阅读时长 ≈ 3 分钟

问：飞哥，帮老人打理他们的钱，其中一小部分拿来定投基金池里的长牛型基金可以吗？

【羿飞】 回答：这样的资金是适合配置长牛型的，不太适合弄短期的板块基金。而且很建议给老人用银行卡买，不是基金软件。这样老人经常可以看看自己的钱在银行里，心里踏实。哎，老人其实不是图赚多少钱嘛，就是希望稳稳当当的要安全感。

  • [大吴] : 嗯，我只给老妈买了兴全趋势
  • [苹果树 回复 大吴] : 规模变大，董承非越来越谨慎，感觉控制回撤做得比较好的一支
  • [大吴 回复 苹果树] : 是啊，回撤时最好的防御产品
  • [Henry 回复 大吴] : 我给父母账号配90%兴全，10%黄金概念基金

问：飞哥，手头刚回一笔资金，现在还可以买二季度基金池里的基金吗？

【羿飞】 回答：可以买，随机大环境稳定向上。

  • [羿飞] : 我选基金池的时候，都不是超短线的思维，持有3-6个月都是问题不大的。还有一个月就有季报了，开始准备筛选新基金池啦。大体思路已经有了，只等季报出来明确判断。

问：一直没顾得上切换房贷利率。最近收到银行通知，说除了固定利率，lpr，在还可以不切换，保持原来的基准利率。请教下，lpr和基准利率选择哪个，更合适些？

【羿飞】 回答：降息大势所趋，绝大多数情况下lpr划算。

问：现在主打基金持仓了，目前对于主题型基金应该更关注板块行情的机会。
飞哥从技术图形上如何看待板块行情的结束(基本面的机会飞哥已经提过，是疫情后的民生通胀），这样能让群友能吃完整波板块行情。相信最近的医疗器械服务板块的行情好多人没吃够就跑了，最好结合同花顺图形，图文并茂

【羿飞】 回答：从来没想过能吃完整段行情。对于医疗最近的飙升不意外，这是一开始配置的目的，是判断的结果验证。那么既然实现了，也就要开始套现。我会分批逐步坚持，开始布局其他的。就像一二月份减持掉之前的科技和新能源，我不想去猜最终的高点在哪里。
目前继续持有基金池其他产品，农业，消费，黄金，科技等。还有一个多月就有季报，又能筛选新基金池了，到时候医疗应该也基本减持完，可以切入新加入的品种。

  • [羿飞] : 我已经开始卖出，但会分多次。
  • [羿飞] : 整体来说，对今年上半年的业绩非常满意。对下半年也充满期待，等七月份的季报啦！[拳头]
  • [大吴] : 从收益上看来飞哥的医疗减持是分批，而且是很慢的。目前生物制品，医疗器械等最近是最后的冲刺阶段了，踏实拿好我的农业食品了。
  • [羿飞 回复 大吴] : 是的，我在上期基金池中也说了，对于医疗的操作是且战且退，平滑过渡。我自己的操作是按照这个思路来的，其他人需要自己去体会节奏感，目标是操作的越来越好，没必要追求极致。
  • [羿飞] : 对于整个基金池，大家对不同基金的配比不同，导致收益上会有差别的。肯定有人医疗占比比我高，收益比我更好的。
    其实我自身来讲，不是特别介意整体涨幅是否特别大，我更介意通过市场波动验证方法的有效性。可以从上半年的整体表现看出来，对比300指数，在市场的波动中，随着时间优势在不断的扩大，说明是整体的持仓策略有效，不是偶尔运气好，这是最让人欣慰的。
    如果图形表现是一波特别猛赚了一下，但是其他大多数时候跟大盘波动基本一样，甚至优势收窄了。说明只是偶尔赌对了一个板块的行情，纯运气。那么赚的多少，都不值得欣喜。
  • [苹果树] : 北京疫情的新情况会不会让市场放不下对医疗板块的关注?
  • [羿飞 回复 苹果树] : 确实是带动注意力回到医疗了，催动了最近的一波行情。
  • [羿飞 回复 苹果树] : 目前看，疫情经济，最赚钱的是做试剂的，这个一直要消耗的。反倒是治疗的呼吸机那些，就是卖一批，各地医院够用了就不再买了。

问：飞哥你好，佳都科技和中粮科技这两只个股怎么样？后续有没有行情？谢谢。

【羿飞】 回答：看了一下，佳都科技覆盖的概念很广泛，想象空间蛮大。但是一季度的业绩下跌实在太多，看着有点吓人。而且大股东质押率超过70%，有一定财务风险。所以还是多观望观望比较好。中粮这个安全边际比较好一些，大股东都是国家的没有财务风险，今年农业应该是可持续的主题，业绩稳定增长中。

  • [龙猫] : 多谢多谢

问：飞哥，二季度的基金主打是农业和食品，但最近表现一般。三季度主打新基建了，还有2个月时间更新基金池，是否留给农业食品的时间不多了

【羿飞】 回答：布局了，放着慢慢等就是，业绩表现是会有的。新的季度有新的热点，但是也应该慢慢切换过去。比如医疗至今还表现不错，并不是说一个季度了马上就都结束了，基金不像股票一波行情周期那么短的。
加仓建仓，都用定投的方式分批搞是最佳方式。

问：飞哥您好，为什么说三季度汽车板块会涨？有什么逻辑在里面吗？

【羿飞】 回答：逻辑是疫情影响的周期可能会跟往年不同。在正常的年份是四个季度都会有销售额比较平均。今年一季度的销售额只有1月正常，二季度只有一半正常。所以销售额基本上都要压到下半年了，下半年的业绩会明显好于往年同期。当前的价格应该是处于一个利空出尽的状态。

  • [牛牛] : 有一些好股或基金推荐吗
  • [龙猫 回复 牛牛] : 个人观点，自负盈亏哦。我对比分析下来，广汇汽车，福田，江淮好一些。

问：飞哥你好，看个股现金流量表的时候，有三个参数，经营现金流，投资现金流，筹资现金流，这三个指标哪个更重要一些？现金流主要关注跟自身历史的数据对比，还是更关注和其他同行业企业对比？谢谢。

【羿飞】 回答：那应该是经营现金流最重要，因为这个是说明主营业务是否稳定可靠的。那些卖资产，投资收益之类的产生的营收都是暂时的。筹资多的也不太好，说明不是靠借钱过日子，就是项目还在筹资投入期，变成产能还得等很久。

问：飞哥你好，好久没看到个股诊断了，是没人提问了，还是个股提问都屏蔽不回答了？圈子少了个股分析，感觉没以前精彩了。

【羿飞】 回答：不会啊，最近就没有什么人问个股。你如果问可以提。不过我是比较建议大家优先基金的，前面也说过了，机构的时代来了，都是大基金报团取暖。以后个人散户越来越难生存的，实力较小，进不了大基金圈子的小基金，小私募也是玩不转的，很多小型私募会清盘。

问：飞哥好，请教个关于“中日韩自贸区”的问题。李总在5月28日又提到了自贸区的事儿，结合香G和中美毛战的国家对策，自贸区是否会加快进程?另外涉及到的相关城市大连、青岛、威海(因萨德中韩自贸区搁浅)等等，飞哥能帮忙逐个分析分析不？

【羿飞】 回答：中日韩自贸区应该说是中国最最最重要的贸易战略了。永远都不会放下的，但是任重道远。因为不是中国自己单方着急要办就能办成的事儿。最终是要美国退出亚太，中日韩才能真正走到一起。到时候三国的中贸易规模是超过欧盟的，世界第一大经济圈。

  • [木屐] : 这么多年了，川建国和疫情好像让它有点希望了
  • [羿飞] : 😄还真的是看着有点意思了。

问：饭团现在什么情况。下一步怎么处理？我们要怎么做？

【羿飞】 回答：不知道呀，去年年初，后来的借委会成立以后，我就什么都没管过了，不知道。我是记了坏账，不愿意为这些事情再操心闹心了，还是赚新钱容易一些。
感觉没啥可做的，等老王出来吧。

  • [龙猫] : 老王什么情况？还能出来吗？

问：飞哥，最近债基跌的很厉害，要卖掉暂时规避吗？

【羿飞】 回答：哈哈，没那么严重，又不是p2p，不会打水漂的，也就是短期内受市场影响有点回撤，以后会慢慢再涨回去。不过如果你怕，暂时卖掉买点其他的，以后市场稳定了再买回来也行。没必要影响心情是不是？

  • [罗晓莹] : [憨笑][OK][OK][OK]

问：属于不定期定投类，请教飞哥和圈友们，6月初这样的形势，对于兴全趋势这类主动管理型基金6月是不是还是一个可以逢低吸纳的机会呢？

【羿飞】 回答：兴全趋势是品质很好的长牛，管理能力优秀，持续性也很好。是可以慢慢定投的。你不要指望他每轮行情能特别特别出色，但是长期中前的位置大概率。就像班里一直90分的好孩子。

  • [大吴] : 最长牛的基金，成绩一直中上等
  • [苹果树] : 谢谢，有时候行情涨的时候，反而不太敢买，怕成本高了

问：飞哥，近期有很多大基建的新基金不断上线，可能都是围绕5G主题，但是最近科技股跌的很惨，是不是有点坑啊？还是可以慢慢布局？希望给一些指示

【羿飞】 回答：5g的话，长期看一定是大方向。但是去年科技类已经涨幅很高了，很多是包装在5g，新基建这个大概念里浑水摸鱼找接盘侠的。
需要调整一波，最近观望比较好，等新的一波开始的时候才布局比较好。我目前还是主打消费和农业。

问：飞哥，看你文章解读，马上要放水了，水流向哪些行业？内需集中在哪些行业？

【羿飞】 回答：我认为水会流向所谓的新基建，但是新基建的概念太大了，不是所有的板块都能有很好的机会。
最好的应该是5g铺设的硬件相关。其次是高铁建设施工和材料方。其次是新能源的充电桩施工和材料方。电池的话估值有点高了，新能源车本身还不到规模化盈利的时候。
然后传统基建设备应该也能分享一些的，不会说有新基建了，老基建就万全抛弃了。

  • [龙猫] : 谢谢
