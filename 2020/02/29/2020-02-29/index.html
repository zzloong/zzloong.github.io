2020-02-29问答记录

发表于 2020-02-29 更新于 2021-04-11 分类于 Stock
本文字数： 1.6k 阅读时长 ≈ 1 分钟

问：飞哥，我在转换基金池时，留了一点信达澳银新能源和万家行业，其他都出来了，因为收益率接近40，所以没舍得全部下来😄，是不是要干脆点全出来？后期飞哥对这两个基金有啥看法？

【羿飞】 回答：我不觉得这俩基金马上就会下跌，而是觉得再持续一个季度有难度，从保守的角度会先剔除。大长期看新能源，科技都是高增长的，不会永远不关注，等有回调估值合理的时候再入。
我的思路偏保守，大家自己完全可以根据自己的判断决定去留。

  • [大吴] : 万家行业股票占比从90%多减持到70%多了，换成现金了
  • [向日葵] : 师傅领进门，修行靠个人

问：飞哥，黄金飞起来了，现在能介入白银吗？

【羿飞】 回答：黄金先动起来的时候，短线白银没启动，是可以买一些的，跟涨几率大。但是记得只能做短线，尽快卖出。因为长期看白银缺少金融属性，长期看没有黄金的上涨潜力可持续。

问：飞哥，石油的话您怎么看，现在油价低迷是由于中国需求少吧?这种情况油价能跌到多少美元一桶去?

【羿飞】 回答：其实油价应该是进入复苏阶段了。因为第一阶段协议，中国已经降低了美油关税，美油进口就应该负数了。不过突发的中国疫情可能会影响中国经济增长，进而降低全球原油需求，导致预期降低。
这种变化其实是之前任何人都无法预测的，所以只能走一步看一步了，看中国疫情的复苏情况，现在这个牵动全世界的经济。
我个人认为这种强阻断的政策最多执行到月底。

问：医药股连续调整，是正常调整还是行情结束了？

【羿飞】 回答：我认为是正常调整，是很好的增持机会。医疗这一波的持续时间会比较长，不是几天的概念行情。

  • [羿飞] : 其他股是超跌反弹，缓口气，中期看阴跌或者底部震荡。而医疗产业的，是猛涨一波回调喘口气，均线到了继续上行。
  • [今天Q] : [抠鼻][抠鼻]
  • [牛牛 回复 羿飞] : 医疗板块基金00523前期看还是挺抗跌的，最近回调反而都是领跌挺大阿

问：飞哥，一直想问个问题。在您给的基金池的文档中，1、ETF 指数型和可持续配置长牛型基金，是不是属于可以长期做定投的品种？2、尝试配置的基金，是不是要短期布局的？3、垂直行业基金，是做定投还是长期定投的品种呢？谢谢飞哥了~

【羿飞】 回答：1.etf是最适合做定投的品种，基金经理主动性影响小，适合不相信主动基金的人。
2.长牛的就是长期表现不错，即使短期内表现不太好也不用慌的，至少不会被套。
3.尝试性的属于我认为符合近期市场风格的，但是这个风险性也比较大，因为需要验证。
4.垂直类的其实跟最近市场风格无关，是一些可能会常用到的差异性工具，我只是帮忙挑选出来，在同类里都算不错的，方便大家需要的时候使用。

  • [牛牛] : 飞哥能否教教我们这些基金怎么挑选，从那些维度来看一些基金好与不好呢？
  • [羿飞 回复 牛牛] : 思路每次的基金池，不都详细的讲过了吗？几千个基金，缩小到十来个，也就这样了，看哪个有眼缘就买哪个吧。[捂脸]
  • [大吴 回复 牛牛] : 选飞哥基金池里持仓股跟热点的，不高估的。基金近期业绩好的
  • [牛牛 回复 羿飞] : 我再练练选基的技艺[微笑]

问：飞哥，最近3-6个月医疗基金占基金总额的多少比较合适？

【羿飞】 回答：至少应该占一半吧，最近一段时间，其他产业几乎都停工了，只有医疗产业和相关的，要开足马力往前冲。

  • [大吴] : 我是科技和医疗各一半，看来还得➕
  • [幻想] : 最近券商的比较弱势，按理说券商都基本上是网上交易，受影响很小，看最近的走势应该是过度反应了。不知道这种想法对不？
  • [羿飞 回复 幻想] : 券商是跟指数的，因为指数低开了嘛，所以券商也会跌很正常。但是券商去年至今的业绩确实是好，股价没怎么涨，实际上相当于开始进入低估期。

问：飞哥。您说 买南方原油可以吗？我觉得南方原油可能更贴近一些国际油价

【羿飞】 回答：可以的呀，南方原油属于油价跟踪工具，华宝油气是跟踪油气行业上游资源的企业股票指数的，跟行业的繁荣期有关。属于指向了一个大的行业，但是着力点并不同，根据自己的需要去选择工具即可。
