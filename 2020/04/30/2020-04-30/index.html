2020-04-30问答记录

发表于 2020-04-30 更新于 2021-04-11 分类于 Stock
本文字数： 4.3k 阅读时长 ≈ 4 分钟

问：请问飞哥每次筛选基金的操作流程是什么
是按照基金收益排序查找吗，比如按照近1至3年收益排序，然后排除排名靠前的之后再逐一核对。
还有按照看好的板块筛查近期可能崛起的主题行情，天天基金里正好有基金主题的分类工具

【羿飞】 回答：这基本上就是问我每次筛选基金池的逻辑了，基本上我是按照如下方法做的：
第一步，先研究大趋势，确定自己对哪些行业未来一个季度看好。这个阶段主要是考验宏观的解读判断能力。
第二步，在基金软件的行业分类里，选自己看中的板块，一般每个主题板块里有10-30个基金不等。选出3-5个比较看好的，这个筛选的过程主要看各基金的主力持仓去比较，高估股占比少的优先。这里主要考验对各主力持仓股的月线技术图形解读能力。经验越丰富越省时间。
第三步，在选出来的二三十个基金里交叉对比，进行一波淘汰赛，主要对比各基金的短中长期表现力。各周期都优秀的胜出。这里主要考验市场历史形态的记忆。大脑里要大概有过去十年内起伏的图形，就比较容易做出对比。
第四步，选出来的，以末尾淘汰制的方式，去pk目前基金池里的基金。干下去的剔除。目标是优中选优，让基金池越来越优质。这里需要的是公正，你要忘掉哪个基金给你赚钱多，哪个基金给你赔钱多这样的心理印象，和主观色彩。人们很容易有这种感觉，记忆越强烈，越不容易公正。

以上就是我每次筛选基金池大概的过程，其实就是力气活。并不能肯定每次筛选的就一定正确，一定有多好。都要去通过市场验证，每个季度都存在着巨大的未知。但是花了时间和精力去做，就会更有信心的去持有，不会太容易受到市场的短线变化信息干扰。

  • [大吴] : 精华帖! 好系统的流程，建议自己先按照这步骤选择，然后等飞哥概念基金更新再对比。
    还有看月线技术图像时，有些个股月线时间很短，刚上市几年的，看上去拉的很高不一定高估。还有的白马股，月线斜线上去的，营收是逐年递增的，至少也不算高估
  • [羿飞] : 补充一下第三步历史图形对比。你要对股票市场的大事件有历史记忆。比如一个基金的图形你看的时候，发生在2016年初有比较大的回撤，那就很正常，因为那个时间跌的极凶。而在这些大型事件之外，市场比较稳定的时间，一个基金发生的大的回撤则是不合理的，可能存在利益输送，老鼠仓什么的，这种就要回避。
  • [龙猫] : 20个基金，每个基金10大持仓股，总共200只个股，巨大的工作量。

问：飞哥能否点评下奥佳华，口罩概念股，一季度利润下降为负，现深套中，还有翻身机会否

【羿飞】 回答：我觉得有机会，可以回忆一下，一季度大多是时候在停工，3月陆续恢复上班，才开始有产能。业绩释放应该是在后面，口罩应该还有一波行情，能涨多少不一定，解套应该是问题不大的。

  • [龙猫] : 深套大概率发生在口罩概念最热的时候买入的，后面越跌越深了。[呲牙]
  • [霍尼韦尔-张挺] : 的确，我也是深套，虽然有口罩概念，但是主营业务出口按摩椅等健康相关的产品，Q2海外疫情影响，冲击仍然很大。我已经割肉离场了……虽然还是不甘心，但是错了就承认吧
  • [龙猫 回复 霍尼韦尔-张挺] : 口罩概念股还是挺多的，不差这一个。

问：飞哥，你好，对于某一件商品的价格上涨炒作，比如口罩，医药，食品，我觉得都是类似的。我把它大致分为三类型的企业受益，原材料，加工企业，销售企业，这三种类型的企业一般是如何分配利润的，哪一家受益更多？第二个问题是，在炒作这些热点的时候，一般的板块轮动顺序通常怎么样的。第三个问题是你有一篇公众号文章给出了一些受益板块，那个里面的先后排序是否有参考价值？谢谢。

【羿飞】 回答：这个问题有点笼统，这是个产业链的话题。到底一个产业链，上中下游哪个环节最强大。其实这不是一个固定的答案。不同行业，不同时期都会有不同的结果。
比如原油现在暴跌的很低，那么上游会亏损的很惨，但是下游的销售企业，收入会缩小，利润还是能保持比较稳定，相对是不特别难受的。但是如果是油价很高的时候，爆赚的就是上游企业了，下游销售企业还是适中。
但如果是汽车产业，大多数时候都是上游制造商跟强势，渠道议价能力不强，利润很微博。但是在电器行业，渠道就比厂商强势很多，是下游厉害，一直都是。
所以，你这个问题，并没有标准答案，需要我们花更多的时间去研究了解我们的投资标的。

问：由粮食安全话题引起的农业股近期涨疯了，但是产业链相关的化肥却没怎么涨，这波行情有可能传导吗？国际农业如大豆玉米各大宗期货价格目前都是历史低点，后期价格肯定会反弹，农业股还不能追？

【羿飞】 回答：板块级的机会应该还是会持续一阵子，但是个股就很难讲了，影响因素太多。还是建议找相关基金去配置。

问：飞哥，感觉一次行情的切换都伴随着一次大盘调整或大跌。
比如去年的科技板块大涨到今年大盘调整后科技熄火。 还有2016年初到2018年初的沪深300上证50行情，也是一次大盘整个2018年的下跌后行情切换到中小盘

【羿飞】 回答：这大概就是盛极必衰，阴阳切换的自然规律吧。每次有行情的时候，必然是有一个类型的股票主导的。有时候是大股票主导行情，有时候例如15年是创业板主导。
主导行情的股票们在一波到达定点的时候，也是泡沫化最严重的。泡沫破裂以后，股市下跌。下一次行情起来，一般就换其他类型主导了，因为他需要继续释放泡沫，需要更长的时间休养生息。
2017-2018基本上是大股票在主导，所以近期我更看好中小的会表现机会更多，而且会有一定的持续性，不会仅仅是1、2个季度，风水轮流转嘛。

  • [大吴] : 所以科技，新能源不能碰了，需要休养生息
  • [苹果树 回复 大吴] : 你说今年中小的保险度高，至今观察下来挺准的
  • [大吴 回复 苹果树] : 不难啊，你看中证500现在什么位置，沪深300和上证50已经到大盘3500-4000点位置了

问：飞哥好，我看桥水基金收益率和余额宝差不多，谁会把钱放在这种基金里呢，怎么就名声在外了呢

【羿飞】 回答：桥水是对冲基金呀，大资本配置这个资产类型，追求的是稳定，以抵御不可预测的突发性损失。所以收益率不是主要目的，他有自己存在的原因和价值。
你觉得余额宝受益没意思，人民币利率高，他们进不来呀。
对于0利率的那些发达国家，这就是很高的受益了。能承接那些大资本规模的产品并不多。

  • [木屐] : 飞哥好，从这次疫情，看这基金对冲效果也不行啊，听说亏大了，08年的风采不在鸟
  • [无敌草 回复 木屐] : 因为对冲基金太多了，要卖一起卖，之前美元流动性不足，造成踩踏

问：飞哥，前不久买入了三江购物和步步高两个零售概念股。等一季度报出了以后，五一后还可以继续拿吗？

【羿飞】 回答：我觉得可以，要倾向于超市业务占比高的零售企业，不要商场类的。今年一整年的业绩都会比往年好。

  • [大吴] : 前几天刚买的步步高，超市业务70%

问：招商债券，限额2000元，这是有什么预兆么？飞哥

【羿飞】 回答：一般限额，都是想买的人太多了，退出的份额不够申购的多，才会限购。所以如果债券开始限购，说明最近普遍避险意识更强了，大家打算多持有债券，不敢碰风险资产。

  • [霍尼韦尔-张挺] : 谢谢飞哥
  • [龙猫] : 可以换其他的债券啊
  • [羿飞 回复 龙猫] : [憨笑]他应该知道也能买其他的。但是比较警惕，想弄明白是不是有什么特殊信号。
  • [苹果树] : 不是一直在限吗？放开过?

问：请问飞哥，场内原油比场外比较溢价很多，最终放开场外时，应该是和场内价格相近时。那么会是如何实现价格相近呢？
是场外涨的比场内多，慢慢追上场内的价格呢，还是场内跌到和场外价格相近才放开场外呢

【羿飞】 回答：就是两种可能啦，一种是油价始终没有快速反弹，场内的慢慢失去了希望，就会价格慢慢下滑到和场外接近。
还有一种是油价反弹了，场外会跟随反弹上升，最终和场内接近了，两种一致化。
所以不管是上面的哪种，现在买场内的都很不聪明，上升没份儿，下降却会亏。

问：飞哥，参考09年4万亿后A股的表现，疫情的大放水过后A股会迎来牛市吗？
A股散户居多，过程还是慢牛变快牛变疯牛再跳崖的过程吧，像美股这样的十年长牛估计不会有

【羿飞】 回答：我个人是这样的观点，疫情缓和以后，放水效应先把股市顶起来一波，然后下一波是房地产。再后来就要持有债权比较好了，进入新的一轮去杠杆。到那个时候，欧美应该也复苏了，去抄底。
希望我看的对，这样大家人生都能上个台阶。

  • [秋天的小哥哥] : 债权是什么
  • [大吴] : 飞哥，这几个操作时间跨度比较长吧，大放水阶段怎么后面又去杠杆了
  • [苹果树] : 房地产? 据说北京的房价降了10％了

问：飞哥您好🙂图是今年基金的收益，股票的收益跑输大盘一个点。谁也别拦着我，我承认自己笨了，以后继续加大基金占比，飞哥啥时更新股票池和基金池？我想等更新了在调整。还有就是原油都停了不能买、黄金历史高位，这两样想加仓也都有点纠结。

【羿飞】 回答：哈哈，这就对啦，做人一定要客观，通过数据，不断修正和调整资产结构，越来越优化就是进步。

  • [大吴] : 我也想把股票都换基金了
  • [华子] : 飞哥基金池是不是快更新了
  • [华子 回复 大吴] : 股票还得天天跟着操心
  • [秋天的小哥哥 回复 华子] : 还是基金好，大盘跌基金少跌，大盘涨基金多涨

问：飞哥，兴全趋势基金从2005年大盘1000点时，经历了15年到现在大盘2700点时通过稳赢大盘复利15年已经涨了16倍多。
目前的大盘2700点肯定在低估状态，如果现在建仓兴全趋势基金，在目前持仓股高估的状态下(白马股），持有下一个15年还想再翻15倍有可能不？就当一份退休金了

【羿飞】 回答：我这么看，这个基金肯定属于长期白马基金，选他作为长期定投的工具属于稳中有进的方案，至少不算犯错误。
但是你不能指望15年一定翻15倍。要看大盘给不给机会呀，也许10年就给你实现了呢，也没准30年才实现。

  • [大吴] : 我觉得给两次大牛市就能做到了，和大盘同步涨幅，下跌时跌的少

问：飞哥，疫情不断恶化，世界经济已经很可能步入危机甚至萧条了，但国内由于控制的较早较好，所以机会很大。鉴于这种情况，你认为现阶段股票包括基金，黄金，债券，银行理财各占多大比例比较合适？谢谢

【羿飞】 回答：建议比例：基金40%，黄金10%，原油10%，债券基金30%，准现金货基5-10%。银行理财不考虑，没有性价比。说明一下这是普适性的参考，不代表是我的持仓比例，每个人根据自身情况不同，肯定是不一样的，避免误导。

  • [秋天的小哥哥] : 原油才10%呀，我都打算提到50%
  • [大吴 回复 秋天的小哥哥] : 别超20%
  • [羿飞 回复 秋天的小哥哥] : 慢慢加嘛，不要太激进，万一判断错误，就没有未来了。
  • [秋天的小哥哥 回复 羿飞] : 是的慢慢加，就是买不到[流泪]啥时候能正常申购呢
