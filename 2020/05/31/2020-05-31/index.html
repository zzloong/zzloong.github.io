2020-05-31问答记录

本文字数： 3.5k 阅读时长 ≈ 3 分钟

问：飞哥你好，怎么看有一些雷台推出的换购东西？谢谢。

【羿飞】 回答：其实就是变相降低清债的成本。进货价批量买低一些，高价换购。但是从很难拿到本金的现实情况，如果能选一些很容易在咸鱼卖掉变现的东西，还是可取的。

  • [不进则退] : 希望饭团也赶紧出个换购吧，比较担心饭团，最近没啥声音，其他台子能退退了，能换也换了[流泪]
  • [鹅卵石] : 饭团网站都登不了呀！
  • [王姐] : 请问哪些雷台出了换购东西的政策？
  • [龙猫 回复 王姐] : 丁丁，融金所。
  • [龙猫] : 平台就不能简单一下啊。直接按照物品购买价格把债权打折就行了，还藏着掖着呢。
  • [霞] : 融金所、和信贷……能兑换东西的我都兑换东西了！就剩365和饭团了。希望早点出方案，懒得耗下去。
  • [沙发上的小枕头 回复 霞] : 和信贷换的啥，待收十几万也想换[流泪]
  • [霞 回复 沙发上的小枕头] : 一醉解千愁！酒[捂脸]还有一些床上用品。

问：飞哥，我的股票持仓都是股票池里的口罩概念，医药商业和中药，目前跌的很惨，后面这几种还有机会吗，是否热点已经转移其他了

【羿飞】 回答：上期的基金池文章里已经说过观点，热点应该是已经在向农业和食品，新基建等转移。之前的会有反弹，但是比这些新热点的能量，相对应该是弱的。
还是建议大家更多配置基金板块把，最近市场整体还是偏弱，个股风险会大过基金。

问：后面新三板出来了，会不会有吸血效应？主板会有受影响吧？

【羿飞】 回答：抱歉，我仍然不看好新三板。这个东西一直都太妖，就是各种套资金割韭菜。前阵子很多券商在推新三板打新，这里应该又是一个大坑，建议大家不要碰，绕开。

问：飞哥你好，今年的年报行情我感觉很尴尬，2019年年报即使特别好看，1季度普遍缩水到去年的一半甚至更低，对冲了一下，这样股票基本不涨甚至下跌。食品、超市板块营收利润普遍增长，医药板块下跌20-40这个区间，其他的跌倒一半甚至一倍以上的比比皆是，所以医药个股即使下跌，也还可以接受。您怎么看以上观点。谢谢。

【羿飞】 回答：这个怎么说呢，我跟你一样有很重的纠结。即一方面因为疫情年报数据很难看，以及美国疫情崩溃，肯定对A股也有影响，有风险方面的担忧。
一方面呢，也担心大放水之下，随时资产价格会膨胀，退出来踏空。所以最终我的操作策略是跟着业绩稳定一些的板块走，即使可能回撤，跑赢大盘即可，人不可能有完美的操作的。一季度看好医药，二季度看好食品农业超市等的观点继续。如图实际业绩表现我也基本满意，比我退出持币看着着急要强。
个股方面最近尽量回避，业绩雷随时出现，谁也说不清疫情到底对每个企业具体的影响是什么，都要揭开盖子才知道。
事实上，我们可能发现，虽然疫情对实体经济影响很大，但是股市好像没有实体经济那么大创伤，还是小幅区间震荡。个人认为，这可能是放水已经在进行中，流动性对股市进行了支撑。即使业绩这么差，股市也没有大幅下跌。也就是说如果下半年上市公司业绩稳定以后，整体创新高走强就可能出现了。

  • [木屐] : 美国这么烂的疫情数据，股市都追回来了。可见影响经济的不是疫情，可能是应对疫情的措施，群体免疫这条路说不定真被人家走出来了。让时间证明吧，我们都是见证者。

问：飞哥，这次债券出事，对债券基金影响会有多久? 建议移仓吗？

【羿飞】 回答：最近出问题的主要是可转债吧，不是债券基金的主要业务类型。会受市场波动有点小影响，长期影响不大，估计不会影响到一个月这样周期的收益。我们基金池 选的几个债基还都是一直挺稳的。

  • [王姐] : 请问飞哥，我之前一直没有接触过债券基金，上个月刚刚购入好几只债基，最近有好几天收益都是负的，还有些日期收益为0，这个正常吗？大概需要多久的时间才能看到稳定收益？
  • [龙猫 回复 王姐] : 三个月以上吧。

问：飞哥您好🙂基金池新加了几个沪港深基金，是对港股整体已经看好了吗，还是只对这几只基金现在持有的股票看好。我犹豫加不加这几只呢

【羿飞】 回答：是对他们持有的股票和板块看好，而非港股看好，阶段性行情也有机会。

  • [华子] : 谢谢飞哥明白了[抱拳]

问：飞哥，之前大吴哥筛选的610002这个基金，你说如果暴跌可以买着长期放，现在这个逻辑还存在吗？之前在高位买了一些，现在是否可以再买一些来拉低均价？

【羿飞】 回答：我不记得有这样的话，你可能搞错号码了……

  • [罗晓莹] : 是信达澳银精华配置混合
  • [羿飞 回复 罗晓莹] : 这个之前已经操作完减持了，而且减持逻辑也说过，清清楚楚，去看以前的基金池，不知道你现在这个逻辑在哪蹦出来的。
  • [罗晓莹 回复 羿飞] : 不是之前基金池的哦
  • *[罗晓莹] : *
  • *[罗晓莹] : *
  • [大吴 回复 罗晓莹] : 000166是很好的替代品，新基建概念
  • [羿飞 回复 罗晓莹] : 奥，知道了，看成澳银新能源了。这个基金从历史成绩看一直跑的不错，如果喜欢可以慢慢定投增持一些。我并没有深度的研究过。
  • [罗晓莹 回复 大吴] : 谢大吴哥！

问：飞哥能不能诊断一下中油工程，原油跌他跌，原油涨不涨，基建涨他不涨，而且越陷越深

【羿飞】 回答：原油价格这样剧烈的暴跌以后，暂时是不适合碰原油相关的东西的。只有油本身可以抄底，通过原油基金。但是最近又限购，所以只能观望等待了，切勿贸然在不了解的地方出手，之前纸原油已经有人交学费了。
钱在手里，总有好机会的， 干嘛着急呢？

  • [大吴] : 只有原油上涨了，原油工程类才有需求

问：飞哥，感谢新的基金池。你能否讲一下基金购买的策略呢？ 长牛型适合一把买入还是定投呢？ 尝试建仓的用什么策略会比较稳妥呢？ 谢谢

【羿飞】 回答：这并没有绝对的方法呀，每个人资金情况不同的。长期收入现金流好的人就每期定投，只是操作固定资金没有新血液的人谈不上定投，属于移仓。
我比较建议的就是分批操作。比如你看好几个基金，也选好了撤退的基金，那么也不要一次操作完，分多次去操作。

问：飞哥你好，我记得只之前有那种股票分析，云天化，海航科技之类的投资分析，想问一下，能不能搞一只或者几只基金也分析分析具体思路。谢谢。

【羿飞】 回答：个股分析搞怕了，因为很可能我就做一段时间，获利了结早走了。但是有人因为关注过，就始终盯住不放，导致结果很惨。我越来越发现，炒股是个有趣的游戏，但是99%并不适合拿这个东西赚钱。
以后多关注基金吧，我们多聊基金，对更多的人来说是好事。

  • [龙猫] : 飞哥，题目是能不能具体分析一只基金，感觉你这个回答有点歪。[呲牙][呲牙][呲牙]
  • [羿飞 回复 龙猫] : 每一期的基金池，其实都在讲述筛选逻辑，其实常看基金报告，基本上能搞清楚我筛选的逻辑的哈。还有就是选每个的思考是什么，我都写了。并不是神神叨叨的拿一个什么方案出来故作神秘的。
    我做投资的风格其实一直是这样，依据逻辑，依据方法，结果随缘哈。也不敢肯定就对，一切需要市场检验。

问：飞哥，后续看好农业，逻辑能简单分享下么？

【羿飞】 回答：逻辑确实就是非常简单。不管是中国还是世界，都会因为贸易倒推，老百姓降低收入。也就是大多数产业会有收入倒推。老百姓手里有限的钱，会优先消费在吃上，这是生存根本，然后才考虑其他消费。所以这个产业相对影响会比较小。

  • [霍尼韦尔-张挺] : 所言极是

问：请教飞哥，新基金池里的164403，近6月线大幅跑输给大盘是什么原因？是有老鼠仓利益输送，还是前面涨了小高潮之后的退潮呢

【羿飞】 回答：因为之前农业不是很火热的板块嘛，跑输很正常。最近是全球动荡，食品应该都开始紧张了，他就优势大了，而且短期内好像结束不了。

  • [大吴] : 飞哥选择食品农业的原因是看的国外市场吗，国内复工差不多了吧 对食品等基本物质需求量再减弱吧
  • [不进则退] : 这个好像前几天都买不了是吧？

问：🙂飞哥您好：上次更新基金池您非常看好医疗板块，给的建议是医疗板块至少配置五成，事实也证明了判断👍，您这次有比较看好或重点配置哪一类吗

【羿飞】 回答：基金池已经给出答案了，后续看好农业食品，以及泡沫化不太严重的科技。

问：飞哥，同济堂是什么情况？属于黑天鹅吗，下一步怎么操作？

【羿飞】 回答：我认为属于黑天鹅了，证监会忽然发了检查，还要立案，鸭梨甚大。感觉有时候真的没话说，股市比较强劲的时候你不搞规范化。现在这么虚弱，人心惶惶的瞎tm折腾啥啊。

  • [龙猫] : 飞哥，这也是选择基金的一个好处吧。避免个股黑天鹅
  • [羿飞 回复 龙猫] : 对，现在感觉基金池分散的方式越来越好。主要是黑天鹅赶上一个就被捅一刀。那种未知的不安全感让人很难受。

问：飞哥你好，看来油价要长期低迷一段时间了，原油消耗企业消耗掉以前买的那些高价原油以后，马上应该要进入暴利期了。请问一下，哪些板块的相关企业哪些收益最多？

【羿飞】 回答：现在情况很复杂的，不是能肯定的判断。虽然是用油成本降低，但是经济不景气，企业的销售额一般也都下降很多。到底是收益大还是负面大，都要等1、2个季度各企业的财报表现。
