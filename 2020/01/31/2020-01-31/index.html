2020-01-31问答记录

发表于 2020-01-31 更新于 2021-04-11 分类于 Stock
本文字数： 2.1k 阅读时长 ≈ 2 分钟

问：羿飞，新的基金池什么时候更新

【羿飞】 回答：应该是下半月，各大基金公司更新上个季度最新持仓以后，我就能筛选了。[憨笑]大家不用太焦虑的，目前基金池仍然不弱。一月目前一周半5.2%，上证1.38%，跑赢了3.82%，大幅领先中。

问：飞哥，您空的时候能给我们小白普及一下，您的卖股依据不，您买股票是按行业分析挖掘价值低洼的股票，长期持有等市场发掘，前面您说过股票买的好，卖的不好，小赚，卖的好，买的不好，小赚，买的不好，卖的不好，大赔，买的好，卖的好大赚。我想请教一下，您一般的您一般是看什么卖出的，大盘疲软？价值分析到20倍PE?金叉，死叉？股市泡沫6000点，全民炒股时期，另外再推荐一下，您认为适合小白理论学习的书籍，视频。

【羿飞】 回答：真的没有什么绝对的依据，很多时候就是看盘感，个股操作因人而异，每个人都会做出不同的判断。
综合来说，我要持有一个股票的前提，是他仍然处于低估状态。如果短期内没有明显的负面消息但价格下跌比较多，我会增持摊薄成本。如果是黑天鹅例如发生了财务造假那么不论现在亏着多少，都马上卖出了结。
如果持有的一堆股票里，有的表现比较好，短期内不会动。要看大盘最近行情是不是到了比较中后期。如果是行情刚刚启动我不会急着马上卖出，后面可能会有更超预期的表现。
大盘猛涨了一波，震荡整理的时候，我卖出的可能性比较高，这时候适合换股。但是并没有你说的一个什么绝对指标。

  • [羿飞] : [微笑]我觉得也不用花时间去琢磨什么依据，如果有一个方法有用，那么所有人都会用，这个方法也会失效。方法正确有效，所有人都能赚钱，这种追求本身不切实际。市场长期来看是博弈，更像是赛场。请问在赛场上，有没有一个投球的方法肯定能赢呢？基本的方法就那些，竞技状态，面对意外情况的处理能力，等等才是胜败的因素。锻炼更努力不代表一定赢，胜率会更高一些。
  • [幻想 回复 羿飞] : 确实，投资是门艺术，时机，人，运气缺一不可。

问：飞哥，怎么看这个基金？越南好像发展得不错

【羿飞】 回答：我不建议大家碰这种产品。第一是我们看不到很多数据，无法判断什么时候可以持有什么时候能卖出。第二是因为中美贸易战，越南得利了一下这两年股市不错，最近去怕是要接盘侠。

  • [剑气满天花满楼] : 一般买旧不买新啊，就算是明星经理领军的，我也是遵守此原则，为什么去触发那个不确定性呢？买基金追求的是长期成长性，而不是新的1元认购很便宜，旧的净值好几元太贵了
  • [羿飞 回复 剑气满天花满楼] : 有些基金很无耻，长期跑输大盘，但是老搞分红忽悠。比如大盘涨了20%，但是他们涨了10%而已，但是他们分红10%，净值回到了1元。这样很多不了解的小白还以为是他们赚钱了有分红不错，因为他们不清楚大盘涨了多少没有对比。然后他们会跟亲友说这个产品不错可以买点。亲友一看，矮油，净值还是1块钱，不贵啊，就买了。
    必须要懂得看长期收益的，还要多对比几个纬度的数据。最需要确定的就是基金是没有道德问题的，是靠业绩吃饭，不是歪门邪道。

问：选了一个新能源主题基金，近期业绩也不错的，飞哥给看看，农银新能源主题 002190

【羿飞】 回答：这个基金最近表现不错，因为新能源板块整体飙升，属于跟涨。但是长周期看这个基金的管理能力一般，并不适合长期持有。鉴于近期正是要重新筛选基金池，要等各大公司更新了持仓，我才能做出下一步的布局调整，所以近期我的策略还是之前的持有不动。

问：飞哥，中东局势导致黄金和原油暴涨，是否应该暂时获利了结呢，等回调后再入

【羿飞】 回答：这几天中东紧张形势不断升级，前些天加原油的朋友们应该已经赚了，可以再拿几天看看。另外没有入的也不用急，追高犯不上。原油涨价，必然带动其他化工产品涨价，大家可以开始选择化工股开始布局了。

  • [羿飞] : 股票池追加一个600339中油工程，属于板块内比较低估的。因为中东大乱，美国油目前又不能买，我估计国内石油石化会加大开采勘测力度，利好石油工程类的企业。
    剔除前些天临时加的小米，一个多月已经涨幅30-40%，应该大跌不至于，但是中期盘整稳固等均线会有的。调整稳定了再杀回去不迟。
  • [霍尼韦尔-张挺] : 已经买入，应该有机会
  • [会飞的鱼] : 股票池更新了吗？

问：请问：如果理财收益率低于房贷利率，是不是应该提前还贷？

【羿飞】 回答：你已经预设了前提，就是确定理财受益低于房贷利率了，那么是肯定要还掉比较划算。这样财产损失更小，而且降低杠杆，降低风险率。

问：飞哥，这些基金怎么看

【羿飞】 回答：

  • [羿飞] : 在投资过程中，幸存者偏差错觉普遍存在。会给很多人产生不现实的参照目标，进而在操作上造成误导。所谓追涨杀跌就是如此。
  • [霞] : 对号入座[捂脸]
  • [羿飞] : [偷笑]其实大家去游戏厅，赌场玩的时候就是这样的感觉。大厅里到处都是哗啦哗啦响，好像所有人都在中奖，于是一冲动下了重注。结果就是自己投的币不中奖，默默的走开了。[憨笑]其实90%的人没中奖，但是没有人会跳起来嚷嚷，我是sb，我老不中奖！我们听到的都是10%中奖者的欢呼。
  • [罗晓莹] : 谢谢飞哥[抱拳]

问：飞哥 半导体等科技股要熄火，想把基金池里的科技类基金往中小板转移如何

【羿飞】 回答：

  • [大吴] : 那就逢高减持逐步转移
