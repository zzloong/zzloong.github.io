问：羿飞老师
等额本息投标什么意思？

【羿飞】 回答：等额本息，是指一种贷款的还款方式。等额本息是在还款期内，每月偿还同等数额的贷款(包括本金和利息)。它和等额本金是不一样的概念，虽然刚开始还款时每月还款额可能会低于等额本金还款方式，但是最终所还利息会高于等额本金还款方式，该方式经常被银行使用。

  • [自然山水] : 是不是每个板块埋伏一只，那只涨到高估就换掉没启动另一只
  • [王睿晗] : 飞哥推荐的股票基本属于低位，或者已经启动，基本大盘没风险的话，股票就没什么风险。不会出现大盘跌，股票暴跌，大盘张，股票不涨的情况，所以安心买，别乱加仓就行
  • [妮妮] : 飞哥，请问是最好股票池里每个板块买一个股票吗
  • [顺 回复 王睿晗] : 总结的不错，老手
  • [龙猫 回复 王睿晗] : 忍不住加仓怎么搞
  • [王睿晗 回复 龙猫] : 本多就加呗

问：飞哥，锡业股份现在低估吗？目前-2.5%，是否该切换？

【羿飞】 回答：问题不大，业绩连续上升。横盘消化前一波上涨也已经到了尾声，价格也站稳了年线之上，所以长期看我觉得没什么可担心的。
以上是我的观点，至于换与不换，是你自己的事情。

问：飞哥，请教一下，豫金刚石这只股票近期有上升趋势吗？

【羿飞】 回答：这样不是已经在强势中很久了吗？目前属于大宗板块中涨幅比较大的。如果有持仓可以切换板块中其他股票，如果没有持仓，现在追高风险是蛮大的。

问：飞哥，长城汽车和吉利控股怎么样？两家是国内汽车领头羊

【羿飞】 回答：都是好企业，你说的没错。但是估值也都处于比较高估的状态，所以现在买了容易被套。

问：飞哥，000541佛山照明因听在里面做管理的朋友推介而买入，目前亏损20+%，今年业绩也不理想，很大一部分是转让股份所得，我原来设想随着房地产火热，照明方面的需求应该也会增加，想听听飞哥的看法！

【羿飞】 回答：佛山照明的话，属于业绩比较平稳的状态，目前的价格属于在价值区间，没有高低估。
你亏20%的话，大盘如果持续上涨，你还是有机会解套的，但是也不能有太高的期待。

  • [陀飞轮] : 佛山照明股性挺好的，以前超过，赚了点就跑了
  • [大吴] : 在10元多就跑吧，去年业绩做透支了，后面有可能业绩变脸

问：飞哥，我从14年开始投P2P，一开始走过弯路，雷了一些，后来发现您，买了书，还加过群，现在投的基本是您评级里利率较高的，点名说就是广信，融金所，一起好，投复利，丁丁这些。基本都是投24或36个月锁定高收益，而且每月回款都是续投。但现在有些担心，这些平台现在虽然稳定，可未来两三年后的情况谁也不清楚，如果政策收紧或自身经营不善，最终是不是还有可能倒闭跑路？这种风险如何避免？因为现在投资金额是家庭基本全部积蓄，雷了真是不知怎么办好。分散理财之前也试过，之前15年左右也跟着飞哥炒股，有一定盈余，但炒股，基金，黄金等都需要大量时间分析，而且有运气成分，一旦没选好，收益远比不上P2P，所以请飞哥指教

【羿飞】 回答：对于这些平台会不会出问题。
你不清楚，我也不清楚，多那么多的利率不是天上掉下来的。高收益一定是要面临高风险的考验的，所以才要不断的盯着分析研究啊，要不我花那么多时间去做评级研究干嘛。

网贷仍然处于收益红利期，多下点功夫值得的。

  • [CANAAN] : 你还没有最高的易融恒信，担心啥？[呲牙]
    我是这样认为的：1、这么多平台不会同时雷；2、雷需要一个过程，在你认为需要退出的因素出现时，可以减仓或转债；3、要有万一中雷“确实无所谓”的心态和心理准备。最后当然是少不了持续的学习研究！
  • [鹅卵石] : 把积蓄的多少用于投资理财，因为每个人的情况不同，还不好把握定数。我觉得有固定收入来源的，可将大部分积蓄用于投资理财。
  • [yxs] : 其实进了飞榜 能锁定不少资金

问：飞哥，梅花生物已经跌破三十日线，请问这支股票大概什么时候可以企稳

【羿飞】 回答：个股基本面没有什么变化，这种情况下，都是主要看大盘，大盘企稳了，就都进入下一轮行情。

  • [顺 回复 康平] : 先减持，再买入[微笑]
  • [追梦人 回复 康平] : 短期看跌

问：飞哥，请教一下，我在本地的县城房子，有的只要15万~20万就可以买，没有房产证的，是安置点的房子，买按揭的好还是一次性付，只有20万，谢谢飞哥，

【羿飞】 回答：安置房，按说是早晚也应该发房产证的，以后也能恢复正常交易功能。
这个你要问清楚，只要以后房产能发，那么很便宜就一定要买，能按揭就一定要按揭。
